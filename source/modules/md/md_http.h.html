231/**
232 * Perform requests in parallel as retrieved from the nextreq function.
233 * There are as many requests in flight as the nextreq functions provides. 
234 *
235 * To limit the number of parallel requests, nextreq should return APR_ENOENT when the limit
236 * is reached. It will be called again when the number of in_flight requests changes.
237 * 
238 * When all requests are done, nextreq will be called one more time. Should it not
239 * return anything, this function returns.
240 */ view japanese
