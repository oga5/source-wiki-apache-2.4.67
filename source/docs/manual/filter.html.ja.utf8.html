42<h2><a name="filters" id="filters">フィルタ</a></h2>
43

45
46    <p><em>フィルタ</em> とは、サーバが送受信したデータに
47    適用される処理プロセスのことをいいます。クライアントからサーバに
48    送られたデータは <em>入力フィルタ</em> によって、サーバから
49    クライアントに送られるデータは<em>出力フィルタ</em>によって
50    処理されます。複数のフィルタを適用することができ、
51    その順番を厳密に指定することもできます。</p>
52
53    <p>Apache 内部では、チャンク (データのぶつ切り) を行ったり、
54    バイト範囲の指定されたリクエストを扱ったりといった機能を
55    行う際に、フィルタが使われています。それに加えて、
56    実行時の設定ディレクティブで選択が可能なフィルタを
57    モジュールが提供できます。
58    データに適応されるフィルタのセットは、
59    <code class="directive"><a href="./mod/core.html#setinputfilter">SetInputFilter</a></code>,
60    <code class="directive"><a href="./mod/core.html#setoutputfilter">SetOutputFilter</a></code>,
61    <code class="directive"><a href="./mod/core.html#addinputfilter">AddInputFilter</a></code>,
62    <code class="directive"><a href="./mod/core.html#addoutputfilter">AddOutputFilter</a></code>,
63    <code class="directive"><a href="./mod/mod_mime.html#removeinputfilter">RemoveInputFilter</a></code>,
64    <code class="directive"><a href="./mod/mod_mime.html#removeoutputfilter">RemoveOutputFilter</a></code>
65    ディレクティブで制御できます。</p>
66
67    <p>現行の Apache HTTP サーバの配布では、
68    次のユーザ選択可能なフィルタが提供されています。</p>
69
70    <dl>
71      <dt>INCLUDES</dt>
72      <dd><code class="module"><a href="./mod/mod_include.html">mod_include</a></code> で Server-Side Include をします。</dd>
73      <dt>DEFLATE</dt>
74      <dd><code class="module"><a href="./mod/mod_deflate.html">mod_deflate</a></code>
75      を使って、クライアントに送信する前に出力を圧縮します。</dd>
76    </dl>
77
78    <p>また、<code class="module"><a href="./mod/mod_ext_filter.html">mod_ext_filter</a></code> モジュールで
79    外部プログラムをフィルタとして指定することができます。</p>
