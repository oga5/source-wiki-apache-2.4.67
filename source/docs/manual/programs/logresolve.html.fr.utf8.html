34     <p><code class="program"><a href="../programs/logresolve.html">logresolve</a></code> est un programme agissant après
35     traitement pour résoudre les adresses IP dans les journaux d'accès
36     d'Apache. Pour minimiser la charge de votre serveur de noms,
37     logresolve possède son propre cache interne sous forme d'une table
38     de hashage. Cela implique que chaque numéro IP ne fera l'objet
39     d'une requête DNS que la première fois où il est rencontré dans le
40     fichier journal.</p>
41
42     <p>Le programme reçoit le fichier journal sur son entrée standard.
43     Les adresses IP doivent se trouver en tête de chaque ligne et
44     doivent être séparées du reste de la ligne par un espace.</p>

58<h2><a name="options" id="options">Options</a></h2>
59
60<dl>
61
62<dt><code>-s <var>nom-fichier</var></code></dt>
63
64<dd>Spécifie le nom du fichier où seront enregistrées des
65statistiques.</dd>
66
67<dt><code>-c</code></dt>
68
69<dd>Avec cette option, <code class="program"><a href="../programs/logresolve.html">logresolve</a></code> effectue certaines
70vérifications DNS : après avoir trouvé le nom d'hôte correspondant à une
71adresse IP, <code class="program"><a href="../programs/logresolve.html">logresolve</a></code> effectue une recherche DNS sur ce
72nom d'hôte et vérifie si une des adresses IP trouvées correspond à
73l'adresse originale.</dd>
