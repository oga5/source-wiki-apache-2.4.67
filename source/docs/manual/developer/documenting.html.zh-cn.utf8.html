31    <p>Apache 2.0 使用 <a href="http://www.doxygen.org/">Doxygen</a> 从代码中
32    生成 API 和全局变量的文档。下面是对使用 Doxygen 生成文档的简介。</p>

36<h2><a name="brief" id="brief">简要说明</a></h2>
37    <p>使用 <code>/**</code> 开始文档块<br />
38    使用 <code>*/</code> 结束文档块</p>
39
40    <p>在文档块中，我们可以使用多个标签:</p>
41
42    <div class="example"><p><code>
43      Description of this functions purpose<br />
44      @param parameter_name description<br />
45      @return description<br />
46      @deffunc signature of the function<br />
47    </code></p></div>
48      
49    <p>一般不需要 <code>deffunc</code> 。DoxyGen 没有完整的解析器，所以任何
50    在返回类型声明中使用宏的原型，都是太复杂了。这些函数就需要使用 <code>deffunc</code>。
51    例如 (使用 &amp;gt; 而不是 &gt;):</p>
52
53    <div class="example"><p><code>
54      /**<br />
55 &nbsp;* return the final element of the pathname<br />
56 &nbsp;* @param pathname The path to get the final element of<br />
57 &nbsp;* @return the final element of the path<br />
58 &nbsp;* @tip Examples:<br />
59 &nbsp;* &lt;pre&gt;<br />
60 &nbsp;*                 "/foo/bar/gum"   -&amp;gt; "gum"<br />
61 &nbsp;*                 "/foo/bar/gum/"  -&amp;gt; ""<br />
62 &nbsp;*                 "gum"            -&amp;gt; "gum"<br />
63 &nbsp;*                 "wi\\n32\\stuff" -&amp;gt; "stuff"<br />
64 &nbsp;* &lt;/pre&gt;<br />
65 &nbsp;* @deffunc const char * ap_filename_of_pathname(const char *pathname)<br />
66 &nbsp;*/
67    </code></p></div>
68
69    <p>总是在头文件开始包含:</p>
70    <div class="example"><p><code>
71      /**<br />
72 &nbsp;* @package Name of library header<br />
73 &nbsp;*/
74    </code></p></div>
75
76    <p>Doxygen 为每个包生成一个新的 HTML 文件，名字是
77    {Name_of_library_header}.html，所以请简化名称。</p>
78
79    <p>更深入的讨论，请参见
80    <a href="http://www.doxygen.org/">Doxygen 站点</a>。</p>
