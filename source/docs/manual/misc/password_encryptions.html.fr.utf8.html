38<h2><a name="basic" id="basic">Authentification de base</a></h2>
39
40    <p>Voici les cinq formats de mots de passe qu'Apache reconnaît
41    pour l'authentification de base. Notez que tous les formats ne sont
42    pas supportés par toutes les plates-formes :</p>
43
44    <dl>
45         
46       <dt>bcrypt</dt>
47       <dd>"$2y$" + the result of the crypt_blowfish algorithm. Dérivé
48       de l'algorythme de chiffrement crypt_blowfish. Voir le fichier
49       source APR <a href="http://svn.apache.org/viewvc/apr/apr/trunk/crypto/crypt_blowfish.c?view=markup">crypt_blowfish.c</a>
50       pour plus de détails à propos de cet algorithme.</dd>
51
52       <dt>MD5</dt>
53       <dd>"$apr1$" + le résultat d'un algorithme spécifique à Apache
54       utilisant un condensé MD5 réitéré (1000 fois) de combinaisons
55       variées du mot de passe et d'une source d'entropie sur 32 bits.
56       Voir le fichier source APR <a href="http://svn.apache.org/viewvc/apr/apr/trunk/crypto/apr_md5.c?view=markup">apr_md5.c</a>
57       pour les détails de l'algorithme.</dd>
58    
59
60    <dt>SHA1</dt>
61       <dd>"{SHA}" + un condensé SHA-1 du mot de passe codé en
62       Base64. Non sûr.</dd>
63
64    <dt>CRYPT</dt>
65       <dd>Unix seulement. Utilise la fonction Unix traditionnelle
66       <code>crypt(3)</code> avec une source d'entropie sur 32 bits
67       (seuls 12 bits sont utilisés), et seulement les 8 premiers
68       caractères du mot de passe. Non sûr.</dd>
69
70    <dt>PLAIN TEXT (autrement dit <em>non chiffré</em>)</dt>
71      <dd>Windows &amp; Netware seulement. Non sûr.</dd>
72    </dl>
73    <h3>Générer des mots de passe avec htpasswd</h3>
74
75      <div class="example"><h3>bcrypt</h3><p><code>
76      $ htpasswd -nbB monNom monMot-de-passe<br />
77      monNom:$2y$05$c4WoMPo3SXsafkva.HHa6uXQZWr7oboPiC2bT/r7q1BB8I2s0BRqC
78      </code></p></div>    
79
80      <div class="example"><h3>MD5</h3><p><code>
81      $ htpasswd -nbm monNom monMot-de-passe<br />
82      monNom:$apr1$r31.....$HqJZimcKQFAMYayBlzkrA/
83      </code></p></div>
84
85      <div class="example"><h3>SHA1</h3><p><code>
86      $ htpasswd -nbs monNom monMot-de-passe<br />
87      monNom:{SHA}VBPuJHI7uixaa6LQGWx4s+5GKNE=
88      </code></p></div>
89
90      <div class="example"><h3>CRYPT</h3><p><code>
91      $ htpasswd -nbd monNom monMot-de-passe<br />
92      monNom:rqXexS6ZhobKA
93      </code></p></div>
94
95    
96
97    <h3>Générer des mots de passe CRYPT and MD5 avec le programme
98      OpenSSL en ligne de commande</h3>
99      
100
101      <p>OpenSSL connaît l'algorithme MD5 spécifique à Apache.</p>
102
103      <div class="example"><h3>MD5</h3><p><code>
104      $ openssl passwd -apr1 monMot-de-passe<br />
105      $apr1$qHDFfhPC$nITSVHgYbDAK1Y0acGRnY0
106      </code></p></div>
107
108      <div class="example"><h3>CRYPT</h3><p><code>
109      openssl passwd -crypt monMot-de-passe<br />
110      qQ5vTYO3c8dsU
111      </code></p></div>
112    
113
114    <h3>Valider des mots de passe CRYPT and MD5 avec le programme
115      OpenSSL en ligne de commande</h3>
116      
117      <p>La source d'entropie pour un mot de passe CRYPT est constituée
118      des deux premiers caractères (convertis en valeur binaire). Pour
119      valider <code>monMot-de-passe</code> par rapport à
120      <code>rqXexS6ZhobKA</code></p>
121
122      <div class="example"><h3>CRYPT</h3><p><code>
123      $ openssl passwd -crypt -salt rq monMot-de-passe<br />
124      Warning: truncating password to 8 characters<br />
125      rqXexS6ZhobKA
126      </code></p></div>
127
128      <p>Notez que spécifier <code>monMot-d</code> au lieu de
129      <code>monMot-de-passe</code> produira le même résultat car seuls
130      les 8 premiers caractères des mots de passe CRYPT sont pris en
131      compte.</p>
132
133      <p>La source d'entropie pour un mot de passe MD5 se situe entre
134      <code>$apr1$</code> et le caractère <code>$</code> suivant (sous
135      la forme d'une valeur binaire codée en Base64 - au maximum 8
136      caractères). Pour valider <code>monMot-de-passe</code> par rapport
137      à <code>$apr1$r31.....$HqJZimcKQFAMYayBlzkrA/</code></p>
138
139      <div class="example"><h3>MD5</h3><p><code>
140      $ openssl passwd -apr1 -salt r31..... monMot-de-passe<br />
141      $apr1$r31.....$HqJZimcKQFAMYayBlzkrA/
142      </code></p></div>
143    
144
145    <h3>Champs mot de passe de base de données pour
146    mod_dbd</h3>
147      <p>La variante SHA1 constitue probablement le format le mieux
148      approprié pour l'authentification DBD. Comme les fonctions SHA1 et
149      Base64 sont en général disponibles, d'autres logiciels peuvent
150      renseigner une base de données avec des mots de passe chiffrés
151      utilisables par l'authentification basique d'Apache.</p>
152
153      <p>Pour créer des mots de passe au format SHA1 pour
154      l'authentification de base d'Apache dans divers langages :</p>
155
156      <div class="example"><h3>PHP</h3><p><code>
157      '{SHA}' . base64_encode(sha1($password, TRUE))
158      </code></p></div>
159
160      <div class="example"><h3>Java</h3><p><code>
161      "{SHA}" + new sun.misc.BASE64Encoder().encode(java.security.MessageDigest.getInstance("SHA1").digest(password.getBytes()))
162      </code></p></div>
163
164      <div class="example"><h3>ColdFusion</h3><p><code>
165      "{SHA}" &amp; ToBase64(BinaryDecode(Hash(password, "SHA1"), "Hex"))
166      </code></p></div>
167
168      <div class="example"><h3>Ruby</h3><p><code>
169      require 'digest/sha1'<br />
170      require 'base64'<br />
171      '{SHA}' + Base64.encode64(Digest::SHA1.digest(password))
172      </code></p></div>
173
174      <div class="example"><h3>C ou C++</h3><p><code>
175      Utilisez la fonction APR : apr_sha1_base64
176      </code></p></div>
177
178      <div class="example"><h3>Python</h3><p><code>
179      import base64<br />
180      import hashlib<br />
181      "{SHA}" + format(base64.b64encode(hashlib.sha1(password).digest()))
182      </code></p></div>
183
184      <div class="example"><h3>PostgreSQL (avec les fonctions contrib/pgcrypto
185               installées)</h3><p><code>
186        
187        '{SHA}'||encode(digest(password,'sha1'),'base64')
188      </code></p></div>

193<h2><a name="digest" id="digest">Authentification à base de condensés</a></h2>
194    <p>Apache ne reconnaît qu'un format pour les mots de passe
195    d'authentification à base de condensés - le condensé MD5 de la
196    chaîne <code>utilisateur:domaine-de-protection:mot-de-passe</code>
197    sous la forme d'une chaîne de 32 caractères au format hexadécimal.
198    <code>domaine-de-protection</code> est l'identifiant du domaine de
199    protection de l'autorisation passé en argument à la directive
200    <code class="directive"><a href="../mod/mod_authn_core.html#authname">AuthName</a></code> dans
201    httpd.conf.</p>
202
203    <h3>Champs de mot de passe de base de données pour
204    mod_dbd</h3>
205
206      <p>Comme la fonction MD5 est en général disponible, d'autres
207      logiciels peuvent renseigner une base de données avec des mots de
208      passe chiffrés utilisables par l'authentification à base de
209      condensés d'Apache.</p>
210
211      <p>Pour créer des mots de passe pour l'authentification à base de
212      condensés d'Apache dans divers langages :</p>
213
214      <div class="example"><h3>PHP</h3><p><code>
215      md5($user . ':' . $realm . ':' .$password)
216      </code></p></div>
217
218      <div class="example"><h3>Java</h3><p><code>
219      byte b[] = java.security.MessageDigest.getInstance("MD5").digest( (user + ":" + realm + ":" + password ).getBytes());<br />
220      java.math.BigInteger bi = new java.math.BigInteger(1, b);<br />
221      String s = bi.toString(16);<br />
222      while (s.length() &lt; 32)<br />
223      <span class="indent">
224        s = "0" + s;
225      </span>
226      // La chaîne s contient le mot de passe chiffré
227      </code></p></div>
228
229      <div class="example"><h3>ColdFusion</h3><p><code>
230      LCase(Hash( (user &amp; ":" &amp; realm &amp; ":" &amp; password) , "MD5"))
231      </code></p></div>
232
233      <div class="example"><h3>Ruby</h3><p><code>
234      require 'digest/md5'<br />
235      Digest::MD5.hexdigest(user + ':' + realm + ':' + password)
236      </code></p></div>
237
238      <div class="example"><h3>PostgreSQL (avec les fonctions contrib/pgcrypto
239	installées)</h3><p><code>
240        
241        encode(digest( user || ':' || realm || ':' || password , 'md5'), 'hex')
242      </code></p></div>
