37<h3>Sommaire</h3>
38
39    <p><code class="module"><a href="../mod/mod_slotmem_plain.html">mod_slotmem_plain</a></code> est un fournisseur de mémoire qui
40    permet la création et l'utilisation d'un segment de mémoire contigu
41    dans lequel les ensembles de données sont organisés en "slots".
42    </p>
43
44    <p>Si la mémoire doit être partagée entre des threads et des
45    processus, il est préférable d'utiliser le fournisseur
46    <code class="module"><a href="../mod/mod_slotmem_shm.html">mod_slotmem_shm</a></code>.
47    </p>
48
49    <p><code class="module"><a href="../mod/mod_slotmem_plain.html">mod_slotmem_plain</a></code> fournit une API comprenant les
50    fonctions suivantes :
51    </p>
52
53   
54<pre class="prettyprint lang-c">/* appelle le callback sur tous les slots actifs */
