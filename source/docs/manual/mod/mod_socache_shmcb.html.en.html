36<h3>Summary</h3>
37
38    <p><code>mod_socache_shmcb</code> is a shared object cache provider
39    which provides for creation and access to a cache backed by a
40    high-performance cyclic buffer inside a shared memory segment.
41    </p>
42    
43    <div class="example"><p><code>
44    shmcb:/path/to/datafile(512000)
45    </code></p></div>
46
47    <p>Details of other shared object cache providers can be found
48    <a href="../socache.html">here</a>.
49    </p>
