43<h3>概要</h3>
44
45    <p>与えられたプロバイダ <span class="transnote">(<em>訳注:</em> 認証での照会を行う問い合わせ先)</span>
46    でユーザを検索し、HTTP 基本認証でアクセス制限できるようになります。
47    HTTP ダイジェスト認証については <code class="module"><a href="../mod/mod_auth_digest.html">mod_auth_digest</a></code>
48    で提供されます。このモジュールを使う際はこのモジュールのほかに
49    <code class="module"><a href="../mod/mod_authn_file.html">mod_authn_file</a></code> といった認証モジュールと、
50    <code class="module"><a href="../mod/mod_authz_user.html">mod_authz_user</a></code> といった承認モジュールとの両方を、
51    それぞれひとつ以上組み合わせて使うことになります。</p>

73<div class="directive-section"><h2><a name="AuthBasicAuthoritative" id="AuthBasicAuthoritative">AuthBasicAuthoritative</a> <a name="authbasicauthoritative" id="authbasicauthoritative">ディレクティブ</a></h2>
74<table class="directive">
75<tr><th><a href="directive-dict.html#Description">説明:</a></th><td>認証と承認を、より低いレベルのモジュールに移行させるかを
76設定します。</td></tr>
77<tr><th><a href="directive-dict.html#Syntax">構文:</a></th><td><code>AuthBasicAuthoritative On|Off</code></td></tr>
78<tr><th><a href="directive-dict.html#Default">デフォルト:</a></th><td><code>AuthBasicAuthoritative On</code></td></tr>
79<tr><th><a href="directive-dict.html#Context">コンテキスト:</a></th><td>ディレクトリ, .htaccess</td></tr>
80<tr><th><a href="directive-dict.html#Override">上書き:</a></th><td>AuthConfig</td></tr>
81<tr><th><a href="directive-dict.html#Status">ステータス:</a></th><td>Base</td></tr>
82<tr><th><a href="directive-dict.html#Module">モジュール:</a></th><td>mod_auth_basic</td></tr>
83</table>
84    <p>通常は、<code class="directive"><a href="#authbasicprovider">AuthBasicProvider</a></code>
85    ディレクティブで指定した承認モジュールを順に使ってユーザを検査しようとして、
86    どのプロバイダでもユーザを検査できなかった場合、アクセス拒否します。
87    <code class="directive">AuthBasicAuthoritative</code> を <code>Off</code>
88    と明示的に設定すると <strong>ユーザ ID がなかったり</strong>、
89    <strong>ルールがなかったり</strong>する際に、認証と承認の両方について、
90    プロバイダー機構で実装されていないモジュールに処理を移行させることができます。
91    <code class="directive"><a href="#authbasicprovider">AuthBasicProvider</a></code>
92    ディレクティブで設定できないサードパーティ製のモジュールと、
93    <code class="module"><a href="../mod/mod_auth_basic.html">mod_auth_basic</a></code>
94    とを組み合わせるときにのみ必要になるでしょう。
95    そのようなモジュールを使う場合、処理順序はモジュールのソースコードが
96    どうなっているかによって決まり、処理順序を指定することはできません。</p>

115<div class="directive-section"><h2><a name="AuthBasicProvider" id="AuthBasicProvider">AuthBasicProvider</a> <a name="authbasicprovider" id="authbasicprovider">ディレクティブ</a></h2>
116<table class="directive">
117<tr><th><a href="directive-dict.html#Description">説明:</a></th><td>この位置に対する認証プロバイダを設定します。</td></tr>
118<tr><th><a href="directive-dict.html#Syntax">構文:</a></th><td><code>AuthBasicProvider <var>provider-name</var>
119[<var>provider-name</var>] ...</code></td></tr>
120<tr><th><a href="directive-dict.html#Default">デフォルト:</a></th><td><code>AuthBasicProvider file</code></td></tr>
121<tr><th><a href="directive-dict.html#Context">コンテキスト:</a></th><td>ディレクトリ, .htaccess</td></tr>
122<tr><th><a href="directive-dict.html#Override">上書き:</a></th><td>AuthConfig</td></tr>
123<tr><th><a href="directive-dict.html#Status">ステータス:</a></th><td>Base</td></tr>
124<tr><th><a href="directive-dict.html#Module">モジュール:</a></th><td>mod_auth_basic</td></tr>
125</table>
126    <p><code class="directive">AuthBasicProvider</code> ディレクティブで、
127    この位置に対するユーザ認証に用いられる認証プロバイダを設定します。
128    デフォルトになっている <code>file</code> プロバイダは
129    <code class="module"><a href="../mod/mod_authn_file.html">mod_authn_file</a></code> モジュールで実装されています。
130    指定したプロバイダを実装しているモジュールが、
131    必ずサーバに組み込まれているようにしてください。</p>
132
133    <div class="example"><h3>Example</h3><p><code>
134      &lt;Location /secure&gt;<br />
135      <span class="indent">
136        AuthType basic<br />
137        AuthName "private area"<br />
138        AuthBasicProvider  dbm<br />
139        AuthDBMType        SDBM<br />
140        AuthDBMUserFile    /www/etc/dbmpasswd<br />
141        Require            valid-user<br />
142      </span>
143      &lt;/Location&gt;
144    </code></p></div>
145
146    <p>認証プロバイダは <code class="module"><a href="../mod/mod_authn_dbm.html">mod_authn_dbm</a></code>,
147    <code class="module"><a href="../mod/mod_authn_file.html">mod_authn_file</a></code>,
148    <code class="module"><a href="../mod/mod_authn_dbd.html">mod_authn_dbd</a></code>,
149    <code class="module"><a href="../mod/mod_authnz_ldap.html">mod_authnz_ldap</a></code> で実装されています。</p>
