42<h3>概要</h3>
43
44    <p>本モジュールには <code class="module"><a href="../mod/mod_proxy.html">mod_proxy</a></code> が<em>必要です</em>。
45    <code>CONNECT</code> HTTP メソッドをサポートします。
46    このメソッドは主にプロキシに SSL リクエストを通す
47    <span class="transnote">(<em>訳注:</em> SSLトンネリング)</span>に使われます。</p>
48
49    <p><code>CONNECT</code> リクエストを扱えるようにするには
50    <code class="module"><a href="../mod/mod_proxy.html">mod_proxy</a></code> と <code class="module"><a href="../mod/mod_proxy_connect.html">mod_proxy_connect</a></code>
51    をサーバに組み込む必要があります。</p>
52
53    <div class="warning"><h3>警告</h3>
54      <p><a href="mod_proxy.html#access">安全なサーバにする</a>までプロキシ機能は有効にしないでください。
55      オープンプロキシサーバはあなた自身のネットワークにとっても、
56      インターネット全体にとっても危険です。</p>
57    </div>
