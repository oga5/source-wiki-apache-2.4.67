37<h3>Sommaire</h3>
38
39<p>
40<code class="module"><a href="../mod/mod_heartmonitor.html">mod_heartmonitor</a></code> interprète les messages d'état générés
41par les serveurs d'origine pour lesquels <code class="module"><a href="../mod/mod_heartbeat.html">mod_heartbeat</a></code> est activé et
42fournit ces informations à <code class="module"><a href="../mod/mod_lbmethod_heartbeat.html">mod_lbmethod_heartbeat</a></code>, ce
43qui permet d'utiliser la <em>lbmethod</em> "heartbeat" au sein des
44directives <code class="directive"><a href="../mod/mod_proxy.html#proxypass">ProxyPass</a></code>.
45</p>
46
47<p>Ce module utilise les services de <code class="module"><a href="../mod/mod_slotmem_shm.html">mod_slotmem_shm</a></code>,
48lorsqu'il est disponible, au lieu d'un simple fichier texte. Aucune
49configuration supplémentaire n'est requise pour utiliser
50<code class="module"><a href="../mod/mod_slotmem_shm.html">mod_slotmem_shm</a></code>.</p>
51
52    <div class="warning">
53        Pour utiliser <code class="module"><a href="../mod/mod_heartmonitor.html">mod_heartmonitor</a></code>,
54	<code class="module"><a href="../mod/mod_status.html">mod_status</a></code> et <code class="module"><a href="../mod/mod_watchdog.html">mod_watchdog</a></code>
55	doivent être soit des modules statiques, soit des modules
56	dynamiques, et dans ce dernier cas, ils doivent être chargés
57	avant <code class="module"><a href="../mod/mod_heartmonitor.html">mod_heartmonitor</a></code>.
58    </div>

71<div class="directive-section"><h2><a name="heartbeatlisten" id="heartbeatlisten">Directive</a> <a name="HeartbeatListen" id="HeartbeatListen">HeartbeatListen</a></h2>
72<table class="directive">
73<tr><th><a href="directive-dict.html#Description">Description:</a></th><td>Adresse multicast d'écoute des requêtes entrantes heartbeat</td></tr>
74<tr><th><a href="directive-dict.html#Syntax">Syntaxe:</a></th><td><code>HeartbeatListen <var>addr:port</var></code></td></tr>
75<tr><th><a href="directive-dict.html#Default">Défaut:</a></th><td><code>disabled</code></td></tr>
76<tr><th><a href="directive-dict.html#Context">Contexte:</a></th><td>configuration globale</td></tr>
77<tr><th><a href="directive-dict.html#Status">Statut:</a></th><td>Expérimental</td></tr>
78<tr><th><a href="directive-dict.html#Module">Module:</a></th><td>mod_heartmonitor</td></tr>
79</table>
80    <p>La directive <code class="directive">HeartbeatListen</code> permet de
81    spécifier l'adresse multicast sur laquelle le serveur va surveiller les
82    informations d'état en provenance de serveurs où
83    <code class="module"><a href="../mod/mod_heartbeat.html">mod_heartbeat</a></code> est activé. Cette adresse correspond
84    en général à la valeur de la directive <code class="directive"><a href="../mod/mod_heartbeat.html#heartbeataddress">HeartbeatAddress</a></code> sur le serveur
85    d'origine.
86    </p>
87
88    <pre class="prettyprint lang-config">HeartbeatListen 239.0.0.1:27999</pre>
89
90
91    <p>Tant que cette directive n'est pas utilisée, le module est
92    désactivé.</p>

96<div class="directive-section"><h2><a name="heartbeatmaxservers" id="heartbeatmaxservers">Directive</a> <a name="HeartbeatMaxServers" id="HeartbeatMaxServers">HeartbeatMaxServers</a></h2>
97<table class="directive">
98<tr><th><a href="directive-dict.html#Description">Description:</a></th><td>Spécifie le nombre maximal de serveurs qui pourront envoyer
99des requêtes heartbeat à ce serveur.</td></tr>
100<tr><th><a href="directive-dict.html#Syntax">Syntaxe:</a></th><td><code>HeartbeatMaxServers <var>nombre-de-serveurs</var></code></td></tr>
101<tr><th><a href="directive-dict.html#Default">Défaut:</a></th><td><code>HeartbeatMaxServers 10</code></td></tr>
102<tr><th><a href="directive-dict.html#Context">Contexte:</a></th><td>configuration globale</td></tr>
103<tr><th><a href="directive-dict.html#Status">Statut:</a></th><td>Expérimental</td></tr>
104<tr><th><a href="directive-dict.html#Module">Module:</a></th><td>mod_heartmonitor</td></tr>
105<tr><th><a href="directive-dict.html#Compatibility">Compatibilité:</a></th><td>La valeur 0 est prise en charge à partir de la version 2.4.55 du
106serveur HTTP Apache</td></tr>
107</table>
108    <p>La directive <code class="directive">HeartbeatMaxServers</code>
109    spécifie le nombre maximal de serveurs qui pourront envoyer des
110    requêtes heartbeat à ce serveur de monitoring. Elle permet ainsi de
111    contrôler la quantité de mémoire partagée allouée pour le stockage
112    des données heartbeat lorsqu'on utilise
113    <code class="module"><a href="../mod/mod_slotmem_shm.html">mod_slotmem_shm</a></code>.</p>
114    <p>Pour utiliser un stockage de type fichier bidimensionnel (flat-file)
115    lorque le module <code class="module"><a href="../mod/mod_slotmem_shm.html">mod_slotmem_shm</a></code> n'est pas chargé, cette
116    directive doit être définie à 0. La valeur doit être soit égale à 0, soit
117    supérieure ou égale à 10.</p>

121<div class="directive-section"><h2><a name="heartbeatstorage" id="heartbeatstorage">Directive</a> <a name="HeartbeatStorage" id="HeartbeatStorage">HeartbeatStorage</a></h2>
122<table class="directive">
123<tr><th><a href="directive-dict.html#Description">Description:</a></th><td>Chemin vers le stockage des données heartbeat lorsqu'on utilise un
124fichier bidimensionnel (flat-file)</td></tr>
125<tr><th><a href="directive-dict.html#Syntax">Syntaxe:</a></th><td><code>HeartbeatStorage <var>chemin fichier</var></code></td></tr>
126<tr><th><a href="directive-dict.html#Default">Défaut:</a></th><td><code>HeartbeatStorage logs/hb.dat</code></td></tr>
127<tr><th><a href="directive-dict.html#Context">Contexte:</a></th><td>configuration globale</td></tr>
128<tr><th><a href="directive-dict.html#Status">Statut:</a></th><td>Expérimental</td></tr>
129<tr><th><a href="directive-dict.html#Module">Module:</a></th><td>mod_heartmonitor</td></tr>
130</table>
131    <p>La directive <code class="directive">HeartbeatStorage</code> permet de spécifier
132    le chemin de stockage des données heartbeat. Ce fichier bidimensionnel n'est
133    utilisé que si <code class="module"><a href="../mod/mod_slotmem_shm.html">mod_slotmem_shm</a></code> n'est pas chargé et si la
134    directive <code class="directive"><a href="#heartbeatmaxservers">HeartbeatMaxServers</a></code>
135    est définie à 0.</p>
