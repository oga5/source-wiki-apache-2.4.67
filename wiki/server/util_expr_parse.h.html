server/util_expr_parse.h

概要

本ファイルは、Apache HTTP Server の式評価エンジン（ap_expr）における構文解析器のインターフェース定義です。GNU Bison によって生成されたヘッダーファイルであり、構文解析に必要なトークン定義とセマンティック値の型定義を提供します。

モジュール内での役割

server モジュール内の式評価サブシステムにおいて、構文解析の「契約」を定義します。util_expr_parse.c（パーサー実装）と、字句解析器である util_expr_scan.c を繋ぐ共通言語として機能します。

責務

  • 構文解析器が認識する全トークンの列挙（yytokentype）。
  • 構文解析の各ノードが保持する値の型（YYSTYPE）の定義。
  • 字句解析器と構文解析器間での型安全なデータ受け渡し規約の提供。

重要なシンボル

  • enum yytokentype: 式に含まれる演算子（T_OP_EQ 等）やリテラル（T_STRING 等）を識別するためのトークン定数群。
  • union YYSTYPE: 構文解析ツリーのノードが保持する値の共用体。文字列（cpVal）、式ノード（exVal）、数値（num）を格納します。
  • YYSTYPE_IS_DECLARED: 型定義の重複を防ぐためのフラグ。

制御フローとデータフロー

本ファイル自体は宣言のみを提供します。 1. **字句解析**: util_expr_scan.c が入力文字列を読み取り、yytokentype に基づくトークンを生成します。 2. **値の受け渡し**: 字句解析器は、解析した値を YYSTYPE 型の変数（yylval）に格納し、構文解析器へ渡します。 3. **構文解析**: util_expr_parse.c 内の ap_expr_yyparse がこれらのトークンと値を受け取り、抽象構文木（AST）を構築します。
