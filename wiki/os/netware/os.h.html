os/netware/os.h

概要

os/netware/os.h は、Apache HTTP Server が NetWare プラットフォーム上で動作するために必要な OS 固有の定義、マクロ、および互換性レイヤーを提供するヘッダーファイルです。

モジュール内での役割

os/netware モジュールにおける中心的な設定ファイルであり、すべてのソースコードからインクルードされることを想定しています。プラットフォーム特有の挙動を抽象化し、コア機能が NetWare の環境差異を意識せずに動作できるようにします。

責務

  • NetWare 固有のプラットフォーム識別子（PLATFORM）の定義。
  • 標準的なシステムコールやライブラリ関数の NetWare 向けマッピング（例: getpid の置換）。
  • プロセス終了時の画面保持制御など、NetWare 特有のライフサイクル管理の定義。
  • ファイルシステム特性（大文字小文字の区別なし）の宣言。

重要なシンボル

  • APACHE_OS_H: 多重インクルードを防ぐためのガードマクロ。
  • CASE_BLIND_FILESYSTEM: NetWare のファイルシステムが大文字小文字を区別しないことを示すフラグ。
  • getpid: NXThreadGetId へのマクロ置換。
  • exit(s): 終了時に画面を保持する pressanykey() を呼び出し、apr_terminate() を経由して終了するカスタムマクロ。

制御フローとデータフロー

本ファイルは実行コードではなく定義ファイルです。

  • exit() マクロが呼び出されると、hold_screen_on_exit フラグの状態に応じて pressanykey() が実行され、ユーザーの入力を待機してからプロセスが終了します。
  • getpid が呼び出されると、NetWare のスレッド ID を取得する NXThreadGetId に透過的にリダイレクトされます。

直接依存

  • screen.h: NetWare の画面制御関数（pressanykey 等）を利用するためにインクルードされています。

関連コード

  • os/netware/util_nw.c: NetWare 固有のユーティリティ関数群。
  • os/netware/pre_nw.h: コンパイル前の環境設定や依存関係の調整を行うヘッダー。
  • os/bs2000/os.c: 他のプラットフォームにおける OS 固有実装の対照例。

読者向けメモ

  • 本ファイルで定義されている exit マクロは、標準の exit() をオーバーライドします。デバッグ時や終了処理の挙動を追う際は、このマクロが apr_terminate() を呼び出している点に注意してください。
  • AP_PLATFORM_REWRITE_ARGS_HOOK が現在 NULL に設定されている通り、コマンドライン引数の書き換えは NetWare MPM 側に依存しています。将来的な MPM の拡張時にはこの定義が見直される可能性があります。
