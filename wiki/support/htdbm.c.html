support/htdbm.c

概要

htdbm.c は、Apache HTTP Server が使用する DBM 形式のパスワードデータベースを操作するためのコマンドラインユーティリティです。ユーザーの追加、削除、検証、およびデータベース内容のリスト表示機能を提供します。

モジュール内での役割

support モジュールの一部として、サーバー管理者が認証用データベースをオフラインで管理するためのツールを提供します。passwd_common.c と連携し、パスワードハッシュ生成や入力処理の共通ロジックを共有しています。

責務

  • **データベース操作**: apr_dbm を介した DBM ファイルの読み書きおよび管理。
  • **認証管理**: ユーザー名とパスワードのペアの保存、削除、検証。
  • **ハッシュ処理**: passwd_common を利用した多様なハッシュアルゴリズム（MD5, SHA, BCRYPT, CRYPT, Plaintext）の適用。
  • **CLI インターフェース**: コマンドライン引数の解析と、対話的または非対話的なパスワード入力処理。

重要なシンボル

データ構造

  • htdbm_t: データベース接続、ユーザー情報、コンテキスト、および設定状態を保持するメイン構造体。

ライフサイクル・管理

  • htdbm_init(): APR プールと htdbm_t 構造体の初期化、シグナルハンドラの登録。
  • htdbm_terminate(): DBM 接続をクローズし、リソースを解放。
  • htdbm_open(): 指定された DBM ファイルをモード（読み取り専用/書き込み/作成）に応じて開く。

操作・ミューテータ

  • htdbm_save(): ユーザー名とハッシュ化されたパスワードを DBM に保存。
  • htdbm_del(): 指定ユーザーをデータベースから削除。
  • htdbm_make(): パスワードをハッシュ化し、htdbm_t に格納。
  • htdbm_verify(): 既存のハッシュと入力パスワードを比較検証。
  • htdbm_list(): データベース内の全レコードを標準エラーに出力。

制御フローとデータフロー

1. **初期化**: main が htdbm_init を呼び出し、APR 環境と htdbm_t を準備。 2. **引数解析**: apr_getopt を使用してオプションを解析し、parse_common_options で共通設定を適用。 3. **実行モード決定**: cmd フラグ（HTDBM_MAKE, HTDBM_DELETE 等）に基づき、htdbm_open でファイルを開いた後、各操作関数を呼び出す。 4. **データ処理**: htdbm_make が mkhash を呼び出し、結果を htdbm_t に保持。htdbm_save が apr_dbm_store を使用して永続化。 5. **終了**: htdbm_terminate でリソースをクリーンアップし、atexit で登録された terminate が APR を終了させる。

直接依存

  • passwd_common.h: パスワードハッシュ生成の共通ロジック。
  • apr_dbm.h: DBM ファイル操作の抽象化レイヤー。
  • apr_pools.h: メモリ管理。
  • apr_getopt.h: コマンドライン引数解析。
  • apr_md5.h, apr_sha1.h: ハッシュアルゴリズム。

関連コード

  • support/passwd_common.c: パスワード入力の取得やハッシュ生成の具体的な実装。
  • support/htpasswd.c: テキストファイルベースのパスワード管理ツール（類似の責務を持つ）。

読者向けメモ

  • **メモリ管理**: すべての動的メモリは apr_pool_t によって管理されます。htdbm_t の各フィールドはプールから割り当てられており、明示的な free は不要です。
  • **データレイアウト**: DBM 内のデータは username をキーとし、password[:comment] を値として保存されます。\0 終端は保存されないため、apr_datum_t の dsize を常に参照する必要があります。
  • **EBCDIC 対応**: APR_CHARSET_EBCDIC が定義されている場合、ハッシュ計算前に文字コード変換が行われます。
  • **省略された詳細**: mkhash や parse_common_options などの詳細なハッシュ計算ロジックは passwd_common.c に委譲されており、本ファイルには含まれていません。
