test/modules/http2/htdocs/cgi/hello.py

概要

HTTP/2 サーバーのテスト環境において、CGI 経由でサーバー環境変数およびリクエスト情報を JSON 形式で出力する診断用スクリプトです。

モジュール内での役割

test/modules/http2/htdocs/cgi ディレクトリに配置された一連のテスト用 CGI スクリプトの一つです。主にサーバーがリクエストをどのように処理し、どのような環境変数を CGI に引き渡しているかを確認するための「情報開示」の役割を担います。

責務

  • サーバーから提供される環境変数（os.environ）の読み取り。
  • HTTP/2 関連のヘッダーやサーバー状態（H2_STREAM_ID, HTTPS, SERVER_PROTOCOL 等）の抽出。
  • 抽出した情報を JSON 形式に整形し、標準出力へ書き出すこと。

重要なシンボル

  • resp: サーバー環境変数をマッピングした辞書オブジェクト。
  • os.getenv(): CGI 環境変数から値を取得するための標準関数。
  • json.JSONEncoder: 辞書データを整形された JSON 文字列に変換するために使用。

制御フローとデータフロー

1. **初期化**: os.environ から特定の環境変数を取得し、resp 辞書を構築します。 2. **条件分岐**: X_HOST が存在する場合はそれを優先し、存在しない場合は SERVER_NAME を使用するフォールバックロジックが含まれています。 3. **出力**: Content-Type: application/json ヘッダーを標準出力し、空行を挟んでから resp を JSON 文字列として出力します。

直接依存

  • os: 環境変数の取得に使用。
  • json: JSON 形式でのレスポンス生成に使用。

関連コード

  • test/modules/http2/htdocs/cgi/env.py: 同様に環境変数をダンプするスクリプトですが、出力形式や取得対象が異なる場合があります。
  • test/modules/http2/htdocs/cgi/requestparser.py: リクエストの解析ロジックを共有している可能性があるため、複雑なテストケースでは併せて参照してください。

読者向けメモ

  • 本スクリプトは、HTTP/2 接続におけるストリーム ID やプロトコル情報が正しく CGI に伝播しているかを検証する際に非常に有用です。
  • 実行環境の os.environ に依存するため、Web サーバー（Apache 等）の CGI 設定が正しく行われていることが前提となります。
  • 開発やデバッグ時に、特定の環境変数が欠落している場合は、サーバー側の設定（PassEnv や SetEnv 等）を確認してください。
