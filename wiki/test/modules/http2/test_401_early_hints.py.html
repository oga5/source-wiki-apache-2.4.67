test/modules/http2/test_401_early_hints.py

概要

HTTP/2 の「Early Hints (103 ステータスコード)」機能が、Apache HTTP Server (mod_http2) で正しく動作するかを検証するテストスイートです。

モジュール内での役割

test/modules/http2 テストスイートの一部として、サーバーがリソースのプッシュやプリロードのヒントをクライアントに先行送信する機能を検証します。

責務

  • H2EarlyHints ディレクティブの有効化と設定の検証。
  • H2PushResource および H2EarlyHint 設定による 103 Early Hints の生成確認。
  • プッシュ機能がオフの場合でも、Early Hints が適切に送信されるかの確認。
  • Apache 2.4.58 以降のバージョン要件の強制。

重要なシンボル

  • TestEarlyHints: テストクラス。pytest を使用して環境構築とテスト実行を管理。
  • H2Conf: テスト用の Apache 設定を動的に生成するヘルパークラス。
  • nghttp: HTTP/2 通信をシミュレートし、レスポンスヘッダーやプッシュされたストリームを解析するツール。

制御フローとデータフロー

1. **セットアップ**: _class_scope フィクスチャが H2Conf を用いて仮想ホストを設定し、Apache を再起動します。 2. **リクエスト**: nghttp を使用して特定の URL に GET リクエストを送信します。 3. **検証**:

  • レスポンスの previous フィールドを確認し、103 ステータスコードが含まれているかチェックします。
  • r.results["streams"] を通じて、サーバーがリソースをプッシュしたか（promises の有無）を検証します。

直接依存

  • pytest: テストフレームワーク。
  • .env: H2Conf および H2TestEnv を提供し、テスト環境の構築と Apache の制御を行います。

関連コード

  • test/modules/http2/test_400_push.py: HTTP/2 プッシュ機能の基本動作を検証する関連テスト。
  • test/modules/http2/env.py: テスト環境の基盤となる設定クラス群。

読者向けメモ

  • 本テストは Apache 2.4.58 以降を前提としています。それ以前のバージョンではスキップされます。
  • H2EarlyHint は Link ヘッダーを 103 レスポンスとして先行送信するために使用されます。
  • H2PushResource を使用した場合、103 レスポンスとプッシュの両方が発生する挙動を確認しています。
  • nghttp のレスポンスオブジェクトにある previous 属性は、103 Early Hints をキャプチャするために重要です。
