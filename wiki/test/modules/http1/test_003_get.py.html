test/modules/http1/test_003_get.py

概要

HTTP/1.1 プロトコルにおける基本的な GET リクエストの動作を検証するテストスイートです。特に CGI スクリプトを介したレスポンスの整合性と、SSL/TLS 環境変数の伝播を確認することに主眼を置いています。

モジュール内での役割

test/modules/http1 モジュールの一部として、HTTP/1.1 の標準的なリクエスト処理が期待通りに動作するかを検証する機能テストを提供します。

責務

  • **環境構築**: pytest フィクスチャを利用し、テスト実行前に Apache サーバーへ必要なバーチャルホスト（CGI および test1）を設定し、再起動を行う。
  • **GET リクエスト検証**: curl を用いたリクエスト発行と、レスポンスのステータスコード、プロトコルバージョン、および CGI 経由で取得した環境変数の検証。

重要なシンボル

  • TestGet: テストクラス。クラススコープで環境設定を一度だけ実行する設計。
  • _class_scope: autouse フィクスチャ。H1Conf を用いてサーバー設定を注入し、Apache を再起動する。
  • test_h1_003_01: 具体的なテストケース。HTTPS 経由で CGI を呼び出し、TLS バージョンやプロトコル情報が正しく渡されているかを確認する。

制御フローとデータフロー

1. **セットアップ**: クラス初期化時に env オブジェクトを介して H1Conf が呼び出され、サーバー設定が適用されます。 2. **リクエスト**: env.mkurl で生成された URL に対して env.curl_get が実行され、HTTP リクエストが発行されます。 3. **検証**: レスポンスオブジェクト r から JSON 形式のボディを抽出し、期待値（HTTP/1.1, https=on, TLS バージョン）と比較します。

直接依存

  • socket: ネットワーク通信の基盤として利用。
  • pytest: テストフレームワーク。
  • .env: H1Conf クラスおよび env フィクスチャの定義元。

関連コード

  • test/modules/http1/env.py: テスト環境の構成（H1Conf）や、curl ラッパーを含む共通基盤。
  • test/modules/http1/conftest.py: env フィクスチャの定義や、テスト実行に必要な共有設定。

読者向けメモ

  • 本ファイルは、HTTP/1.1 の機能テストにおける「正常系」の代表的な実装例です。
  • H1Conf による設定変更はクラス全体に影響するため、他のテストケースを追加する際は、設定の競合や再起動のコストに注意してください。
  • CGI 経由のレスポンス検証を行っているため、サーバー側の CGI 実行環境が正しく設定されていることが前提となります。
