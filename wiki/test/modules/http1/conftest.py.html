test/modules/http1/conftest.py

概要

本ファイルは pytest の設定ファイルであり、test/modules/http1 ディレクトリ配下のテストスイート全体で共有される環境構築およびテスト実行のフックを提供します。

モジュール内での役割

HTTP/1 テストスイートの基盤となる H1TestEnv を初期化し、テスト実行に必要な Apache HTTP Server のライフサイクル管理（起動・停止・ログ管理）を担います。

責務

  • **環境の初期化**: H1TestEnv を用いて Apache の設定、起動、ログのクリアを行う。
  • **テストレポートの拡張**: pytest のレポートヘッダーに Apache のバージョンや MPM モジュール情報を付与する。
  • **テストのパラメータ化**: repeat オプションによるテストの繰り返し実行をサポートする。
  • **リソースのクリーンアップ**: パッケージスコープの終了時に Apache を確実に停止させる。

重要なシンボル

  • env: package スコープの pytest フィクスチャ。テスト環境のセットアップと Apache の起動を一度だけ実行し、各テストケースに共有する。
  • H1TestEnv: HTTP/1 テスト用の環境設定クラス。Apache の制御やログパスの管理を行う。
  • pytest_report_header: テスト実行開始時に環境情報を出力するためのフック。
  • pytest_generate_tests: repeat 引数を持つテストに対して、指定回数分のパラメータ化を動的に適用する。

制御フローとデータフロー

1. **初期化**: pytest 実行時、env フィクスチャが呼び出され、H1TestEnv がインスタンス化される。 2. **セットアップ**: env.setup_httpd() が実行され、Apache が起動。ログファイルがクリアされる。 3. **テスト実行**: 各テスト関数は env フィクスチャを注入し、Apache へのリクエスト送信やログ検証を行う。 4. **終了処理**: _stop_package_scope フィクスチャが yield 後に実行され、env.apache_stop() を呼び出して Apache を停止させる。

直接依存

  • pytest: テストフレームワークのコア機能。
  • logging: テスト実行中のログ出力設定。
  • .env.H1TestEnv: テスト環境の抽象化クラス。
  • sys, os: モジュールパスの解決に使用。

関連コード

  • test/modules/http1/env.py: H1TestEnv の実装詳細。環境変数の管理や Apache の制御ロジックが定義されている。
  • test/modules/http1/test_*.py: 本ファイルで定義された env フィクスチャを利用する個別のテストケース群。

読者向けメモ

  • 本ファイルは package スコープのフィクスチャを使用しているため、このディレクトリ内のテストは Apache を再起動することなく、共有された環境で実行されます。
  • sys.path.append により親ディレクトリをパスに追加しているため、実行環境のディレクトリ構成に依存します。
  • ログレベルは logging.INFO に固定されています。デバッグ時には必要に応じて変更してください。
