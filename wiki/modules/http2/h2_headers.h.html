modules/http2/h2_headers.h

概要

HTTP/2 通信におけるレスポンスヘッダーおよび関連メタデータを、Apache APR バケットフレームワーク上で扱うためのデータ構造と API を定義します。

モジュール内での役割

modules/http2 において、HTTP/2 ストリームのレスポンスヘッダーをバケットチェーンに挿入可能な形式（H2HEADERS バケット）としてカプセル化します。これにより、ストリームの出力データフローにヘッダー情報を透過的に組み込むことが可能になります。

責務

  • h2_headers 構造体の定義と管理（ステータスコード、ヘッダーテーブル、メモの保持）。
  • カスタムバケット型 h2_bucket_type_headers の提供。
  • HTTP/2 レスポンスヘッダーの生成、コピー、クローン、およびサイズ計算機能の提供。
  • AP_HAS_RESPONSE_BUCKETS が定義されていない環境における、互換性レイヤーとしての機能。

重要なシンボル

  • h2_headers: ヘッダー情報と関連するメタデータを保持するメイン構造体。
  • h2_bucket_type_headers: APR バケットフレームワークに登録されるカスタムバケット型。
  • H2_BUCKET_IS_HEADERS(e): 指定されたバケットが h2_headers 型であるかを判定するマクロ。
  • h2_headers_create(): 新規ヘッダーオブジェクトの生成関数。

制御フローとデータフロー

1. **生成**: h2_headers_create 等によりヘッダーデータが構築されます。 2. **バケット化**: h2_bucket_headers_create を通じて、データが apr_bucket にラップされます。 3. **挿入**: 生成されたバケットは、Apache の出力フィルタチェーンやストリームのバケットチェーンに挿入されます。 4. **抽出**: バケットチェーンを走査する際、H2_BUCKET_IS_HEADERS で判定し、h2_bucket_headers_get で内部の h2_headers 構造体を取り出して処理を行います。

直接依存

  • h2.h: モジュール共通定義。
  • apr_buckets.h (間接的): バケット操作の基盤。
  • httpd.h: Apache HTTP Server のコア構造体。

関連コード

  • modules/http2/h2_headers.c: 本ヘッダーファイルの関数実装およびバケット操作のロジック。
  • modules/http2/h2_stream.h: ストリーム単位でのヘッダー処理の文脈。
  • modules/http2/h2_util.h: ヘッダー操作に関連するユーティリティ関数群。

読者向けメモ

  • 本ファイルは AP_HAS_RESPONSE_BUCKETS が定義されていない環境（古い Apache バージョン等）でのみ有効化される設計です。
  • h2_headers_clone はヘッダー文字列の深いコピー（deep copy）を行いますが、h2_headers_copy は浅いコピー（shallow copy）である点に注意してください。
  • バケットのライフサイクル管理は APR のバケットフレームワークに依存しているため、メモリリークを防ぐために適切な destroy 処理が組み込まれています。
