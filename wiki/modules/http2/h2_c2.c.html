modules/http2/h2_c2.c

概要

h2_c2.c は、HTTP/2 のストリームを個別の「セカンダリ接続（c2）」として扱うためのインフラを提供します。Apache HTTP Server の標準的な接続処理を HTTP/2 の多重化モデルに適合させるため、接続のライフサイクル管理、フィルタチェーンの構築、および入出力のブリッジングを担います。

モジュール内での役割

mod_http2 における「ワーカー」側の接続処理を定義します。マスター接続（c1）から分離されたストリームを、あたかも独立した HTTP 接続であるかのように見せかけることで、既存の HTTP 処理パイプライン（ap_process_request 等）を再利用可能にします。

責務

  • **接続ライフサイクル管理**: セカンダリ接続（conn_rec）の生成、破棄、および MPM 固有設定の継承。
  • **フィルタチェーンの制御**: HTTP/2 特有の入出力フィルタ（H2_C2_NET_IN/OUT 等）を登録し、標準の HTTP/1.1 フィルタを置換またはバイパスする。
  • **データブリッジング**: h2_bucket_beam を介して、HTTP/2 セッションとワーカー間のデータ転送を仲介する。
  • **MPM 互換性**: 実行環境（MPM）のサポート状況を判定し、必要に応じてダミーソケットを使用して接続をエミュレートする。

重要なシンボル

ライフサイクルと管理

  • h2_c2_create: セカンダリ接続用の conn_rec を生成し、メモリプールを分離して初期化する。
  • h2_c2_destroy: 接続に関連付けられたリソースとプールを解放する。
  • h2_c2_abort: 接続の I/O を強制終了し、関連する beam をエラー状態にする。
  • h2_c2_child_init: 子プロセス初期化時に MPM 情報を取得し、ダミーソケットを準備する。

フィルタとフック

  • h2_c2_filter_in: ネットワーク入力フィルタ。beam_in からデータを読み取り、apr_bucket_brigade に変換する。
  • h2_c2_filter_out: ネットワーク出力フィルタ。beam_out を介してデータを HTTP/2 セッションへ送出する。
  • h2_c2_register_hooks: HTTP/2 接続処理に必要なフック（pre_connection, post_read_request 等）を登録する。

MPM 判定

  • h2_mpm_supported: 現在の MPM が HTTP/2 のワーカーモデルをサポートしているか判定する。

制御フローとデータフロー

1. **接続生成**: h2_c2_create が呼ばれ、マスター接続から設定をコピーした c2 が作成される。 2. **フック実行**: ap_hook_process_connection により h2_c2_hook_process が起動し、c2_process を通じて ap_process_request が呼び出される。 3. **データ入力**: h2_c2_filter_in が h2_beam_receive を呼び出し、HTTP/2 セッションから送られてきたデータを beam_in 経由で取得する。 4. **データ出力**: h2_c2_filter_out が h2_beam_send を呼び出し、レスポンスデータを beam_out 経由で HTTP/2 セッションへ戻す。 5. **終了**: ap_process_request 完了後、h2_beam_close で出力がクローズされ、接続が破棄される。

読者向けメモ

  • **メモリ所有権**: h2_c2_create で作成される conn_rec は、専用のメモリプールを所有します。これにより、マスター接続とは独立したスレッドでの処理が可能になります。
  • **レイアウト**: AP_HAS_RESPONSE_BUCKETS の有無により、フィルタの構成が大きく異なります。新しい Apache バージョンではレスポンスバケットが標準化されているため、コードパスが分岐しています。
  • **エッジケース**: h2_c2_abort は、接続が異常終了した際に beam を確実にクローズしてデッドロックを防ぐために重要です。
  • **省略**: 本ファイルは接続の「枠組み」に特化しており、HTTP/2 プロトコル自体のフレーム解析（h2_protocol.c）や、ワーカーのスケジューリング（h2_workers.c）の詳細は省略されています。
