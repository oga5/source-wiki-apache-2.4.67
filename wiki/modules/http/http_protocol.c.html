modules/http/http_protocol.c

概要

HTTP プロトコル処理の核心を担うソースファイルです。クライアントとの直接的な通信、HTTP ステータスコードの管理、リクエストメソッドのレジストリ管理、および条件付きリクエスト（If-Match 等）の評価ロジックを所有しています。

モジュール内での役割

modules/http サブシステムにおいて、HTTP プロトコルレベルのセマンティクスを実装する最下層のレイヤーです。http_filters.c がフィルタチェーンを介してデータを送受信するのに対し、本ファイルはプロトコル仕様（RFC）に基づく判断と、レスポンス生成の定型処理を提供します。

責務

  • **ステータス管理**: HTTP ステータスコードからステータス行への変換およびエラーレスポンスの生成。
  • **メソッド管理**: 標準および動的に登録された HTTP メソッドの識別とレジストリ管理。
  • **条件付きリクエスト評価**: If-Match, If-Modified-Since 等のヘッダーに基づくキャッシュ制御ロジック。
  • **接続制御**: Keep-Alive の可否判定と接続状態の管理。
  • **コンテンツ制御**: Content-Type や Accept-Ranges ヘッダーの操作。

重要なシンボル

メソッド管理

  • ap_method_register: 新規 HTTP メソッドを動的に登録し、内部 ID を割り当てる。
  • ap_method_number_of: 文字列からメソッド ID を取得。組み込みメソッドは高速な lookup_builtin_method を使用。
  • methods_registry: 動的メソッドを保持する apr_hash_t。

条件付きリクエスト評価

  • ap_meets_conditions: リクエストが条件を満たすか判定するメイン関数。
  • ap_condition_if_match 等: 各種 If-* ヘッダーの評価を行い、AP_CONDITION_STRONG などの列挙値を返す。

レスポンス生成・制御

  • ap_send_error_response: エラー発生時に定型的な HTML エラーページを生成し、フィルタチェーンを再構築する。
  • ap_set_keepalive: Keep-Alive の可否を判定し、必要なヘッダーを headers_out に追加する。
  • status_lines: 全 HTTP ステータスコードと理由句の静的配列。

制御フローとデータフロー

1. **メソッド識別**: ap_method_number_of は、まず lookup_builtin_method で静的解析を行い、見つからない場合に methods_registry を検索する。 2. **条件評価**: ap_meets_conditions は、If-Match -> If-Unmodified-Since -> If-None-Match -> If-Modified-Since -> If-Range の順で評価し、最終的なステータス（HTTP_NOT_MODIFIED 等）を決定する。 3. **エラー処理**: ap_send_error_response は、r->output_filters を r->proto_output_filters にリセットし、insert_error_filter フックを呼び出した後、get_canned_error_string で生成した HTML を出力する。

読者向けメモ

  • **メモリ所有権**: request_rec のプール (r->pool) を多用します。apr_palloc で確保されたメモリはリクエスト終了時に自動解放されるため、永続的なデータ保持には注意が必要です。
  • **パフォーマンス**: status_lines 配列は RESPONSE_CODES 定数に厳密に依存しており、インデックス計算は index_of_response 内の shortcut 配列による定数時間アクセスを前提としています。
  • **設計の意図**: lookup_builtin_method は shilka ツールによる自動生成コードであり、手動でメソッドを追加する際は注意が必要です。
  • **未実装・浅い部分**: ap_send_error_response 内の get_canned_error_string は、セキュリティ上の理由から詳細なエラー情報の表示を制限しており、設定による柔軟な制御は今後の課題としてコメントアウトされています。
