modules/aaa

システム内での役割

modules/aaa は、Apache HTTP Server における認証（Authentication）および認可（Authorization）サブシステムの中心的なディレクトリです。HTTP リクエストに対するユーザーの身元確認と、リソースへのアクセス権限の検証を行うための多様なプロバイダーモジュール群を提供します。

責務

  • **認証プロバイダーの提供**: ファイル、DBM、データベース（DBD）、LDAP、フォームベースなど、多様なバックエンドを用いたユーザー認証の実現。
  • **認可プロバイダーの提供**: ユーザー名、グループ、ホスト名、IP アドレス、および論理演算（AND/OR/NOT）を組み合わせた柔軟なアクセス制御の実現。
  • **認証・認可フレームワークの統合**: mod_authn_core および mod_authz_core を通じた、設定とプロバイダーの抽象化・管理。
  • **互換性の維持**: レガシーなアクセス制御構文（Order, Allow, Deny）のサポート。

このモジュールのファイル

本モジュールは、認証・認可に関する以下の 22 個のソースファイルで構成されています。

  • **認証系**: mod_auth_basic.c, mod_auth_digest.c, mod_auth_form.c, mod_authn_anon.c, mod_authn_core.c, mod_authn_dbd.c, mod_authn_dbm.c, mod_authn_file.c, mod_authn_socache.c, mod_authnz_fcgi.c, mod_authnz_ldap.c
  • **認可系**: mod_authz_core.c, mod_authz_dbd.c, mod_authz_dbm.c, mod_authz_groupfile.c, mod_authz_host.c, mod_authz_owner.c, mod_authz_user.c
  • **互換・その他**: mod_access_compat.c, mod_allowmethods.c および関連ヘッダーファイル。

主要なデータ構造

  • **設定構造体**: 各モジュール固有の _config_rec（例: auth_basic_config_rec, authz_core_dir_conf）。
  • **プロバイダーインターフェース**: authn_provider および authz_provider（認証・認可のフック関数を保持）。
  • **論理ツリー**: authz_section_conf（mod_authz_core における Require ディレクティブの論理構造）。
  • **クライアント状態**: client_entry（mod_auth_digest における共有メモリ上の状態管理）。

主なフロー

1. **認証フェーズ**: mod_authn_core が設定に基づき適切なプロバイダーを呼び出し、ユーザー資格情報を検証。成功時に r->user を設定。 2. **認可フェーズ**: mod_authz_core が Require ディレクティブに従い、登録された認可プロバイダーを再帰的に評価。 3. **キャッシュ連携**: mod_authn_socache 等が認証結果を一時保存し、次回以降の認証処理を高速化。

外部依存

  • **Apache コア**: httpd.h, http_config.h, http_request.h, http_protocol.h
  • **APR**: apr_strings.h, apr_dbm.h, apr_shm.h, apr_network_io.h
  • **その他**: mod_auth.h（認証・認可共通インターフェース）, ap_provider.h, ap_expr.h（式評価エンジン）

内部構成

本モジュールは、コアとなるオーケストレーター（mod_authn_core, mod_authz_core）と、それらにプラグインとして機能する個別の認証・認可モジュール群という構成をとっています。

推奨読書順

1. mod_authn_core.c / mod_authz_core.c（フレームワークの理解） 2. mod_auth_basic.c（標準的な認証フローの理解） 3. mod_authz_host.c（ネットワークベースの認可） 4. mod_auth_digest.c（セキュアな認証方式） 5. mod_authnz_ldap.c（外部バックエンド連携） 6. mod_authz_dbd.c（データベース連携）
