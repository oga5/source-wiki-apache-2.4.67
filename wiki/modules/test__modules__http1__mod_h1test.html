test/modules/http1/mod_h1test

システム内での役割

本モジュールは、Apache HTTP Server の HTTP/1.1 プロトコル処理を検証するためのテスト用モジュールです。ディレクトリ境界に基づき定義されており、主にプロトコルスタックの挙動確認を目的としています。

責務

  • h1test-echo ハンドラによるリクエストのインターセプト。
  • クライアントから送信されたリクエストボディのストリーミング（エコーバック）。
  • HTTP トレーラーヘッダーの伝搬および処理能力の検証。

このモジュールのファイル

  • test/modules/http1/mod_h1test/mod_h1test.c

主要なデータ構造

  • h1test: Apache モジュール定義構造体。
  • h1test_hooks: モジュールフック登録関数。
  • h1test_echo_handler: リクエスト処理のメインハンドラ関数。

主なフロー

1. **登録**: h1test_hooks により h1test_echo_handler がハンドラとして登録される。 2. **判定**: リクエストのハンドラ名が h1test-echo であることを確認。 3. **処理**: チャンク転送エンコーディングを強制し、ap_get_client_block で読み込んだボディを ap_pass_brigade でレスポンスとしてストリーミングする。 4. **検証**: 必要に応じてトレーラーヘッダーを検出し、レスポンスヘッダーへ反映させる。

外部依存

  • **APR (Apache Portable Runtime)**: メモリ管理および I/O 抽象化。
  • **Apache HTTP Server Core**: リクエスト処理およびフィルタリング API (httpd.h, http_protocol.h 等)。
