modules/filters/mod_proxy_html.c

概要

mod_proxy_html は、Apache HTTP Server の出力フィルタとして動作し、HTML コンテンツ内の URL や属性を動的に書き換えるモジュールです。主にリバースプロキシ環境において、バックエンドの HTML 内にある絶対パスやホスト名を、クライアントから見た適切なパスへ変換するために使用されます。libxml2 の HTML パーサを利用してストリーム処理を行います。

モジュール内での役割

modules/filters サブシステムにおいて、HTML コンテンツの構造を解析し、特定のルールに基づいてコンテンツを改変する「コンテンツ変換フィルタ」の役割を担います。mod_xml2enc と密接に連携し、文字コードの正規化やエンコーディング変換をサポートします。

責務

  • **HTML 解析と変換**: libxml2 の SAX インターフェースを利用し、タグや属性、コメント、CDATA を走査して置換処理を行う。
  • **設定管理**: ProxyHTMLURLMap などのディレクティブを通じた置換ルールの保持と、リクエストごとのマージ処理。
  • **フィルタパイプライン制御**: ap_filter_t を用いた出力フィルタの登録、初期化、およびバッファリング。
  • **メタデータ処理**: HTML の <meta http-equiv> タグから HTTP ヘッダーを抽出・設定する機能。

重要なシンボル

設定構造体

  • proxy_html_conf: モジュール設定の保持。置換ルール (urlmap)、Doctype、バッファサイズ、有効化フラグなどを管理。
  • urlmap: 置換ルールの定義。正規表現または文字列マッチング、条件式 (ap_expr) を含む。

実行時コンテキスト

  • saxctxt: フィルタ実行時の状態保持。libxml2 のパーサコンテキスト、出力用バッファ、現在の置換ルールセットを保持。

SAX コールバック (HTML 解析用)

  • pstartElement, pendElement: タグの開始・終了時の処理。属性の書き換えや Doctype の強制適用を行う。
  • pcharacters, pcomment, pcdata: テキストノード、コメント、CDATA セクションの処理。

ユーティリティ

  • normalise: 属性値の正規化（小文字化、パス区切り文字の変換）。
  • interpolate_vars: 設定値内の変数展開（${var} 形式）。
  • fixup_rules: リクエスト時に条件式を評価し、適用可能なルールのみを抽出。

制御フローとデータフロー

1. **初期化**: proxy_html_hooks でフィルタを登録。mod_proxy_html で libxml2 の SAX ハンドラを初期化。 2. **フィルタ起動**: proxy_html_filter が呼び出され、check_filter_init でコンテキスト (saxctxt) を作成。 3. **ストリーム処理**:

  • apr_bucket_brigade からデータを読み込み、htmlParseChunk でパーサに渡す。
  • パーサがタグやテキストを検出すると、登録された SAX コールバックが発火。
  • pstartElement 内で属性値を検査し、urlmap に基づいて置換を実行。
  • dump_content がバッファ内のテキストを置換し、出力バケットへ書き出す。

    • 4. **終了**: EOS (End of Stream) バケット受信時にパーサを解放し、残りのバッファをフラッシュ。

読者向けメモ

  • **メモリ管理**: saxctxt 内のバッファは apr_pool_t に紐付いており、リクエスト終了時に自動解放されます。realloc を行う際は apr_pool_cleanup_register でクリーンアップを登録する設計になっています。
  • **パフォーマンス**: libxml2 のパーサは強力ですが、大規模な HTML ではメモリ消費が増大します。ProxyHTMLBufSize でバッファサイズを調整可能です。
  • **制約**: libxml2 のパーサは HTML の断片を処理するため、完全なドキュメント構造が不明な場合、一部のタグが正しく閉じられない可能性があります。
  • **省略**: libxml2 内部の複雑なパーサ状態遷移や、libsed 関連の低レイヤーな正規表現処理の詳細は、本ファイルでは抽象化されています。
