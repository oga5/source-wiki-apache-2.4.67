ap_thread_current

概念

ap_thread_current は、Apache HTTP Server (httpd-2.4.67) において、現在の実行スレッドに関連するコンテキストや情報を管理・取得するための仕組みです。ヘッダーファイル include/httpd.h で宣言され、コア実装の一部である server/util.c で定義されています。
