DEFAULT_USER

概念

DEFAULT_USER は、Unix系オペレーティングシステム上で動作する Apache HTTP Server（httpd）において、明示的なユーザー設定が指定されていない場合に適用されるデフォルトの実行ユーザーを定義するためのマクロまたは定数です。

なぜ重要か

Unix環境において、Webサーバーは通常、特権ポート（80や443など）をバインドするために root 権限で起動されます。しかし、セキュリティ確保のために起動後は速やかに非特権ユーザーへ権限を放棄する必要があります。DEFAULT_USER は、設定ファイルで明示的なユーザー（User ディレクティブなど）が指定されていない場合に、安全なデフォルトの実行ユーザーを決定するために重要となります。
