start_threads

概念

start_threads は、Apache HTTP Server (httpd) のマルチプロセッシングモジュール (MPM) である event および worker において、クライアントからのリクエストを処理するためのワーカースレッド群を生成・起動する関数です。

なぜ重要か

event および worker MPM は、マルチプロセスとマルチスレッドを組み合わせたハイブリッド型の並行処理モデルを採用しています。start_threads は、各子プロセス内で実際にリクエストを処理するスレッド群を立ち上げる役割を担うため、Webサーバーの並行処理性能とリクエスト処理能力の基盤として極めて重要です。
