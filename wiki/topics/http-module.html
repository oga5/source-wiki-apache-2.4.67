http_module

概念

http_module は、Apache HTTP Server（httpd-2.4.67）において、特定のオペレーティングシステム（NetWareおよびWindows）向けにビルドされる際に、プラットフォーム固有のモジュール構成ファイル内で定義または参照されるシンボルです。

なぜ重要か

このシンボルは、NetWareやWindows（Win32）環境において、Webサーバーのコア機能であるHTTPプロトコル処理モジュールを適切にロード・初期化するために必要となります。プラットフォームごとのビルド構成（modules.c）において、組み込むべきモジュールのリストを管理する上で重要な役割を果たします。
