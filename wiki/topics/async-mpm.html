async_mpm

概念

async_mpm は、Apache HTTP Server (httpd-2.4.67) において、非同期マルチプロセッシングモジュール（MPM: Multi-Processing Module）の有効性や動作状態を判定・制御するために、複数のモジュール間で参照されるシンボルです。

なぜ重要か

HTTP/1.1 接続を処理する modules/http や、HTTP/2 接続を管理する modules/http2 において、使用されている MPM が非同期（Event MPM など）であるか同期（Worker や Prefork など）であるかによって、接続の維持、タイムアウト処理、および I/O イベントのハンドリング方法を切り替える必要があります。ただし、具体的な制御ロジックの詳細は現時点では推論されていません。
