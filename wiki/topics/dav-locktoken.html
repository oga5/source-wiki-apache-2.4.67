dav_locktoken

概念

dav_locktoken は、Apache HTTP Server の WebDAV（mod_dav）モジュール群において、WebDAV ロックを一意に識別するためのトークンを保持するラッパー構造体です。内部的には APR（Apache Portable Runtime）の apr_uuid_t 型の UUID を保持するシンプルなデータコンテナとして定義されています。

なぜ重要か

WebDAV プロトコルでは、リソースの同時編集を防ぐために排他/共有ロックを管理する必要があります。dav_locktoken は、各ロックを一意に識別するための識別子（トークン）として機能し、ロックの作成、検索、比較、解決、および永続化（ファイルシステム等への保存）を行う際の基盤となります。

関与するモジュール

  • modules/dav/fs
  • modules/dav/lock
  • modules/dav/main

主要エンティティ

  • **struct dav_locktoken** (modules/dav/fs/lock.c および modules/dav/lock/locks.c):

    • ``c struct dav_locktoken { apr_uuid_t uuid; }; `` UUID を保持する構造体。

  • **dav_lock_discovery**: 直接ロック情報を保持する構造体。メンバとして dav_locktoken を含みます。
  • **dav_lock_indirect**: 間接ロック情報を保持する構造体。メンバとして dav_locktoken を含みます。
  • **関連関数群** (例: dav_fs_parse_locktoken, dav_fs_format_locktoken, dav_fs_compare_locktoken, dav_fs_load_lock_record): トークンの解析、フォーマット、比較、およびロードを行います。

エンドツーエンドの流れ

WebDAV クライアントからのロック要求やロック確認要求に対する、dav_locktoken を用いた詳細なエンドツーエンドの処理フロー（生成から検証、破棄に至る全体像）は、提供された情報からは完全には明らかになっていません。

現時点で判明している部分的な流れは以下の通りです： 1. ロックの作成や解決のプロセスにおいて、apr_uuid_t を用いて一意な UUID が生成され、dav_locktoken に格納されます。 2. dav_fs_load_lock_record などの関数により、永続化されたロック情報からトークンが構築・比較されます。 3. dav_fs_parse_locktoken や dav_fs_compare_locktoken などの専用関数を介して、トークンの解析や一致判定が行われます。

重要な不変条件

  • **UUID による一意性**: dav_locktoken は apr_uuid_t を介して、システム全体で一意であることが期待されます。
  • **読み取り専用の扱い**: ロック操作の文脈において、この構造体へのポインタは一般的に const（const dav_locktoken *）として扱われます。
  • **データコンテナとしての性質**: 構造体自体は単一の apr_uuid_t uuid フィールドのみを持つ単純なコンテナであり、ロジックは含みません。

よくある落とし穴

  • **直接操作の回避**: dav_locktoken 自体は単なるデータ保持用の構造体です。トークンの生成、解析、比較を行う際は、構造体のメンバを直接操作するのではなく、dav_fs_compare_locktoken などの専用のヘルパー関数を使用する必要があります。
  • **モジュール間の重複定義**: modules/dav/fs/lock.c と modules/dav/lock/locks.c の両方に同名の構造体が定義されているため、モジュール境界を越えて変更を行う場合は、双方の実装および mod_dav のロックプロバイダインターフェースへの影響を慎重に考慮する必要があります。

次に読む場所

  • modules/dav/fs/lock.c: ファイルシステムベースのロック実装における dav_locktoken の定義と、関連するヘルパー関数（dav_fs_parse_locktoken など）の実装。
  • modules/dav/lock/locks.c: 汎用ロックモジュールにおける dav_locktoken の定義。
  • modules/dav/main/mod_dav.h: mod_dav のコアヘッダーにおけるロックプロバイダインターフェースの定義。
