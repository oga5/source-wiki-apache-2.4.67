set_max_workers

概念

set_max_workers は、Apache HTTP Server (httpd-2.4.67) のマルチプロセッシングモジュール（MPM）において、同時に動作可能な最大ワーカー数を設定・制御するための内部関数または設定ハンドラです。このシンボルは、スレッドベースの並行処理モデルを採用している event および worker の両MPMに存在します。
