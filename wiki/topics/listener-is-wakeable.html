listener_is_wakeable

概念

listener_is_wakeable は、Apache HTTP Server (httpd-2.4.67) のマルチプロセッシングモジュール（MPM）である event および worker において定義または使用されているシンボルです。リスナースレッドが外部からのシグナルやイベントによって起床可能（wakeable）な状態にあるかどうかを判定、あるいは制御するために用いられます。
