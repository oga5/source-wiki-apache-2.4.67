listener_is_wakeable

概念

listener_is_wakeable は、Apache HTTP Server のマルチプロセッシングモジュール（MPM）である event および worker において、リスナースレッド（listener thread）がシグナル等によって起床可能な状態にあるかどうかを判定、あるいは管理するためのシンボルです。

なぜ重要か
