worker_thread

概念

worker_thread は、Apache HTTP Server (httpd) のマルチプロセッシングモジュール (MPM) である event および worker において、クライアントからのリクエスト処理を実際に実行するスレッド、またはそのエントリーポイントとなる関数を指します。

なぜ重要か

worker_thread は、Webサーバーの並行処理性能を支える中核要素です。worker MPM や event MPM において、ネットワーク接続の処理やリクエストの解析、レスポンスの生成・送信といった実際の実務処理をマルチスレッドで並行して実行するために不可欠な役割を担っています。
