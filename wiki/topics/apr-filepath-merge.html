apr_filepath_merge

概念

apr_filepath_merge は、Apache Portable Runtime (APR) が提供する、2つのパス（ベースパスと相対パスなど）を安全に結合・マージするための関数です。Apache HTTP Server (httpd-2.4.67) において、ファイルシステム上のパスを解決し、一貫したパス表現を生成するために使用されます。

なぜ重要か

Webサーバーにおいて、リクエストされたURIを物理的なファイルシステム上のパスにマッピングする処理はセキュリティ上極めて重要です。apr_filepath_merge を使用することで、.. などの相対セグメントを適切に処理し、意図しないディレクトリへのアクセス（パス・トラバーサル）を防止しながら、正確な絶対パスを構築できます。

関与するモジュール
