log_virtual_host

概念

log_virtual_host は、Apache HTTP Server (httpd-2.4.67) においてバーチャルホストに関連するログ出力処理を行うためのシンボル（関数）です。このシンボルは、アクセスログのフォーマットを処理するモジュールと、サーバーコアのログ機能を担うモジュールの両方に存在します。

なぜ重要か

複数のドメインやホストを単一のWebサーバーで運用するバーチャルホスト環境において、各ログエントリー（アクセスログやエラーログ）がどのバーチャルホストに関連しているかを正確に識別・記録することは、システムの監視、トラブルシューティング、およびアクセス解析において極めて重要です。

関与するモジュール

  • module:modules/loggers
  • module:server

主要エンティティ

  • symbol:modules/loggers/mod_log_config.c#log_virtual_host：カスタムログフォーマット（例: %v などのフォーマット文字列）に従って、リクエストが送られたバーチャルホスト名をログに出力するためのハンドラ。
  • symbol:server/log.c#log_virtual_host：サーバーコアのログ機構において、バーチャルホスト情報を出力するための内部処理。
