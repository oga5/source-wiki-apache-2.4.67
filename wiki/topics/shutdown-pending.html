shutdown_pending

概念

shutdown_pending は、Apache HTTP Server の特定のマルチプロセッシングモジュール（MPM）において、サーバープロセスまたは子プロセスに対してシャットダウン（停止）要求が送られたことを示すフラグ（シグナル状態を保持する変数）です。

なぜ重要か

このフラグは、OS/2 や NetWare などのプラットフォーム向け MPM において、メインループやリクエスト処理ループの終了条件を制御するために重要です。これがセットされることで、プロセスは安全にクリーンアップ処理へと移行できます。

現時点では、shutdown_pending の状態遷移や排他制御に関する具体的な不変条件は不明です。
