clean_child_exit

概念

clean_child_exit は、Apache HTTP Server (httpd) の各種マルチプロセッシングモジュール (MPM) において、子プロセスまたはワーカースレッドを安全かつクリーンに終了させるための内部関数です。各MPMのアーキテクチャ（プロセスモデルかスレッドモデルかなど）に合わせて個別に実装されており、メモリプールやバケットアロケータなどのリソース解放、スコアボードの更新、シグナル処理、および最終的なプロセス/スレッドの終了処理を担当します。

なぜ重要か

子プロセスやワーカースレッドが終了する際、割り当てられたリソース（メモリプール pchild や ptrans、バケットアロケータ bucket_alloc など）を確実に解放し、スコアボードの状態を正しく更新（例: WORKER_DEAD）しないと、メモリリークやゾンビプロセスの発生、あるいは親プロセスが子プロセスの状態を誤認する原因となります。この関数は、それらのクリーンアップ処理をカプセル化し、安全な終了を保証するために重要です。

エンドツーエンドの流れ

各MPMにおける具体的な呼び出し元から終了に至るエンドツーエンドの完全なフローは、提供された情報からは**未だ完全には判明していません**。ただし、各実装内での処理の流れは以下の通りです。

OS/2 MPM (mpmt_os2_child.c)

1. 子プロセスの生存期間を管理するメモリプール pchild が存在する場合、apr_pool_destroy で破棄します。 2. exit(code) を呼び出してプロセスを終了します。

NetWare MPM (mpm_netware.c)

1. apr_bucket_alloc_destroy を呼び出し、バケットアロケータを破棄します。 2. サーバーが停止中でない場合、トランザクションプール ptrans を解放します。 3. アトミックに worker_thread_count を減算します。 4. worker_num が 0 以上であれば、スコアボード上の当該スレッド状態を WORKER_DEAD に更新します。 5. NXThreadExit を呼び出してスレッドを終了します。

Prefork MPM (prefork.c)

1. MPMの状態を AP_MPMQ_STOPPING に設定します。 2. SIGHUP および SIGTERM シグナルを無視するように設定します。 3. 終了コードが 0 の場合、ap_run_child_stopping フックを実行します。 4. 子プロセス固有のメモリプール pchild が存在する場合、これを破棄します。 5. one_process モードの場合、prefork_note_child_killed を呼び出して終了を記録します。 6. POD (Pipe of Death) を閉じます。 7. chdir_for_gprof を呼び出し、exit(code) でプロセスを終了します。

重要な不変条件

システム全体にわたる統一的な不変条件は**完全には判明していません**が、各MPMの実装において以下の前提条件や不変条件が観察されます。

  • **OS/2**: メモリプール pchild が初期化済みであること（NULL でない場合のみ破棄が実行される）。
  • **NetWare**: worker_num が 0 以上の場合のみ、スコアボードの更新（WORKER_DEAD への遷移）が行われます。また、サーバー停止中 (shutdown_pending が真) の場合は ptrans の破棄をスキップします。
  • **Prefork**: pchild メモリプールが有効な場合にのみ破棄されます。

よくある落とし穴

  • **MPM間での実装の混同**: clean_child_exit は複数のMPMで同名の関数として定義されていますが、シグネチャや内部動作（プロセスを終了させる exit か、スレッドを終了させる NXThreadExit かなど）が大きく異なります。他MPMのコードやドキュメントと混同しないよう注意が必要です。
  • **制御の非返還**: 本関数は呼び出し後に exit() や NXThreadExit() を実行するため、呼び出し元に制御が戻りません。関数の後に記述されたコードは実行されないことに留意する必要があります。
  • **リソースの二重解放やリーク**: shutdown_pending などの状態フラグによって、プール（ptrans など）の破棄をスキップする条件があるため、呼び出し側の状態管理と整合性が取れていないと、リソースリークや不正アクセスを招く可能性があります。
