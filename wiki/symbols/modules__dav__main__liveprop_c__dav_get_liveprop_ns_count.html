dav_get_liveprop_ns_count

定義

  • 種別: function
  • 定義箇所: modules/dav/main/liveprop.c:61

目的

現在登録されているライブプロパティの名前空間（Namespace）の総数を取得します。

シグネチャまたは形状

long dav_get_liveprop_ns_count(void)

入力と出力

  • 入力: なし
  • 出力: 登録済みの名前空間の数を示す long 型の値を返します。

前提条件と不変条件

  • dav_liveprop_count は 1 から始まる連番として管理されており、本関数はその現在の値を返します。
  • この値は XML 出力時の lpN（N はインデックス）といったプレフィックス生成の基準として利用されます。

アルゴリズムまたは振る舞い

静的変数 dav_liveprop_count の値をそのまま返却します。

呼び出し元と呼び出し先

  • 呼び出し元: modules/dav/main/props.c:764 において、marks_liveprop 配列のメモリ確保サイズを決定するために呼び出されています。
  • 呼び出し先: なし（直接的な関数呼び出しは行いません）。

関連データ構造

  • dav_liveprop_count: 本関数が参照する静的変数。
  • apr_pcalloc: 呼び出し元で本関数の戻り値を使用してメモリ確保を行う際に使用されます。

失敗モードと境界条件

  • 失敗モード: なし。
  • 境界条件: ライブプロパティが一つも登録されていない場合、初期値（0 または 1）を返します。

保守担当者向けメモ

  • 本関数は mod_dav のプロパティ処理サブシステムにおいて、名前空間のインデックス管理と密接に関連しています。
  • 戻り値は lpN 形式の XML 属性生成に直結するため、変更時は XML 出力ロジックへの影響を考慮してください。
