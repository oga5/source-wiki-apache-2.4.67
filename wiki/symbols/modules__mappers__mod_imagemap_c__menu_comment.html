menu_comment

定義

  • 種別: function
  • 定義箇所: modules/mappers/mod_imagemap.c:504

目的

画像マップの代替メニュー生成時、設定されたメニュー形式（menu）に応じてコメント行を出力する。

シグネチャまたは形状

static void menu_comment(request_rec *r, char *menu, char *comment)

入力と出力

  • r: リクエストレコード（出力先）。
  • menu: メニューの形式を指定する文字列（formatted, semiformatted, unformatted）。
  • comment: 出力対象のコメント文字列。
  • 出力: なし（ap_rputs または ap_rvputs を介してレスポンスストリームへ書き出す）。

前提条件と不変条件

  • menu および comment は有効な文字列であること。
  • r は有効なリクエストレコードであること。

アルゴリズムまたは振る舞い

1. menu の値に基づき、以下の条件分岐を行う：

  • formatted: 改行のみを出力する。
  • semiformatted: comment が空でない場合、comment と改行を出力する。
  • unformatted: comment が空でない場合、comment と改行を出力する。

    • 2. いずれの条件にも合致しない場合、何も出力しない。

呼び出し元と呼び出し先

  • 呼び出し元: modules/mappers/mod_imagemap.c:653
  • 呼び出し先: ap_rputs, ap_rvputs（Apache HTTP Server API）

関連データ構造

  • request_rec: Apache のリクエスト情報を保持する構造体。

失敗モードと境界条件

  • comment が空文字列の場合、semiformatted および unformatted モードでは何も出力されない。
  • menu に指定された形式以外が渡された場合、何も出力されない。

保守担当者向けメモ

  • 本関数は mod_imagemap のメニュー生成ロジックの一部であり、マップファイル内のコメント行を HTML 出力に反映させる役割を持つ。
  • strcasecmp を使用しているため、menu 引数の大文字小文字は区別されない。
