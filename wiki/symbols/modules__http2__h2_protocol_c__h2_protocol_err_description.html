h2_protocol_err_description

定義

  • 種別: function
  • 定義箇所: modules/http2/h2_protocol.c:78

目的

HTTP/2 エラーコードに対応する人間が読める説明文（文字列）を取得します。

シグネチャまたは形状

const char *h2_protocol_err_description(unsigned int h2_error);

入力と出力

  • **入力**: h2_error (HTTP/2 エラーコードを表す符号なし整数)
  • **出力**: 対応するエラー説明文の文字列。コードが範囲外の場合は "unknown http/2 error code" を返します。

前提条件と不変条件

  • 内部で h2_err_descr 配列を参照します。
  • 入力値 h2_error が配列のインデックスとして有効であるかどうかが境界条件となります。

アルゴリズムまたは振る舞い

1. 入力された h2_error が h2_err_descr 配列のサイズ内にあるかを確認します。 2. 範囲内であれば、該当するインデックスの文字列を返します。 3. 範囲外であれば、デフォルトの文字列 "unknown http/2 error code" を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: modules/http2/h2_session.c:277 にて、エラーコードをログ出力する際に呼び出されています。
  • **呼び出し先**: 本関数は直接的な呼び出し先を持ちません。

関連データ構造

  • h2_err_descr: 本関数が参照する、エラーコードと説明文を保持する静的配列（定義箇所はソース excerpt 外）。

失敗モードと境界条件

  • h2_error が h2_err_descr 配列の要素数以上である場合、エラーコードの解決に失敗し、デフォルトの不明エラーメッセージを返します。

保守担当者向けメモ

  • 本関数は h2_err_descr 配列の構造に依存しています。新しい HTTP/2 エラーコードが追加された場合は、対応する配列定義の更新が必要です。
