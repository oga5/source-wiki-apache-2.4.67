auth_ldap_optype

定義

  • 種別: enum
  • 定義箇所: modules/aaa/mod_authnz_ldap.c:97

目的

LDAP 認可フェーズにおいて実行される操作の種類を識別するために使用されます。

シグネチャまたは形状

enum auth_ldap_optype {
    LDAP_SEARCH,
    LDAP_COMPARE,
    LDAP_COMPARE_AND_SEARCH
};

入力と出力

  • 入力: なし（列挙型定数）
  • 出力: なし（列挙型定数）

前提条件と不変条件

  • LDAP_COMPARE_AND_SEARCH はネストされたグループの処理に使用されます。
  • get_connection_for_authz 関数において、実行する LDAP 操作の種別を特定するための引数として利用されます。

アルゴリズムまたは振る舞い

LDAP サーバーに対する認可リクエストの性質（検索のみか、比較か、あるいはその両方か）を定義し、適切な接続管理やキャッシュ戦略を選択するためのフラグとして機能します。

呼び出し元と呼び出し先

  • 呼び出し元: get_connection_for_authz (modules/aaa/mod_authnz_ldap.c:437) がこの型を引数として受け取ります。
  • 呼び出し先: 情報なし。

関連データ構造

  • util_ldap_connection_t: get_connection_for_authz が返す LDAP 接続構造体。

失敗モードと境界条件

  • 列挙型に含まれない値を渡した場合の挙動は未定義です。

保守担当者向けメモ

  • 本シンボルは mod_authnz_ldap.c 内の認可ロジックにおける操作の抽象化レイヤーです。新しい LDAP 操作タイプを追加する必要がある場合は、この列挙型を拡張し、それを利用する get_connection_for_authz 側のロジックを更新してください。
