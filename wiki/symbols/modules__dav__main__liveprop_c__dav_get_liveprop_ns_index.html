dav_get_liveprop_ns_index

定義

  • 種別: function
  • 定義箇所: modules/dav/main/liveprop.c:56

目的

指定された名前空間 URI に対応するグローバルなライブプロパティ名前空間インデックスを取得します。

シグネチャまたは形状

long dav_get_liveprop_ns_index(const char *uri)

入力と出力

  • **入力**: uri (名前空間を表す文字列)
  • **出力**: 登録済みの名前空間インデックス（long 型）。

前提条件と不変条件

  • dav_liveprop_uris ハッシュテーブルが初期化済みであること。
  • 戻り値のインデックスは、XML 出力時に lp1, lp2 といったプレフィックス生成に使用される 1 から始まる連番です。

アルゴリズムまたは振る舞い

1. 静的変数 dav_liveprop_uris（apr_hash_t）に対し、引数 uri をキーとして検索を行います。 2. ハッシュテーブルから取得した値を long 型にキャストして返却します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/dav/main/liveprop.c:122
  • modules/dav/main/props.c:814
  • **呼び出し先**:
  • apr_hash_get (APR ライブラリ関数)

関連データ構造

  • dav_liveprop_uris: 登録された URI をキー、インデックスを値として保持するグローバルな apr_hash_t。

失敗モードと境界条件

  • 指定された uri がハッシュテーブルに存在しない場合、apr_hash_get の仕様に従い NULL（0）が返されます。

保守担当者向けメモ

  • 本関数は dav_liveprop_uris ハッシュテーブルの読み取り専用アクセスを提供します。
  • インデックスはプロセス生存期間中維持されるため、動的な名前空間の追加・削除が頻繁に行われる環境では、ハッシュテーブルの整合性に注意してください。
