etag_start

定義

  • 種別: function
  • 定義箇所: modules/http/http_etag.c:67

目的

ETag 文字列の生成を開始し、必要に応じて弱識別子（W/）プレフィックスを付与し、引用符（"）で囲むための初期化を行う。

シグネチャまたは形状

static void etag_start(char *etag, const char *weak, char **next)

入力と出力

  • **入力**:
  • etag: ETag 文字列を書き込むためのバッファポインタ。
  • weak: 弱 ETag の場合は "W/"、それ以外は NULL または空文字列。
  • next: 書き込み終了位置を格納するためのポインタのポインタ。
  • **出力**:
  • etag バッファにプレフィックスと開始引用符が書き込まれる。
  • next には、次の書き込み開始位置（開始引用符の直後）が設定される。

前提条件と不変条件

  • etag バッファは、プレフィックスと引用符を書き込むのに十分なサイズが確保されていること。
  • next は有効な char ** 型のポインタであること。

アルゴリズムまたは振る舞い

1. weak が指定されている場合、その文字列を etag バッファの先頭にコピーする。 2. etag バッファの現在の位置に開始引用符 " を書き込む。 3. etag ポインタをインクリメントし、その位置を next に格納して呼び出し元へ返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/http/http_etag.c:197
  • modules/http/http_etag.c:313
  • modules/http/http_etag.c:344
  • **呼び出し先**:
  • なし（標準的なポインタ操作のみ）。

関連データ構造

  • なし。

失敗モードと境界条件

  • etag バッファが不足している場合、バッファオーバーランが発生する可能性がある（呼び出し元で十分な領域を確保する必要がある）。

保守担当者向けメモ

  • 本関数は etag_end と対で使用されることを想定している。
  • weak 引数に NULL を渡すことで、通常の（強力な）ETag 生成を開始できる。
