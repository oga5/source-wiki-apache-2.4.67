log_info

定義

  • 種別: function
  • 定義箇所: test/clients/h2ws.c:82

目的

プログラムの実行状況に関する情報を、冗長モード（verbose）が有効な場合にのみ出力する。

シグネチャまたは形状

static void log_info(const char *where, const char *msg)

入力と出力

  • **入力**:
  • where: ログ出力元を示す文字列。
  • msg: 出力するメッセージ内容。
  • **出力**: なし（戻り値なし）。

前提条件と不変条件

  • グローバル変数 verbose が真（非ゼロ）である場合にのみ、実際のログ出力処理が実行される。

アルゴリズムまたは振る舞い

1. グローバル変数 verbose の値を評価する。 2. verbose が真であれば、log_out 関数を呼び出し、第一引数に "INFO"、第二引数に where、第三引数に msg を渡してログを出力する。 3. verbose が偽であれば、何も行わずに終了する。

呼び出し元と呼び出し先

  • **呼び出し元**: test/clients/h2ws.c:120 にて呼び出しを確認。
  • **呼び出し先**: log_out 関数を呼び出す。

関連データ構造

  • verbose: ログ出力の有効・無効を制御するグローバル変数。

失敗モードと境界条件

  • where または msg に NULL が渡された場合の挙動は、内部で呼び出される log_out 関数の実装に依存する。

保守担当者向けメモ

  • 本関数は log_out のラッパーであり、ログレベルを "INFO" に固定して呼び出すためのユーティリティである。
  • ログ出力の制御を verbose フラグに一元化しているため、デバッグ時にはこのフラグの状態を確認すること。
