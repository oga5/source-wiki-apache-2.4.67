deflate_set_window_size

定義

  • 種別: function
  • 定義箇所: modules/filters/mod_deflate.c:229

目的

Apache HTTP Server の設定ディレクティブ DeflateWindowSize を解析し、deflate_filter_config 構造体のウィンドウサイズ設定を更新する。

シグネチャまたは形状

static const char *deflate_set_window_size(cmd_parms *cmd, void *dummy, const char *arg)

入力と出力

  • **入力**:
  • cmd: 設定コマンドのパラメータ構造体。
  • dummy: 使用されない引数。
  • arg: 設定値として渡される文字列（整数値）。
  • **出力**:
  • 成功時は NULL を返す。
  • 失敗時はエラーメッセージ文字列を返す。

前提条件と不変条件

  • arg は atoi で変換可能な整数である必要がある。
  • 設定されるウィンドウサイズは 1 から 15 の範囲内であること。

アルゴリズムまたは振る舞い

1. ap_get_module_config を使用して、サーバー設定から deflate_filter_config を取得する。 2. arg を整数 i に変換する。 3. i が 1 未満または 15 を超える場合、エラーメッセージを返す。 4. 正常な場合、c->windowSize に -i を代入する（zlib の仕様に合わせるため負の値に変換）。 5. NULL を返して終了する。

呼び出し元と呼び出し先

  • **呼び出し元**: modules/filters/mod_deflate.c:1923 において、AP_INIT_TAKE1 マクロを通じて DeflateWindowSize ディレクティブのハンドラとして登録されている。
  • **呼び出し先**: ap_get_module_config (Apache API), atoi (標準ライブラリ)。

関連データ構造

  • deflate_filter_config: サーバー全体の設定を保持する構造体。windowSize メンバが更新される。

失敗モードと境界条件

  • arg が 1 から 15 の範囲外である場合、"DeflateWindowSize must be between 1 and 15" というエラーメッセージを返し、設定は更新されない。

保守担当者向けメモ

  • c->windowSize に負の値を格納しているのは、zlib の deflateInit2 関数において、ウィンドウサイズを負の値で指定することで zlib ヘッダーとチェックサムを抑制する（raw deflate ストリームを生成する）仕様に対応するためである。
