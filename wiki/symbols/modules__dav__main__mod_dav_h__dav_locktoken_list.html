dav_locktoken_list

定義

  • 種別: struct
  • 定義箇所: modules/dav/main/mod_dav.h:876

目的

WebDAV ロックトークンを保持するための単方向連結リスト構造を提供します。

シグネチャまたは形状

typedef struct dav_locktoken_list
{
    dav_locktoken *locktoken;
    struct dav_locktoken_list *next;
} dav_locktoken_list;

入力と出力

  • locktoken: リストの各要素が保持する dav_locktoken へのポインタ。
  • next: リストの次の要素を指すポインタ。末尾の場合は NULL。

前提条件と不変条件

  • リストの各要素は、メモリ管理のために apr_pool_t を介して割り当てられることが一般的です（例: modules/dav/main/util.c:1823 での apr_pcalloc 使用）。

アルゴリズムまたは振る舞い

  • 連結リストとして機能し、複数のロックトークンを順次走査するために使用されます。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/dav/fs/lock.c
  • modules/dav/lock/locks.c
  • modules/dav/main/mod_dav.c
  • modules/dav/main/util.c
  • **呼び出し先**:
  • dav_locktoken 構造体（locktoken フィールド経由）。

関連データ構造

  • dav_locktoken: 本構造体が保持する個別のロックトークンデータ。

失敗モードと境界条件

  • リストが空の場合、ポインタは NULL となります。
  • メモリ割り当て失敗時は apr_pcalloc 等の Apache APR 側の挙動に依存します。

保守担当者向けメモ

  • 本構造体は mod_dav のロック管理機能において、複数のトークンを扱う際の標準的なコンテナとして利用されています。
  • modules/dav/main/util.c 等でリストの生成や操作が行われているため、変更時はこれらのユーティリティ関数への影響を確認してください。
