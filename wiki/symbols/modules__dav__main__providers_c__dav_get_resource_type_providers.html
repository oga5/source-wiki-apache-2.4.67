dav_get_resource_type_providers

定義

  • 種別: function
  • 定義箇所: modules/dav/main/providers.c:55

目的

指定された名前の DAV リソースタイププロバイダーをグローバルレジストリから取得します。

シグネチャまたは形状

const dav_resource_type_provider *dav_get_resource_type_providers(const char *name)

入力と出力

  • **入力**: name (取得対象のプロバイダー名を示す文字列)
  • **出力**: 該当する dav_resource_type_provider 構造体へのポインタ。見つからない場合は NULL が返されます。

前提条件と不変条件

  • DAV_RESOURCE_TYPE_GROUP で定義されたグループ内にプロバイダーが登録されている必要があります。
  • プロバイダーのバージョンは "0" として固定されています。

アルゴリズムまたは振る舞い

1. Apache の ap_lookup_provider 関数を呼び出します。 2. 第 1 引数に DAV_RESOURCE_TYPE_GROUP、第 2 引数に指定された name、第 3 引数にバージョン "0" を渡します。 3. 検索結果として得られたプロバイダー構造体へのポインタをそのまま返却します。

呼び出し元と呼び出し先

  • **呼び出し元**: modules/dav/main/std_liveprop.c:82 にて呼び出しを確認。
  • **呼び出し先**: Apache Core の ap_lookup_provider を呼び出します。

関連データ構造

  • dav_resource_type_provider: プロバイダーの実装詳細を保持する構造体。
  • DAV_RESOURCE_TYPE_GROUP: プロバイダー検索に使用されるグループ識別子。

失敗モードと境界条件

  • 指定した name に一致するプロバイダーが存在しない場合、ap_lookup_provider の仕様に従い NULL を返します。

保守担当者向けメモ

  • 本関数は ap_provider API の薄いラッパーです。
  • プロバイダーのバージョン管理は "0" に固定されているため、将来的にバージョン管理が必要となった場合は本実装の変更が必要です。
