h2_util_camel_case_header

定義

  • 種別: function
  • 定義箇所: modules/http2/h2_util.c:80

目的

文字列（HTTPヘッダー名など）をキャメルケース形式（例: content-type → Content-Type）に変換します。

シグネチャまたは形状

void h2_util_camel_case_header(char *s, size_t len)

入力と出力

  • s: 変換対象の文字列（インプレースで書き換えられます）。
  • len: 変換対象とする文字列の長さ。
  • 戻り値: なし。

前提条件と不変条件

  • s は少なくとも len バイト以上のメモリ領域を指している必要があります。
  • 文字列内のハイフン (-) を区切り文字として認識します。

アルゴリズムまたは振る舞い

1. 文字列の先頭（インデックス 0）を大文字変換の対象として開始します。 2. 文字列を走査し、以下のルールを適用します。

  • 現在の文字が小文字（'a'-'z'）であり、かつ直前が文字列の開始またはハイフンである場合、大文字に変換します。
  • ハイフン (-) を検出した場合、次の文字を大文字変換の対象としてマークします。

    • 3. 変換対象のマークは、一度変換を行うと解除されます。

呼び出し元と呼び出し先

  • 呼び出し元:
  • modules/http2/h2_stream.c:735
  • modules/http2/h2_util.c:1756
  • 呼び出し先:
  • なし（標準的な文字操作のみを実行）。

関連データ構造

  • 特になし。

失敗モードと境界条件

  • len が 0 の場合、何も行われません。
  • 文字列内にハイフンが含まれない場合、先頭文字のみが変換対象となります。
  • 非アルファベット文字が含まれる場合、それらは変換されずにスキップされます。

保守担当者向けメモ

  • 本関数はインプレースで文字列を操作するため、呼び出し元で渡すバッファが書き換え可能であることを確認してください。
  • h2_util.h に宣言が存在します。
