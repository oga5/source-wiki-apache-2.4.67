is_scriptaliased

定義

  • 種別: function
  • 定義箇所: modules/generators/mod_cgi.c:74

目的

リクエストが「スクリプトエイリアス（ScriptAlias）」によって強制的に CGI スクリプトとして扱われるべき状態にあるかを判定します。

シグネチャまたは形状

static int is_scriptaliased(request_rec *r)

入力と出力

  • **入力**: request_rec *r （現在のリクエスト情報）
  • **出力**: int 型。r->notes テーブル内に "alias-forced-type" というキーが存在し、その値が "cgi-script" と一致する場合に真（非ゼロ）を返します。それ以外は偽（0）を返します。

前提条件と不変条件

  • r->notes テーブルが初期化されている必要があります。
  • 比較は strcasecmp を使用しているため、大文字小文字を区別しない比較が行われます。

アルゴリズムまたは振る舞い

1. apr_table_get を使用して、リクエストノートから "alias-forced-type" の値を取得します。 2. 取得した値が NULL でなく、かつ文字列 "cgi-script" と一致するかを判定します。 3. 両方の条件を満たす場合にのみ、CGI スクリプトとしてエイリアスされていると判断します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/generators/mod_cgi.c:541
  • modules/generators/mod_cgid.c:1507
  • **呼び出し先**:
  • apr_table_get (APR ライブラリ)
  • strcasecmp (標準 C ライブラリ)

関連データ構造

  • request_rec: Apache のリクエスト情報を保持する構造体。r->notes はモジュール間で情報を共有するためのテーブルです。

失敗モードと境界条件

  • r->notes が NULL の場合、apr_table_get の挙動に依存しますが、一般的にセグメンテーション違反のリスクがあります。
  • alias-forced-type キーが存在しない場合、apr_table_get は NULL を返し、関数は正しく 0 を返します。

保守担当者向けメモ

  • 本関数は mod_cgi および mod_cgid の両方から参照されています。
  • mod_vhost_alias.c 内のコメントで本関数の存在が言及されていますが、直接の呼び出し関係はありません。
  • strcasecmp を使用しているため、設定値の表記揺れ（大文字・小文字）に対して堅牢です。
