dav_core_register_uris

定義

  • 種別: function
  • 定義箇所: modules/dav/main/std_liveprop.c:222

目的

DAV: 名前空間に関連付けられた標準ライブプロパティのグループを、DAV エンジンへ登録します。

シグネチャまたは形状

void dav_core_register_uris(apr_pool_t *p)

入力と出力

  • **入力**: p (メモリ割り当てに使用する apr_pool_t へのポインタ)
  • **出力**: なし（戻り値なし）

前提条件と不変条件

  • dav_core_liveprop_group が有効なライブプロパティグループ構造体として定義されていること。
  • apr_pool_t が有効なメモリプールであること。

アルゴリズムまたは振る舞い

1. dav_register_liveprop_group 関数を呼び出します。 2. その際、引数としてメモリプール p と、本ファイルで定義された dav_core_liveprop_group 構造体のアドレスを渡します。 3. これにより、コアライブプロパティの仕様と名前空間が DAV エンジンに登録されます。

呼び出し元と呼び出し先

  • **呼び出し元**: modules/dav/main/mod_dav.c:5190
  • **呼び出し先**: dav_register_liveprop_group (外部関数)

関連データ構造

  • dav_core_liveprop_group: 本関数が登録対象とする、DAV: 名前空間のプロパティ仕様を保持する構造体。
  • apr_pool_t: Apache Portable Runtime のメモリ管理構造体。

失敗モードと境界条件

  • 内部で呼び出される dav_register_liveprop_group が失敗した場合の挙動については、本ソース excerpt に記述がありません。

保守担当者向けメモ

  • 本関数は mod_dav の初期化プロセスの一部として呼び出されます。
  • DAV: 名前空間以外のプロパティを追加・変更する場合は、dav_core_liveprop_group の定義を確認してください。
