dav_resource_type

定義

  • 種別: enum
  • 定義箇所: modules/dav/main/mod_dav.h:296

目的

WebDAV リソースの抽象的な種別を識別するために使用されます。リポジトリ内のリソースが、通常のファイル/コレクションなのか、バージョン管理に関連する特殊なリソース（バージョン、履歴、ワークスペース等）なのかを区別します。

シグネチャまたは形状

typedef enum {
    DAV_RESOURCE_TYPE_UNKNOWN,
    DAV_RESOURCE_TYPE_REGULAR,
    DAV_RESOURCE_TYPE_VERSION,
    DAV_RESOURCE_TYPE_HISTORY,
    DAV_RESOURCE_TYPE_WORKING,
    DAV_RESOURCE_TYPE_WORKSPACE,
    DAV_RESOURCE_TYPE_ACTIVITY,
    DAV_RESOURCE_TYPE_PRIVATE
} dav_resource_type;

入力と出力

  • 本シンボルは列挙型であり、関数ではありません。
  • dav_resource 構造体のメンバとして保持され、リソースの性質を表現する出力として機能します。

前提条件と不変条件

  • dav_resource 構造体内で type メンバとして使用されます。
  • DAV_RESOURCE_TYPE_REGULAR は、通常のファイル、コレクション、バージョン管理されていないリソース、バージョンセレクタ、ベースラインセレクタを包括します。

アルゴリズムまたは振る舞い

  • リソースの種別に基づき、WebDAV メソッド（PROPFIND, PUT 等）の処理ロジックを分岐させるために利用されます。
  • DAV_RESOURCE_TYPE_PRIVATE は、リポジトリ固有の非公開リソースタイプを示します。

呼び出し元と呼び出し先

  • 呼び出し元および呼び出し先の具体的な関数情報は、提供されたソース excerpt には含まれていません。
  • modules/dav/main/mod_dav.h:388 にて dav_resource 構造体のメンバとして定義されています。

関連データ構造

  • dav_resource: 本列挙型を保持する主要な構造体。

失敗モードと境界条件

  • DAV_RESOURCE_TYPE_UNKNOWN は、リソースの種別が特定できない、または未定義の状態を表すために使用されます。

保守担当者向けメモ

  • 新しいリソースタイプを追加する場合は、本列挙型を拡張し、各プロバイダーの実装で適切にハンドリングされているか確認してください。
  • DAV_RESOURCE_TYPE_GROUP という文字列定義（modules/dav/main/mod_dav.h:2662）が存在しますが、これはデバッグやログ出力等のメタデータ用として利用される可能性があります。
