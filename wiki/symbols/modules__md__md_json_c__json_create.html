json_create

定義

  • 種別: function
  • 定義箇所: modules/md/md_json.c:73

目的

APR プールと json_t オブジェクトを紐付け、メモリ管理を自動化した md_json_t インスタンスを生成する。

シグネチャまたは形状

static md_json_t *json_create(apr_pool_t *pool, json_t *j)

入力と出力

  • **入力**:
  • pool: メモリ割り当てに使用する apr_pool_t。
  • j: ラップ対象の json_t オブジェクト。
  • **出力**:
  • 初期化された md_json_t 構造体へのポインタ。

前提条件と不変条件

  • 引数 j が NULL の場合、プールに設定されたアボート関数を呼び出し、APR_ENOMEM を通知する。
  • md_json_t のライフサイクルは、引数として渡された pool の生存期間に依存する。

アルゴリズムまたは振る舞い

1. j が NULL の場合、プールのアボート関数を呼び出し、アサーションで停止する。 2. apr_pcalloc を使用して md_json_t 構造体分のメモリをプールから確保する。 3. 構造体の p メンバに pool を、j メンバに json_t オブジェクトを格納する。 4. apr_pool_cleanup_register を呼び出し、プール破棄時に json_pool_cleanup が実行されるよう登録する。 5. 生成した md_json_t ポインタを返す。

呼び出し元と呼び出し先

  • **呼び出し元**: modules/md/md_json.c 内の以下の箇所から参照されている。
  • md_json_create, md_json_create_s 関連のラッパー関数群（行 94, 99, 113, 118, 487, 503, 522, 1077, 1131, 1170）。
  • **呼び出し先**:
  • apr_pool_abort_get
  • apr_pcalloc
  • apr_pool_cleanup_register
  • json_pool_cleanup (クリーンアップハンドラとして登録)

関連データ構造

  • md_json_t: json_t と apr_pool_t を保持するラッパー構造体。
  • json_t: Jansson ライブラリの JSON オブジェクト型。
  • apr_pool_t: APR のメモリプール。

失敗モードと境界条件

  • j が NULL の場合、アボート関数が設定されていればそれを実行し、そうでなければ assert により異常終了する。
  • メモリ不足時は apr_pcalloc の挙動に従い、通常はプロセスが終了する。

保守担当者向けメモ

  • 本関数は static であり、md_json.c 内の公開 API（md_json_create 等）からのみ呼び出される内部ヘルパーである。
  • json_pool_cleanup が json_decref を適切に呼び出すため、本関数で作成されたオブジェクトの所有権は APR プールに委譲される。
