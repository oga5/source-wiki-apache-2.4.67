usage

定義

  • 種別: function
  • 定義箇所: support/fcgistarter.c:37

目的

コマンドライン引数の指定が誤っている場合に、使用方法（ヘルプメッセージ）を標準エラー出力へ表示し、プログラムを異常終了させる。

シグネチャまたは形状

static void usage(void)

入力と出力

  • 入力: なし（グローバル変数 usage_message を参照）
  • 出力: なし（標準エラー出力への書き込み後、プロセスを終了）

前提条件と不変条件

  • usage_message が有効な文字列として定義されていること。
  • 本関数が呼び出された時点で、プログラムは正常な実行を継続できない状態にあること。

アルゴリズムまたは振る舞い

1. usage_message に格納されたヘルプテキストを stderr に出力する。 2. exit(EXIT_FAILURE) を呼び出し、終了ステータスとして失敗を返してプロセスを終了する。

呼び出し元と呼び出し先

  • 呼び出し元: support/fcgistarter.c 内の main 関数（引数解析失敗時など）。
  • 呼び出し先: fprintf, exit（標準ライブラリ）。
  • 注記: ツールによる解析では、他のファイルからの呼び出しは確認されていません。

関連データ構造

  • usage_message: ヘルプメッセージを保持する文字列定数（本ファイル内で定義）。

失敗モードと境界条件

  • stderr への書き込みが失敗した場合でも、exit(EXIT_FAILURE) によりプロセスは終了する。

保守担当者向けメモ

  • 本関数は fcgistarter ユーティリティ専用です。他のツールと共通化された usage 関数ではないため、修正時は本ファイル内の usage_message の内容と整合性を取ってください。
