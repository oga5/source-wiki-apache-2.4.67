set_compression_lgblock

定義

  • 種別: function
  • 定義箇所: modules/filters/mod_brotli.c:124

目的

Apache HTTP Server の設定ディレクティブ BrotliCompressionMaxInputBlock を処理し、Brotli 圧縮における入力ブロックサイズの対数値をサーバー設定構造体に適用します。

シグネチャまたは形状

static const char *set_compression_lgblock(cmd_parms *cmd, void *dummy, const char *arg)

入力と出力

  • **入力**:
  • cmd: 設定コマンドのパラメータ構造体。
  • dummy: 使用されていません。
  • arg: 設定ファイルから渡される文字列形式の数値。
  • **出力**:
  • 成功時: NULL を返します。
  • 失敗時: エラーメッセージを示す文字列（const char *）を返します。

前提条件と不変条件

  • cmd->server->module_config から取得される brotli_server_config_t 構造体が有効である必要があります。
  • 入力値 arg は整数に変換可能である必要があります。
  • 変換後の値は 16 から 24 の範囲内である必要があります。

アルゴリズムまたは振る舞い

1. ap_get_module_config を使用して、現在のサーバー設定から brotli_server_config_t を取得します。 2. atoi(arg) を呼び出し、引数を整数値 val に変換します。 3. val が 16 未満または 24 を超える場合、エラーメッセージを返して終了します。 4. 範囲内の場合、conf->lgblock に val を代入し、NULL を返して正常終了します。

呼び出し元と呼び出し先

  • **呼び出し元**: modules/filters/mod_brotli.c:589 において、AP_INIT_TAKE1 マクロを通じて BrotliCompressionMaxInputBlock ディレクティブのハンドラとして登録されています。
  • **呼び出し先**:
  • ap_get_module_config (Apache API)
  • atoi (標準ライブラリ)

関連データ構造

  • brotli_server_config_t: サーバー単位の Brotli 設定を保持する構造体。lgblock メンバを更新します。

失敗モードと境界条件

  • arg が 16 から 24 の範囲外である場合、設定の適用に失敗し、エラーメッセージが返されます。
  • atoi による変換が不適切な文字列（数値以外）である場合、val は 0 となり、範囲外エラーとして扱われます。

保守担当者向けメモ

  • 本関数は Brotli エンコーダの lgblock パラメータを制御します。この値は Brotli の入力ブロックサイズ（2^lgblock）を決定する重要なチューニング項目です。
  • 範囲制限（16-24）は Brotli ライブラリの仕様に基づいています。
