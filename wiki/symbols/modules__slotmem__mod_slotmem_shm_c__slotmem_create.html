slotmem_create

定義

  • 種別: function
  • 定義箇所: modules/slotmem/mod_slotmem_shm.c:338

目的

共有メモリ（SHM）上に固定長スロットを管理するためのインスタンスを生成または取得する。

シグネチャまたは形状

apr_status_t slotmem_create(ap_slotmem_instance_t **new,
                            const char *name, apr_size_t item_size,
                            unsigned int item_num,
                            ap_slotmem_type_t type, apr_pool_t *pool)

入力と出力

  • **入力**:
  • name: インスタンスを識別する名前。
  • item_size: 各スロットのサイズ。
  • item_num: スロットの総数。
  • type: AP_SLOTMEM_TYPE_PERSIST を含むフラグ。
  • pool: インスタンス生成に使用するメモリプール。
  • **出力**:
  • new: 生成または取得された ap_slotmem_instance_t へのポインタ。
  • 戻り値: APR_SUCCESS またはエラーコード。

前提条件と不変条件

  • グローバルプール gpool が初期化されている必要がある。
  • 共有メモリのレイアウトは [desc] [num_free] [data_slots] [inuse_array] の順で構成される。
  • AP_SLOTMEM_OFFSET および AP_UNSIGNEDINT_OFFSET により、メタデータとデータ領域のオフセットが厳密に管理される。

アルゴリズムまたは振る舞い

1. slotmem_filenames を使用して、指定された名前から共有メモリのファイルパスを決定する。 2. 既に同名のインスタンスがグローバルリストに存在する場合は、それを再利用する。 3. 存在しない場合、apr_shm_create（または子プロセスであれば apr_shm_attach）を実行して共有メモリを確保する。 4. 確保したメモリの先頭に sharedslotdesc_t を配置し、サイズや個数等のメタデータを初期化する。 5. persist フラグが有効な場合、restore_slotmem を呼び出し、既存の永続化ファイルから状態を復元する。復元失敗時はメモリをゼロクリアする。 6. ap_slotmem_instance_t 構造体を割り当て、共有メモリへのポインタやオフセット計算値を設定する。 7. 成功時、インスタンスをグローバルリストに追加し、new に格納して終了する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/cluster/mod_heartmonitor.c
  • modules/proxy/mod_proxy_balancer.c
  • modules/slotmem/mod_slotmem_plain.c (プロバイダー登録用)
  • modules/slotmem/mod_slotmem_shm.c (プロバイダー登録用)
  • **呼び出し先**:
  • slotmem_filenames, apr_shm_create, apr_shm_attach, apr_shm_remove, apr_shm_baseaddr_get, restore_slotmem, is_child_process, apr_pcalloc, apr_pstrdup

関連データ構造

  • ap_slotmem_instance_t: インスタンス管理構造体。
  • sharedslotdesc_t: スロット構成情報ヘッダー。

失敗モードと境界条件

  • gpool が未初期化の場合、APR_ENOSHMAVAIL を返す。
  • apr_shm_create または apr_shm_attach が失敗した場合、そのエラーコードを返す。
  • 永続化ファイルの復元に失敗した場合、ログを出力し、メモリを初期化した上で処理を続行する。

保守担当者向けメモ

  • 本関数は mod_slotmem_shm のプロバイダーインターフェースの一部であり、ap_slotmem_provider_t を介して利用される。
  • MPMの世代管理（is_child_process）による共有メモリの継承処理が含まれているため、MPM固有の挙動に注意すること。
