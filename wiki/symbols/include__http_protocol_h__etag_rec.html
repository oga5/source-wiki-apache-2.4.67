etag_rec

定義

  • 種別: struct
  • 定義箇所: include/http_protocol.h:177

目的

HTTP レスポンスの ETag（エンティティタグ）を生成するために必要なメタデータを保持する構造体です。

シグネチャまたは形状

struct etag_rec {
    /* 構造体の詳細は include/http_protocol.h:182 以降に定義 */
};

入力と出力

  • 入力: ap_make_etag_ex 関数等の引数として渡され、ETag 生成の元となる情報を提供します。
  • 出力: ap_make_etag_ex を通じて、生成された ETag 文字列が返されます。

前提条件と不変条件

  • ap_make_etag_ex 関数に渡す前に、適切なメタデータで初期化されている必要があります。

アルゴリズムまたは振る舞い

  • ap_make_etag_ex 関数において、この構造体に含まれる情報をもとに ETag を構築します。
  • modules/http/http_etag.c 内の make_digest_etag 等で、この構造体を利用した具体的なダイジェスト生成処理が行われます。

呼び出し元と呼び出し先

  • 呼び出し元（確認された参照先）:
  • modules/dav/fs/repos.c:1888
  • modules/http/http_etag.c:355, 371, 395
  • 呼び出し先:
  • ap_make_etag_ex (この構造体を利用する主要な API)

関連データ構造

  • request_rec: リクエスト処理のコンテキストとして ap_make_etag_ex 等で併用されます。

失敗モードと境界条件

  • 構造体の初期化が不完全な場合、生成される ETag が不正になる可能性があります。具体的なエラーハンドリングの詳細はソース抜粋からは不明です。

  • 構造体のメンバ定義は include/http_protocol.h:182 以降を参照してください。
