rand_choosenum

定義

  • 種別: function
  • 定義箇所: modules/md/md_crypt.c:96

目的

指定された範囲 [l, h] 内の整数をランダムに生成して返します。

シグネチャまたは形状

static int rand_choosenum(int l, int h)

入力と出力

  • **入力**:
  • l (int): 範囲の下限値。
  • h (int): 範囲の上限値。
  • **出力**:
  • int: l 以上 h 以下の範囲内の整数。

前提条件と不変条件

  • rand() 関数が初期化されていることを前提とします。
  • 内部で apr_snprintf を使用して浮動小数点数を文字列に変換し、atoi で整数に戻す処理を行っています。

アルゴリズムまたは振る舞い

1. rand() を使用して 0 から RAND_MAX までの乱数を生成し、それを 0.0 から 1.0 の範囲の浮動小数点数に正規化します。 2. 正規化した値に範囲幅 (h - l) を乗算します。 3. 結果を文字列バッファに %.0f 形式で書き込み、atoi で整数に変換します。 4. 変換後の値に 1 を加算します。 5. 最終的な値が l 未満であれば l に、h を超えていれば h に補正して返します。

呼び出し元と呼び出し先

  • **呼び出し元**: modules/md/md_crypt.c:134 にて呼び出しを確認。
  • **呼び出し先**: apr_snprintf (APRライブラリ), rand (標準ライブラリ), atoi (標準ライブラリ)。

関連データ構造

  • 特になし。

失敗モードと境界条件

  • l > h の場合、補正ロジックにより h が返される可能性があります。
  • rand() の品質に依存するため、暗号論的な強度は保証されません。

保守担当者向けメモ

  • 本関数は rand() を使用した簡易的な乱数生成器です。暗号学的に安全な乱数が必要な場合は、OpenSSL の RAND_bytes 等を利用する md_rand_bytes への置き換えを検討してください。
  • apr_snprintf と atoi を介した変換は非効率的であるため、将来的な最適化の余地があります。
