md_timeperiod_remaining

定義

  • 種別: function
  • 定義箇所: modules/md/md_time.c:47

目的

指定された期間（md_timeperiod_t）に対して、ある特定の時刻から期間終了までの残り時間を算出します。

シグネチャまたは形状

apr_interval_time_t md_timeperiod_remaining(const md_timeperiod_t *period, apr_time_t time)

入力と出力

  • **入力**:
  • period: 期間の開始時刻と終了時刻を保持する md_timeperiod_t 構造体へのポインタ。
  • time: 残り時間を計算するための基準となる時刻（apr_time_t）。
  • **出力**:
  • 基準時刻から期間終了までの残り時間を apr_interval_time_t 型で返します。

前提条件と不変条件

  • period は有効な md_timeperiod_t 構造体を指している必要があります。
  • 期間の終了時刻（period->end）は開始時刻（period->start）以降であることを前提としています。

アルゴリズムまたは振る舞い

1. 基準時刻 time が期間の開始前（time < period->start）の場合、期間全体の長さ（md_timeperiod_length の戻り値）を返します。 2. 基準時刻 time が期間内（period->start <= time < period->end）の場合、period->end - time を計算して返します。 3. 基準時刻 time が期間終了後（time >= period->end）の場合、0 を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/md/md_ocsp.c:436: OCSP レスポンスの有効期限を確認するために呼び出されています。
  • **呼び出し先**:
  • md_timeperiod_length（期間の開始前の場合に呼び出されます）。

関連データ構造

  • md_timeperiod_t: 期間の開始と終了を保持する構造体。
  • apr_interval_time_t: APR で定義される時間間隔を表す型。
  • apr_time_t: APR で定義される絶対時刻を表す型。

失敗モードと境界条件

  • 基準時刻が期間終了時刻と一致する場合、戻り値は 0 となります。
  • 基準時刻が期間開始時刻より前の場合、期間全体の長さが返されます。

保守担当者向けメモ

  • 本関数は modules/md/md_time.h で宣言されており、モジュール内の時間計算の基盤として利用されています。
  • 戻り値の型 apr_interval_time_t はマイクロ秒単位である点に注意してください。
