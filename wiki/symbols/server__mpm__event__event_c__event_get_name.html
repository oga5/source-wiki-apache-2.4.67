event_get_name

定義

  • 種別: function
  • 定義箇所: server/mpm/event/event.c:760

目的

Event MPM モジュールの名称を識別子として返す。

シグネチャまたは形状

static const char *event_get_name(void)

入力と出力

  • 入力: なし
  • 出力: 文字列リテラル "event" へのポインタを返す。

前提条件と不変条件

  • 常に静的な文字列 "event" を返す。
  • 呼び出しによって状態が変化することはない。

アルゴリズムまたは振る舞い

関数が呼び出されると、即座に "event" という文字列定数を返却する。

呼び出し元と呼び出し先

  • 呼び出し元: server/mpm/event/event.c:4013 において、ap_hook_mpm_get_name の引数として登録されている。
  • 呼び出し先: なし。

関連データ構造

なし。

失敗モードと境界条件

  • 失敗モード: なし。常に成功する。
  • 境界条件: なし。

保守担当者向けメモ

  • 本関数は MPM の名称を Apache HTTP Server のフックシステムに提供するために使用される。名称を変更する必要がある場合は、この戻り値を更新すること。
