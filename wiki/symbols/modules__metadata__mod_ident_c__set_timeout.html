set_timeout

定義

  • 種別: function
  • 定義箇所: modules/metadata/mod_ident.c:251

目的

Apache HTTP Server の設定ディレクティブ IdentityCheckTimeout を解析し、IDENT プロトコルによる照会時のタイムアウト値を設定する。

シグネチャまたは形状

static const char *set_timeout(cmd_parms *cmd, void *d_, const char *arg)

入力と出力

  • **入力**:
  • cmd: コマンドパラメータ構造体。
  • d_: 設定構造体（ident_config_rec）へのポインタ。
  • arg: 設定ファイルから渡されるタイムアウト値（秒単位の文字列）。
  • **出力**:
  • 成功時は NULL を返す。

前提条件と不変条件

  • d_ は ident_config_rec 型としてキャスト可能であること。
  • arg は atoi で変換可能な数値文字列であること。

アルゴリズムまたは振る舞い

1. arg を atoi で整数に変換する。 2. apr_time_from_sec を使用して、変換した秒数を APR 内部形式のタイムアウト値に変換し、d->timeout に格納する。 3. d->timeout_unset フラグを 0 に設定し、値が明示的に設定されたことを示す。 4. NULL を返して終了する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/metadata/mod_ident.c:292: AP_INIT_TAKE1 マクロにより IdentityCheckTimeout ディレクティブのハンドラとして登録されている。
  • server/core.c:4588: AP_INIT_TAKE1 マクロにより Timeout ディレクティブのハンドラとして登録されている（※ソースコード上の直接的な参照）。
  • **呼び出し先**:
  • atoi (標準ライブラリ)
  • apr_time_from_sec (APR API)

関連データ構造

  • ident_config_rec: mod_ident のディレクトリ設定を保持する構造体。timeout および timeout_unset メンバを更新する。

失敗モードと境界条件

  • arg が数値でない場合、atoi の仕様に従い 0 が設定される可能性がある。
  • 負の値を指定した場合の動作は apr_time_from_sec の入力値に依存する。

保守担当者向けメモ

  • 本関数は mod_ident 固有の IdentityCheckTimeout だけでなく、server/core.c からも参照されている。設定の共有や意図しない副作用がないか注意すること。
