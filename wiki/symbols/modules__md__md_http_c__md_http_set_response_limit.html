md_http_set_response_limit

定義

  • 種別: function
  • 定義箇所: modules/md/md_http.c:124

目的

HTTP セッションにおけるレスポンスボディの最大許容サイズを設定します。

シグネチャまたは形状

void md_http_set_response_limit(md_http_t *http, apr_off_t resp_limit)

入力と出力

  • http: 設定対象となる md_http_t セッションへのポインタ。
  • resp_limit: 許容するレスポンスサイズ（apr_off_t 型）。
  • 戻り値: なし。

前提条件と不変条件

  • http 引数は有効な md_http_t インスタンスを指している必要があります。

アルゴリズムまたは振る舞い

渡された resp_limit 値を、指定された md_http_t 構造体の resp_limit メンバに代入します。

呼び出し元と呼び出し先

  • 呼び出し元:
  • modules/md/md_acme.c:798
  • modules/md/md_ocsp.c:919
  • 呼び出し先:
  • 提供されたソースコード内には、この関数から呼び出している他の関数はありません。

関連データ構造

  • md_http_t: HTTP クライアントのコンテキストを保持する構造体。本関数はこの構造体の状態を更新します。

失敗モードと境界条件

  • http が NULL の場合、プログラムはクラッシュする可能性があります（ソースコード上に明示的なガード句はありません）。

保守担当者向けメモ

  • 本関数は単なるセッターであり、設定値の妥当性検証は行っていません。必要に応じて呼び出し側で検証を行うか、将来的に本関数内でチェックを追加することを検討してください。
