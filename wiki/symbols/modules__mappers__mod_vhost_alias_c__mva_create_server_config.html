mva_create_server_config

定義

  • 種別: function
  • 定義箇所: modules/mappers/mod_vhost_alias.c:72

目的

mod_vhost_alias モジュールで使用されるサーバー単位の設定構造体（mva_sconf_t）を初期化し、メモリを割り当てる。

シグネチャまたは形状

static void *mva_create_server_config(apr_pool_t *p, server_rec *s)

入力と出力

  • **入力**:
  • p: 設定の割り当てに使用するメモリプール (apr_pool_t *)。
  • s: 設定が適用されるサーバーレコード (server_rec *)。
  • **出力**: 初期化された mva_sconf_t 構造体へのポインタ。

前提条件と不変条件

  • apr_pcalloc により、構造体内の全フィールドはゼロ初期化される。
  • 戻り値の構造体は、doc_root および cgi_root が NULL に、各モードが VHOST_ALIAS_UNSET に設定された状態で返される。

アルゴリズムまたは振る舞い

1. apr_pcalloc を使用して、メモリプール p から mva_sconf_t サイズのメモリを確保する。 2. 確保した構造体の各メンバを初期状態（NULL または VHOST_ALIAS_UNSET）に設定する。 3. 初期化済みの構造体ポインタを返す。

呼び出し元と呼び出し先

  • **呼び出し元**: modules/mappers/mod_vhost_alias.c:452 にて、サーバー設定の作成用として参照されている。
  • **呼び出し先**: apr_pcalloc（APRメモリ管理関数）。

関連データ構造

  • mva_sconf_t: 本関数が初期化するサーバー設定構造体。
  • server_rec: Apache のサーバー設定を保持するコア構造体。

失敗モードと境界条件

  • メモリ不足時には apr_pcalloc が例外を発生させる可能性がある（APRの標準的な挙動に依存）。

保守担当者向けメモ

  • 本関数はサーバーの起動時または設定再読み込み時に呼び出される。
  • mva_sconf_t 構造体に新しい設定項目を追加した場合は、本関数で適切に初期値を設定すること。
