dav_fs_pathname

定義

  • 種別: function
  • 定義箇所: modules/dav/fs/repos.c:233

目的

WebDAV リソースに関連付けられたファイルシステムの物理パスを取得します。

シグネチャまたは形状

const char *dav_fs_pathname(const dav_resource *resource)

入力と出力

  • **入力**: resource (WebDAV リソースへのポインタ)
  • **出力**: リソースに対応するファイルシステムの絶対パスまたは相対パスを示す const char * 型の文字列。

前提条件と不変条件

  • resource および resource->info が有効なメモリ領域を指している必要があります。
  • resource->info は dav_resource_private 構造体であり、pathname メンバが初期化されていることが前提です。

アルゴリズムまたは振る舞い

resource 構造体内のプライベートデータ (info) にアクセスし、その中に格納されている pathname 文字列を直接返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/dav/fs/lock.c: dav_fs_pathname(resource) を複数箇所で呼び出し。
  • modules/dav/fs/repos.c: 内部的に参照。
  • **呼び出し先**:
  • 本関数はデータアクセスのみを行うため、他の関数を呼び出しません。

関連データ構造

  • dav_resource: WebDAV リソースを表現する構造体。
  • dav_resource_private: dav_resource 内の info メンバとして保持される、ファイルシステム固有の情報を格納する構造体。

失敗モードと境界条件

  • resource が NULL の場合、または resource->info が NULL の場合、セグメンテーション違反が発生する可能性があります。

保守担当者向けメモ

  • 本関数は repos.h で宣言されており、mod_dav_fs におけるファイルシステムパス取得の標準的なアクセサです。
  • パス文字列の所有権は resource 構造体が保持するメモリプールに依存するため、呼び出し側で解放してはいけません。
