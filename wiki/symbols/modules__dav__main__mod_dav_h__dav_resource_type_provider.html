dav_resource_type_provider

定義

  • 種別: struct
  • 定義箇所: modules/dav/main/mod_dav.h:2655

目的

WebDAVリソースの型（リソースタイプ）を識別するためのプロバイダーインターフェースを定義します。

シグネチャまたは形状

typedef struct dav_resource_type_provider
{
    int (*get_resource_type)(const dav_resource *resource,
                  const char **name,
                  const char **uri);
} dav_resource_type_provider;

入力と出力

  • get_resource_type:
  • 入力: resource (対象の dav_resource へのポインタ)
  • 出力: name (リソース型の名称を指すポインタへのポインタ)、uri (リソース型に関連付けられたURIを指すポインタへのポインタ)
  • 戻り値: 成功時に 0 を返し、それ以外の場合はエラーコードを返します。

前提条件と不変条件

  • resource は有効な dav_resource 構造体を指している必要があります。
  • name および uri は、結果を受け取るための出力用引数として機能します。

アルゴリズムまたは振る舞い

  • 指定されたリソースに対して、そのリソースが持つ型情報を name および uri に格納します。
  • この構造体は、WebDAVリソースの型情報を動的に解決するためのフックとして機能します。

呼び出し元と呼び出し先

  • 呼び出し元: modules/dav/main/providers.c および modules/dav/main/std_liveprop.c で参照されています。
  • 呼び出し先: 構造体内の関数ポインタ get_resource_type が、具体的なリソース型解決ロジックを実装する関数を指します。

関連データ構造

  • dav_resource: WebDAVリソースの抽象表現。

失敗モードと境界条件

  • get_resource_type が NULL を返す場合や、リソース型の解決に失敗した際の挙動は、実装側の戻り値に依存します。

保守担当者向けメモ

  • 本構造体はプロバイダーインターフェースの一部であり、mod_dav の拡張機能として利用されます。
  • dav_get_resource_type_providers 関数を通じて、名前によるプロバイダーの取得が可能です。
