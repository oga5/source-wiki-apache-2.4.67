dup_trim

定義

  • 種別: function
  • 定義箇所: modules/md/md_result.c:32

目的

文字列を複製し、その内容に含まれる連続する空白文字を単一のスペースに圧縮（collapse）して正規化する。

シグネチャまたは形状

static const char *dup_trim(apr_pool_t *p, const char *s)

入力と出力

  • **入力**:
  • p: メモリ割り当てに使用する apr_pool_t。
  • s: 複製およびトリム対象のソース文字列。
  • **出力**:
  • 複製・正規化された新しい文字列へのポインタ。s が NULL の場合は NULL を返す。

前提条件と不変条件

  • p は有効な apr_pool_t である必要がある。
  • apr_pstrdup がメモリ不足により失敗した場合、NULL が返される。

アルゴリズムまたは振る舞い

1. apr_pstrdup を使用して、指定されたプール p 上に文字列 s の複製を作成する。 2. 複製された文字列が有効な場合、apr_collapse_spaces を呼び出し、文字列内の連続する空白文字を単一のスペースに置換する。 3. 処理済みの文字列を返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/md/md_result.c:105
  • modules/md/md_result.c:117
  • modules/md/md_result.c:219
  • **呼び出し先**:
  • apr_pstrdup (APR)
  • apr_collapse_spaces (APR)

関連データ構造

  • apr_pool_t: メモリ管理に使用される。

失敗モードと境界条件

  • s が NULL の場合、apr_pstrdup は NULL を返し、そのまま NULL が返される。
  • メモリ割り当てに失敗した場合、apr_pstrdup が NULL を返し、後続の apr_collapse_spaces は実行されず、NULL が返される。

保守担当者向けメモ

  • 本関数は md_result_t 構造体内部の文字列フィールドをクリーンアップするために使用される。
  • apr_collapse_spaces は文字列をインプレースで変更するため、apr_pstrdup による複製が必須である。
