dav_locktoken

定義

  • 種別: struct
  • 定義箇所: modules/dav/main/mod_dav.h:428

目的

WebDAV リソースのロックを識別するためのトークンを保持する構造体です。

シグネチャまたは形状

typedef struct dav_locktoken dav_locktoken;

入力と出力

本構造体は、ロック操作においてロックトークンの受け渡しに使用されます。具体的なフィールド構成はヘッダーファイル上では不透明（opaque）として定義されています。

前提条件と不変条件

  • 本構造体は、WebDAV のロック管理機能において、リソースに対する一意な識別子として機能します。
  • 構造体の内部データへの直接アクセスは推奨されず、提供される API を介した操作が想定されています。

アルゴリズムまたは振る舞い

  • ロックトークンの比較、生成、およびリソースへの割り当てに使用されます。
  • mod_dav のロック管理フックにおいて、特定のロック状態を特定するためのキーとして利用されます。

呼び出し元と呼び出し先

  • **呼び出し元**: modules/dav/fs/lock.c 内の複数の関数（ロックの取得、比較、割り当て処理など）で参照されています。
  • **呼び出し先**: 本構造体自体はデータ保持用であり、関数呼び出しの引数や構造体のメンバとして利用されます。

関連データ構造

  • dav_lock: ロック情報を保持する構造体であり、dav_locktoken をメンバとして保持します。

失敗モードと境界条件

  • 構造体のメモリ割り当てに失敗した場合、または無効なトークンが渡された場合の挙動は、利用するプロバイダーの実装に依存します。
  • 構造体内部のデータが初期化されていない状態で比較処理を行うと、予期せぬ動作を引き起こす可能性があります。

保守担当者向けメモ

  • 本構造体は mod_dav.h で不透明な型として宣言されています。具体的な実装は各プロバイダー（mod_dav_fs など）に依存するため、構造体の定義を変更する際は、すべてのプロバイダー実装との互換性に注意してください。
