cmd_err

定義

  • 種別: function
  • 定義箇所: modules/lua/lua_config.c:188

目的

Lua スクリプトから Apache のログシステムに対し、APLOG_ERR レベルでメッセージを出力するためのインターフェースを提供します。

シグネチャまたは形状

static int cmd_err(lua_State *L)

入力と出力

  • **入力**: lua_State *L（Lua 仮想マシンの状態）。スタックにはログ出力対象のメッセージやコンテキスト情報が含まれることが期待されます。
  • **出力**: int（Lua への戻り値の数）。内部で呼び出される cmd_log_at の戻り値をそのまま返します。

前提条件と不変条件

  • L は有効な lua_State である必要があります。
  • 内部で cmd_log_at を呼び出すため、Lua スタック上に適切な引数が配置されている必要があります。

アルゴリズムまたは振る舞い

1. APLOG_ERR をログレベルとして指定し、cmd_log_at 関数を呼び出します。 2. cmd_log_at が実行した結果（通常はスタック操作の結果）を呼び出し元に返します。

呼び出し元と呼び出し先

  • **呼び出し元**: modules/lua/lua_config.c:254 の cmd_methods テーブル内で、Lua からのメソッド呼び出し用として登録されています。
  • **呼び出し先**: cmd_log_at（同ファイル内）。

関連データ構造

  • cmd_methods: CommandParameters 用のメソッドテーブル。本関数はこのテーブルの要素として保持されています。

失敗モードと境界条件

  • Lua スタックが不正な場合、cmd_log_at 経由で luaL_error 等が発生し、Lua 側へ例外が伝播する可能性があります。

保守担当者向けメモ

  • 本関数は cmd_log_at のラッパーです。ログレベルの変更が必要な場合は cmd_log_at の引数を変更してください。
  • APLOGNO マクロは意図的に省略されています。
