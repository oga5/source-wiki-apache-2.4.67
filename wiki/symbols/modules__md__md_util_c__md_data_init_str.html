md_data_init_str

定義

  • 種別: function
  • 定義箇所: modules/md/md_util.c:102

目的

C言語の文字列（const char *）を md_data_t 構造体に初期設定するために使用されます。

シグネチャまたは形状

void md_data_init_str(md_data_t *d, const char *str)

入力と出力

  • **入力**:
  • d: 初期化対象となる md_data_t 構造体へのポインタ。
  • str: 設定する文字列（ヌル終端文字列）。
  • **出力**: なし（戻り値は void）。

前提条件と不変条件

  • str は有効なヌル終端文字列である必要があります。
  • d は書き込み可能なメモリ領域を指している必要があります。

アルゴリズムまたは振る舞い

1. strlen(str) を呼び出し、文字列の長さを取得します。 2. 取得した長さと文字列ポインタを引数として md_data_init を呼び出し、md_data_t 構造体の内部状態を初期化します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/md/md_acme_authz.c:342
  • modules/md/md_acme_authz.c:452
  • modules/md/md_jws.c:107
  • **呼び出し先**:
  • md_data_init (同モジュール内の関数)
  • strlen (標準ライブラリ)

関連データ構造

  • md_data_t: 本関数によって初期化されるデータチャンク管理構造体。

失敗モードと境界条件

  • str が NULL の場合、strlen の呼び出しにより未定義の動作（セグメンテーション違反等）が発生する可能性があります。

保守担当者向けメモ

  • 本関数は md_data_init のラッパーであり、文字列の長さを自動計算する利便性を提供します。
  • md_data_t のメモリ管理方針（所有権や解放タイミング）については、md_data_t の定義および md_data_clear の仕様を参照してください。
