sed_compile_file

定義

  • 種別: function
  • 定義箇所: modules/filters/sed0.c:118

目的

ファイルストリームから sed スクリプトを読み込み、解析して実行可能な内部表現（sed_commands_t）へコンパイルする。

シグネチャまたは形状

apr_status_t sed_compile_file(sed_commands_t *commands, apr_file_t *fin)

入力と出力

  • **入力**:
  • commands: コンパイル結果を格納する sed_commands_t 構造体へのポインタ。
  • fin: スクリプトを読み込むための apr_file_t ファイルポインタ。
  • **出力**:
  • 戻り値: 成功時は APR_SUCCESS、失敗時は APR_EGENERAL を返す。

前提条件と不変条件

  • commands は事前に sed_init_commands 等で初期化されている必要がある。
  • fin は読み込み可能な状態で開かれている必要がある。

アルゴリズムまたは振る舞い

1. 内部関数 fcomp を呼び出し、fin からスクリプトを一行ずつ読み込んで解析を行う。 2. fcomp の戻り値を確認し、0 以外であれば APR_EGENERAL を、0 であれば APR_SUCCESS を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: ソースコード内での直接的な呼び出し元は明示されていない。
  • **呼び出し先**: fcomp（内部関数）。

関連データ構造

  • sed_commands_t: コンパイルされたコマンドチェーンを保持する構造体。
  • apr_file_t: ファイル入力を扱うための APR 構造体。

失敗モードと境界条件

  • fcomp がエラーを返した場合、本関数は APR_EGENERAL を返す。
  • コンパイルエラーの詳細は、sed_commands_t に登録されたエラーハンドラ（errfn）を通じて報告される。

保守担当者向けメモ

  • 本関数は sed_compile_string と対になるファイル入力用のエントリーポイントである。
  • 内部処理の大部分を fcomp に依存しているため、構文解析の修正は fcomp を参照すること。
