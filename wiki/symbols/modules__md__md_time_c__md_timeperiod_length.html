md_timeperiod_length

定義

  • 種別: function
  • 定義箇所: modules/md/md_time.c:26

目的

md_timeperiod_t 構造体で定義された期間の長さを計算し、apr_time_t 型で返します。

シグネチャまたは形状

apr_time_t md_timeperiod_length(const md_timeperiod_t *period)

入力と出力

  • **入力**: period - 期間の開始時刻と終了時刻を保持する md_timeperiod_t 構造体へのポインタ。
  • **出力**: 期間の長さ（apr_time_t）。

前提条件と不変条件

  • period は有効な md_timeperiod_t 構造体を指している必要があります。
  • 期間の開始時刻（start）が終了時刻（end）よりも後の場合、期間は無効とみなされ、0 が返されます。

アルゴリズムまたは振る舞い

1. period->start と period->end を比較します。 2. start < end であれば、end - start を計算してその差分を返します。 3. それ以外（start >= end）の場合は、0 を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/md/md_time.c:49
  • modules/md/md_time.c:295
  • modules/md/mod_md.c:173
  • **呼び出し先**:
  • 本関数は算術演算のみを行い、他の関数を呼び出しません。

関連データ構造

  • md_timeperiod_t: 期間を表現する構造体（start, end を保持）。
  • apr_time_t: APR で定義された時間型。

失敗モードと境界条件

  • period が NULL の場合、ポインタ参照によるクラッシュが発生する可能性があります。
  • start が end と等しい、または start が end を超えている場合、負の値を返さず 0 を返します。

保守担当者向けメモ

  • 本関数は md_timeperiod_t の整合性を検証するものではなく、単なる算術計算を行うユーティリティです。
  • 戻り値の単位は apr_time_t に依存します（通常はマイクロ秒単位）。
