md_http_set_timeout

定義

  • 種別: function
  • 定義箇所: modules/md/md_http.c:134

目的

特定の HTTP リクエストに対して、全体的なタイムアウト値を設定します。

シグネチャまたは形状

void md_http_set_timeout(md_http_request_t *req, apr_time_t timeout)

入力と出力

  • req: タイムアウトを設定する対象の md_http_request_t 構造体へのポインタ。
  • timeout: 設定するタイムアウト時間（apr_time_t 型）。
  • 戻り値: なし。

前提条件と不変条件

  • req は有効な md_http_request_t 構造体を指している必要があります。

アルゴリズムまたは振る舞い

  • 引数として渡された timeout 値を、req->timeout.overall フィールドに代入します。

呼び出し元と呼び出し先

  • 呼び出し元: 提供されたソース情報からは特定できません。
  • 呼び出し先: なし。
  • 宣言: modules/md/md_http.h:104 にて宣言されています。

関連データ構造

  • md_http_request_t: HTTP リクエストの状態を保持する構造体。
  • apr_time_t: APR で定義された時間表現型。

失敗モードと境界条件

  • req が NULL の場合、プログラムは異常終了（セグメンテーション違反）する可能性があります。

保守担当者向けメモ

  • 本関数はリクエストごとのタイムアウト設定を上書きします。接続タイムアウトの設定については md_http_set_connect_timeout を参照してください。
