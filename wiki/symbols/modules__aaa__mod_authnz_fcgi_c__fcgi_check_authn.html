fcgi_check_authn

定義

  • 種別: function
  • 定義箇所: modules/aaa/mod_authnz_fcgi.c:806

目的

Apache HTTP Server の認証フェーズにおいて、FastCGI プロトコルを用いた外部認証プロバイダーを呼び出し、リクエストの認証および認可判定を行う。

シグネチャまたは形状

static int fcgi_check_authn(request_rec *r)

入力と出力

  • **入力**: request_rec *r（現在のリクエスト情報）。
  • **出力**: int（Apache の認証結果ステータスコード）。
  • OK: 認証成功。
  • DECLINED: 認証をスキップ。
  • HTTP_INTERNAL_SERVER_ERROR: 設定不備や通信エラー。
  • その他: バックエンドが返した HTTP ステータスコード。

前提条件と不変条件

  • dconf->name が設定されていること。
  • fcgi_authn_providers ハッシュテーブルに、指定されたプロバイダー名に対応する fcgi_provider_conf が存在すること。
  • Basic 認証の場合、ap_get_basic_auth_pw によりパスワードが取得可能であること（必須ではない）。

アルゴリズムまたは振る舞い

1. **初期チェック**: ディレクトリ設定からプロバイダー名を取得し、未設定または "None" の場合は DECLINED を返す。 2. **認証情報取得**: Basic 認証の場合、ap_get_basic_auth_pw でパスワードを抽出する。 3. **プロバイダー検索**: fcgi_authn_providers から設定情報を取得し、存在しない場合は HTTP_INTERNAL_SERVER_ERROR を返す。 4. **通信**: req_rsp を呼び出し、FastCGI AUTHORIZER ロールでバックエンドと通信する。 5. **ユーザー設定**: 認証成功時、dconf->user_expr があれば式を評価して r->user を設定する。 6. **認可連携**: プロバイダーが認可も兼ねる場合、fcgi_request_notes をリクエスト設定に保存する。 7. **エラー処理**: 認証失敗時、非権威的（non-authoritative）なプロバイダーであれば警告をログし、権威的であればバックエンドのレスポンスをクライアントへ転送する。

呼び出し元と呼び出し先

  • **呼び出し元**: ap_hook_check_authn に登録され、Apache の認証フックから呼び出される。
  • **呼び出し先**:
  • ap_get_module_config, ap_auth_type, ap_get_basic_auth_pw, ap_expr_str_exec, ap_custom_response (Apache API)
  • apr_hash_get, apr_palloc, apr_pstrdup (APR)
  • req_rsp (本モジュール内)
  • log_provider_info (本モジュール内)

関連データ構造

  • fcgi_dir_conf: ディレクトリごとの設定（プロバイダー名、式など）。
  • fcgi_provider_conf: バックエンド接続情報。
  • fcgi_request_notes: 認可結果を保持する構造体。
  • fcgi_authn_providers: プロバイダーを管理するグローバルなハッシュテーブル。

失敗モードと境界条件

  • **プロバイダー未定義**: HTTP_INTERNAL_SERVER_ERROR を返す。
  • **ユーザー式評価失敗**: r->status を HTTP_INTERNAL_SERVER_ERROR に設定。
  • **レスポンスバッファ溢れ**: rspbuf のサイズ制限により、警告ログを出力しつつ切り詰め処理を行う。
  • **非権威的プロバイダー**: 認証失敗時にレスポンスボディを無視する。

保守担当者向けメモ

  • req_rsp は temp_pool を使用してメモリを管理しているため、本関数内でのメモリ割り当ては r->pool を使用すること。
  • handle_response での再帰的な読み込み処理（goto recv_again）が req_rsp 経由で間接的に行われる点に注意。
  • AF_UNIX ソケットは未サポートであり、TCP 通信のみを前提としている。
