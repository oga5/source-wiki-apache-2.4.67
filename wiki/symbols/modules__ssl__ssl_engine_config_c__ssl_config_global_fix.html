ssl_config_global_fix

定義

  • 種別: function
  • 定義箇所: modules/ssl/ssl_engine_config.c:96

目的

グローバルな SSL 設定構造体に対して、設定が確定（固定）されたことを示すフラグを設定する。

シグネチャまたは形状

void ssl_config_global_fix(SSLModConfigRec *mc)

入力と出力

  • **入力**: mc (SSLModConfigRec *) - 変更対象となるグローバル設定構造体へのポインタ。
  • **出力**: なし（副作用として mc->bFixed を更新）。

前提条件と不変条件

  • 引数 mc は有効な SSLModConfigRec 構造体を指している必要がある。

アルゴリズムまたは振る舞い

1. 引数として受け取った SSLModConfigRec 構造体の bFixed メンバに TRUE を代入する。

呼び出し元と呼び出し先

  • **呼び出し元**: modules/ssl/ssl_engine_init.c:378
  • **呼び出し先**: なし（直接的な関数呼び出しは含まない）。

関連データ構造

  • SSLModConfigRec: mod_ssl のグローバル設定を保持する構造体（ssl_private.h で定義）。

失敗モードと境界条件

  • mc が NULL の場合、セグメンテーション違反が発生する可能性がある。

保守担当者向けメモ

  • 本関数は設定の変更を禁止するフラグを立てる役割を持つ。設定の読み込み完了後に呼び出されることを想定している。
