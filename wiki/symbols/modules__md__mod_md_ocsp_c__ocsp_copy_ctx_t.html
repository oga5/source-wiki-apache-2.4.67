ocsp_copy_ctx_t

定義

  • 種別: struct
  • 定義箇所: modules/md/mod_md_ocsp.c:103

目的

OCSP レスポンスの DER 形式データとそのサイズを保持し、転送やコピーを行うためのコンテキスト構造体です。

シグネチャまたは形状

typedef struct {
    unsigned char *der;
    apr_size_t der_len;
} ocsp_copy_ctx_t;

入力と出力

  • der: OCSP レスポンスの DER エンコードされたバイト列へのポインタ。
  • der_len: der が指すデータのバイト長。

前提条件と不変条件

  • der が指すメモリ領域は、本構造体の生存期間中、有効である必要があります。
  • der_len は der の実際のデータサイズと一致している必要があります。

アルゴリズムまたは振る舞い

本構造体は、OCSP レスポンスデータを一時的に保持し、別のコンテキストや関数へ受け渡すためのデータコンテナとして機能します。

呼び出し元と呼び出し先

  • 呼び出し元および呼び出し先の具体的な関数情報は、提供されたソース excerpt には含まれていません。

関連データ構造

  • md_ocsp_ctx_t: mod_md_ocsp.c 内で定義される、Watchdog 実行コンテキストを保持する構造体。

失敗モードと境界条件

  • der が NULL の場合、または der_len が 0 の場合、データが空であることを示します。これに対する具体的なエラーハンドリングは、本構造体を使用する側の実装に依存します。

保守担当者向けメモ

  • 本構造体は mod_ssl とのインターフェース（ap_ssl_ocsp_copy_resp 等）で受け渡されるデータの受け皿として利用されることが想定されます。
  • メモリ管理（確保・解放）の責任範囲が本構造体の外部にあるか内部にあるかについては、使用箇所を確認する必要があります。
