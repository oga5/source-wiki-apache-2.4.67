dav_core_insert_all_liveprops

定義

  • 種別: function
  • 定義箇所: modules/dav/main/std_liveprop.c:212

目的

リソースに関連付けられたすべての標準的なライブプロパティ（DAV: 名前空間）を、XML レスポンス生成用のヘッダー構造体へ挿入します。

シグネチャまたは形状

void dav_core_insert_all_liveprops(
    request_rec *r,
    const dav_resource *resource,
    dav_prop_insert what,
    apr_text_header *phdr)

入力と出力

  • r: 現在のリクエストレコード。
  • resource: プロパティを取得する対象の DAV リソース。
  • what: プロパティ挿入のモード（dav_prop_insert 型）。
  • phdr: プロパティの XML 文字列を蓄積する apr_text_header 構造体（出力）。

前提条件と不変条件

  • resource が有効な dav_resource 構造体を指していること。
  • phdr が初期化済みであること。

アルゴリズムまたは振る舞い

内部的に dav_core_insert_prop を呼び出し、DAV_PROPID_resourcetype プロパティを phdr に追加します。現状の実装では、すべてのライブプロパティを網羅するのではなく、resourcetype の挿入に特化しています。

呼び出し元と呼び出し先

  • **呼び出し元**: modules/dav/main/mod_dav.c:5182 にて dav_hook_insert_all_liveprops に登録されています。
  • **呼び出し先**: dav_core_insert_prop を呼び出します。

関連データ構造

  • dav_resource: DAV リソースの抽象表現。
  • apr_text_header: XML 生成時に使用されるテキストバッファ管理構造体。
  • dav_prop_insert: プロパティ挿入の挙動を制御する列挙型。

失敗モードと境界条件

  • dav_core_insert_prop が失敗した場合の戻り値は無視されます。
  • resource が NULL の場合の挙動は未定義です。

保守担当者向けメモ

  • 本関数は DAV: 名前空間のプロパティを扱うコア実装の一部です。
  • 現在は resourcetype のみを挿入対象としていますが、将来的に他の標準プロパティを追加する場合は dav_core_insert_prop の呼び出しを拡張してください。
