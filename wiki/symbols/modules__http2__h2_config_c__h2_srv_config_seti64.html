h2_srv_config_seti64

定義

  • 種別: function
  • 定義箇所: modules/http2/h2_config.c:407

目的

サーバー設定構造体（h2_config）内の 64 ビット整数型設定値を更新します。

シグネチャまたは形状

static void h2_srv_config_seti64(h2_config *conf, h2_config_var_t var, apr_int64_t val)

入力と出力

  • conf: 更新対象の h2_config 構造体へのポインタ。
  • var: 更新する設定項目を指定する h2_config_var_t 型の列挙値。
  • val: 設定する 64 ビット整数値（apr_int64_t）。
  • 戻り値: なし（void）。

前提条件と不変条件

  • conf は有効な h2_config 構造体を指している必要があります。
  • var に指定された項目が 64 ビット整数として定義されている場合、対応する構造体メンバが更新されます。

アルゴリズムまたは振る舞い

1. var の値に基づき switch 文で分岐します。 2. H2_CONF_TLS_WARMUP_SIZE、H2_CONF_STREAM_TIMEOUT、H2_CONF_MAX_WORKER_IDLE_LIMIT のいずれかであれば、H2_CONFIG_SET マクロを使用して conf 内の対応するメンバを val で更新します。 3. 上記以外の項目については、h2_srv_config_seti を呼び出し、val を int にキャストして処理を委譲します。

呼び出し元と呼び出し先

  • 呼び出し元: modules/http2/h2_config.c:500 での呼び出しが確認されています。
  • 呼び出し先: h2_srv_config_seti（デフォルトケース時）。

関連データ構造

  • h2_config: サーバー設定を保持する構造体。
  • h2_config_var_t: 設定項目を識別する列挙型。

失敗モードと境界条件

  • default ケースにおいて val を int にキャストするため、64 ビット整数値が int の範囲を超える場合、値の切り詰めが発生する可能性があります。

保守担当者向けメモ

  • 本関数は h2_config 構造体への書き込みを行う低レベルなセッターです。新しい設定項目を追加する際は、h2_config_var_t の定義と本関数の switch 文の両方を更新する必要があります。
