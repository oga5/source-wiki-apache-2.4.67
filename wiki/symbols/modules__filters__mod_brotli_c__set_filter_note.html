set_filter_note

定義

  • 種別: function
  • 定義箇所: modules/filters/mod_brotli.c:66

目的

httpd.conf で指定された BrotliFilterNote ディレクティブを解析し、圧縮統計情報を格納するリクエストノートのキー名をサーバー設定構造体（brotli_server_config_t）に設定する。

シグネチャまたは形状

static const char *set_filter_note(cmd_parms *cmd, void *dummy,
                                   const char *arg1, const char *arg2)

入力と出力

  • **入力**:
  • cmd: 設定コマンドのパラメータ構造体。
  • dummy: 未使用。
  • arg1: ノートのタイプ（"Ratio", "Input", "Output"）またはノート名。
  • arg2: ノート名（arg1 がタイプの場合）。
  • **出力**:
  • 成功時は NULL を返す。
  • 不正なタイプが指定された場合は、エラーメッセージを含む文字列を返す。

前提条件と不変条件

  • cmd->server->module_config から brotli_module に対応する brotli_server_config_t が取得可能であること。

アルゴリズムまたは振る舞い

1. ap_get_module_config を使用してサーバー設定を取得する。 2. arg2 が指定されていない場合、arg1 を note_ratio_name として設定する。 3. arg2 が指定されている場合、arg1 を大文字小文字を区別せずに比較し、以下のいずれかに該当すれば対応するフィールドに arg2 を代入する。

  • "Ratio": note_ratio_name
  • "Input": note_input_name
  • "Output": note_output_name

    • 4. 上記のいずれにも該当しない場合、apr_psprintf を使用してエラーメッセージを生成し返す。

呼び出し元と呼び出し先

  • **呼び出し元**: modules/filters/mod_brotli.c:578 において、AP_INIT_TAKE12 マクロにより BrotliFilterNote ディレクティブのハンドラとして登録されている。
  • **呼び出し先**:
  • ap_get_module_config (Apache API)
  • ap_cstr_casecmp (Apache API)
  • apr_psprintf (APR API)

関連データ構造

  • brotli_server_config_t: サーバー単位の設定を保持する構造体。本関数はこの構造体の note_ratio_name, note_input_name, note_output_name フィールドを更新する。

失敗モードと境界条件

  • arg1 に "Ratio", "Input", "Output" 以外の文字列が指定された場合、設定エラーとしてメッセージを返す。

保守担当者向けメモ

  • 本関数は mod_deflate の同名機能と類似したインターフェースを提供している。
  • 設定値は cmd->pool を介してメモリ管理されるため、個別のメモリ解放は不要である。
