dav_fs_ensure_state_dir

定義

  • 種別: function
  • 定義箇所: modules/dav/fs/dbm.c:119

目的

指定されたディレクトリ内に、WebDAV の状態管理用サブディレクトリ（.DAV）が存在しない場合に作成します。

シグネチャまたは形状

void dav_fs_ensure_state_dir(apr_pool_t *p, const char *dirname)

入力と出力

  • p: メモリ割り当てに使用する apr_pool_t。
  • dirname: 状態ディレクトリを作成する対象の親ディレクトリパス。
  • 出力: なし（戻り値なし）。

前提条件と不変条件

  • dirname は有効なパス文字列である必要があります。
  • DAV_FS_STATE_DIR マクロで定義された名前のディレクトリが作成対象となります。

アルゴリズムまたは振る舞い

1. apr_pstrcat を使用して、dirname に DAV_FS_STATE_DIR を結合したフルパスを生成します。 2. apr_dir_make を呼び出し、APR_OS_DEFAULT 権限でディレクトリの作成を試みます。 3. 作成結果（成功・失敗）に関わらず、エラーは無視されます。

呼び出し元と呼び出し先

  • 呼び出し元:
  • modules/dav/fs/dbm.c:197
  • modules/dav/fs/lock.c:821
  • 呼び出し先:
  • apr_pstrcat (APR)
  • apr_dir_make (APR)

関連データ構造

  • apr_pool_t: メモリ管理に使用。

失敗モードと境界条件

  • ディレクトリ作成に失敗した場合（既に存在する場合や権限不足など）、エラーは明示的に無視されるため、呼び出し元には通知されません。
  • umask の影響については、ソースコード内に考慮が必要かどうかの検討コメントが残されています。

保守担当者向けメモ

  • 本関数はエラーを握りつぶす設計であるため、ディレクトリ作成が必須である処理の直後で利用する場合は、別途ディレクトリの存在確認が必要になる可能性があります。
