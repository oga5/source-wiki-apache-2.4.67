ap_gname2id

定義

  • 種別: function
  • 定義箇所: server/mpm_common.c:249

目的

グループ名（文字列）をシステム上のグループID（gid_t）に変換します。

シグネチャまたは形状

gid_t ap_gname2id(const char *name)

入力と出力

  • **入力**: name (グループ名を表す文字列)
  • **出力**: 成功時は対応する gid_t 値を返します。

前提条件と不変条件

  • name が # で始まる場合、その後の文字列を数値として解釈し、直接 gid_t に変換します。
  • name が # で始まらない場合、システム関数 getgrnam() を使用してグループ情報を検索します。

アルゴリズムまたは振る舞い

1. 入力文字列の先頭が # であるか確認します。 2. # で始まる場合、atoi() を用いて数値へ変換し、その値を返します。 3. # で始まらない場合、getgrnam() を呼び出し、システムからグループ情報を取得します。 4. getgrnam() が失敗（NULL を返却）した場合、エラーログを出力し、exit(1) を呼び出してプロセスを終了します。 5. 成功した場合、取得した struct group 構造体から gr_gid を抽出して返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/arch/unix/mod_privileges.c:434
  • modules/arch/unix/mod_unixd.c:244, 302
  • modules/generators/mod_suexec.c:75
  • **呼び出し先**:
  • getgrnam() (システムライブラリ)
  • atoi() (標準ライブラリ)
  • ap_log_error() (Apache ログ出力関数)
  • exit() (標準ライブラリ)

関連データ構造

  • struct group (POSIX 定義のグループ情報構造体)

失敗モードと境界条件

  • **失敗モード**: getgrnam() が指定されたグループ名を見つけられない場合、エラーログを出力してサーバープロセスを即座に終了させます。
  • **境界条件**: # で始まる数値指定の場合、atoi() の挙動に依存します。

保守担当者向けメモ

  • 本関数はプロセス起動時の設定読み込みフェーズで利用されることが想定されています。
  • 失敗時に exit(1) を呼び出すため、実行中のサーバープロセスを停止させる副作用がある点に注意してください。
