mkconfig

定義

  • 種別: function
  • 定義箇所: modules/generators/mod_suexec.c:33

目的

suexec_config_t 構造体のメモリを割り当て、初期状態として active フラグを無効（0）に設定する。

シグネチャまたは形状

static void *mkconfig(apr_pool_t *p)

入力と出力

  • **入力**: apr_pool_t *p（メモリ割り当てに使用する APR プール）
  • **出力**: 初期化済みの suexec_config_t 構造体へのポインタ

前提条件と不変条件

  • 指定されたプール p が有効であること。
  • 戻り値の active メンバは常に 0 で初期化される。

アルゴリズムまたは振る舞い

1. apr_palloc を使用して、プール p から suexec_config_t サイズのメモリブロックを確保する。 2. 確保した構造体の active メンバに 0 を代入する。 3. 初期化された構造体へのポインタを返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/generators/mod_suexec.c:47
  • modules/generators/mod_suexec.c:55
  • modules/mappers/mod_speling.c:89
  • modules/mappers/mod_speling.c:97
  • **呼び出し先**: apr_palloc

関連データ構造

  • suexec_config_t: 本関数が初期化対象とする設定構造体。

失敗モードと境界条件

  • apr_palloc がメモリ不足により失敗した場合、APR の仕様に従いプロセスが終了する可能性がある。

保守担当者向けメモ

  • 本関数は mod_suexec 以外のモジュール（mod_speling）からも参照されているが、これは mod_suexec.c 内の関数が静的スコープであるため、各ファイルで個別に定義されているか、あるいはツールによる参照判定が重複している可能性がある。実装の共有状況については注意が必要である。
