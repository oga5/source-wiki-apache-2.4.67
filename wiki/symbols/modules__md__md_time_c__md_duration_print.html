md_duration_print

定義

  • 種別: function
  • 定義箇所: modules/md/md_time.c:108

目的

期間（apr_interval_time_t）を人間が読みやすい形式の文字列に変換します。

シグネチャまたは形状

const char *md_duration_print(apr_pool_t *p, apr_interval_time_t duration)

入力と出力

  • **入力**:
  • p: メモリ割り当てに使用する apr_pool_t。
  • duration: 変換対象の期間（マイクロ秒単位の apr_interval_time_t）。
  • **出力**:
  • 変換された期間を表す文字列（例: "3 days"）。メモリは指定されたプール p から割り当てられます。

前提条件と不変条件

  • p は有効な apr_pool_t である必要があります。
  • 内部で duration_print を呼び出すため、その関数が要求する前提条件に準拠します。

アルゴリズムまたは振る舞い

1. 内部関数 duration_print(p, 0, duration) を呼び出します。 2. 第2引数に 0 を渡すことで、デフォルトのフォーマット設定で期間の文字列化を実行します。

呼び出し元と呼び出し先

  • **呼び出し先**: duration_print（同一ファイル内）
  • **呼び出し元**:
  • modules/md/mod_md_drive.c: 証明書の更新タイミングやジョブの次回の実行予定時刻を表示する際に使用。
  • modules/md/mod_md_ocsp.c: OCSP の待機時間を表示する際に使用。

関連データ構造

  • apr_interval_time_t: APR で定義される時間間隔を表す型。
  • apr_pool_t: メモリ管理のための APR プール。

失敗モードと境界条件

  • メモリ不足時には apr_pool_t の挙動に従い、アロケーションエラーが発生する可能性があります。
  • 負の期間が渡された場合の具体的な出力形式については、duration_print の実装に依存します。

保守担当者向けメモ

  • 本関数は duration_print のラッパーです。詳細なフォーマットロジックを変更する場合は duration_print を参照してください。
  • md_time.h で宣言されており、モジュール外からも利用可能な公開インターフェースです。
