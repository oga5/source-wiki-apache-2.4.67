md_pkey_rsa_params_t

定義

  • 種別: struct
  • 定義箇所: modules/md/md_crypt.h:55

目的

RSA 鍵を生成する際のパラメータ（ビット長）を保持するための構造体です。

シグネチャまたは形状

typedef struct md_pkey_rsa_params_t {
    apr_uint32_t bits;
} md_pkey_rsa_params_t;

入力と出力

  • **入力**: bits フィールドに RSA 鍵のビット長（例: 2048, 4096）を格納します。
  • **出力**: なし（本構造体は設定値の保持に使用されます）。

前提条件と不変条件

  • bits は有効な RSA 鍵長である必要があります。
  • 本構造体は md_pkey_spec_t 構造体の一部として利用されることが前提です。

アルゴリズムまたは振る舞い

RSA 鍵生成時に、指定された bits 値を OpenSSL の鍵生成関数へ渡すための設定コンテナとして機能します。

呼び出し元と呼び出し先

  • **呼び出し元**: modules/md/md_crypt.h:66 にて md_pkey_spec_t 構造体のメンバ rsa として参照されています。
  • **呼び出し先**: 本構造体自体はデータ保持用であり、直接関数を呼び出すことはありません。

関連データ構造

  • md_pkey_spec_t: RSA 鍵生成の仕様全体を定義する構造体であり、本構造体をメンバとして保持します。

失敗モードと境界条件

  • 構造体自体に失敗モードはありませんが、保持する bits の値が OpenSSL のサポート範囲外である場合、後続の鍵生成処理（md_pkey_gen 等）が失敗する可能性があります。

保守担当者向けメモ

  • 本構造体は md_crypt.h 内で定義されており、OpenSSL の実装詳細を隠蔽する mod_md の抽象化レイヤーの一部です。
  • 構造体の拡張が必要な場合は、md_pkey_spec_t との整合性に注意してください。
