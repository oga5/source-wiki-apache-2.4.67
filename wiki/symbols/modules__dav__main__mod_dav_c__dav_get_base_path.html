dav_get_base_path

定義

  • 種別: function
  • 定義箇所: modules/dav/main/mod_dav.c:253

目的

現在のリクエストに関連付けられた WebDAV のベースパス（base）を取得します。

シグネチャまたは形状

const char *dav_get_base_path(request_rec *r)

入力と出力

  • **入力**: request_rec *r （Apache のリクエストレコード）
  • **出力**: const char * （設定されたベースパス文字列、または設定が存在しない場合は NULL）

前提条件と不変条件

  • r->per_dir_config に dav_module の設定が格納されている必要があります。
  • 返される文字列は、dav_dir_conf 構造体の生存期間に依存します。

アルゴリズムまたは振る舞い

1. ap_get_module_config を使用して、リクエストのディレクトリ設定から dav_dir_conf 構造体を取得します。 2. 取得した設定構造体が有効であり、かつその中の base メンバが設定されている場合、その値を返します。 3. 設定構造体が存在しない、または base が未設定の場合は NULL を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: 提供されたソース情報からは、特定の呼び出し元関数は特定できません。
  • **呼び出し先**: ap_get_module_config （Apache API）を呼び出します。

関連データ構造

  • dav_dir_conf: ディレクトリごとの DAV 設定を保持する構造体。base メンバを保持します。
  • request_rec: Apache のリクエスト情報を保持する構造体。

失敗モードと境界条件

  • r->per_dir_config から dav_module の設定が取得できない場合、conf は NULL となり、関数は安全に NULL を返します。
  • conf は存在するが conf->base が初期化されていない場合、NULL を返します。
