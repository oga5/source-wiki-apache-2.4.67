mpm_term

定義

  • 種別: function
  • 定義箇所: server/mpm/netware/mpm_netware.c:263

目的

NetWare MPM の終了処理として、コンソールハンドラの解除と終了待機フラグの初期化を行う。

シグネチャまたは形状

static void mpm_term(void)

入力と出力

  • 入力: なし
  • 出力: なし（副作用としてグローバル状態を変更）

前提条件と不変条件

  • 本関数は、サーバーの終了プロセスにおいて呼び出されることを想定している。

アルゴリズムまたは振る舞い

1. RemoveConsoleHandler() を呼び出し、NetWare コンソールからのコマンド受付を停止する。 2. グローバル変数 wait_to_finish を 0 に設定する。 3. NXThreadYield() を呼び出し、現在のスレッドの実行権を解放する。

呼び出し元と呼び出し先

  • 呼び出し元: server/mpm/netware/mpm_netware.c:1113 (atexit への登録)
  • 呼び出し先: RemoveConsoleHandler, NXThreadYield

関連データ構造

  • wait_to_finish: サーバーの終了待機状態を制御するグローバル変数。

失敗モードと境界条件

  • 本関数自体にエラーハンドリングは存在しない。
  • RemoveConsoleHandler の内部状態に依存する。

保守担当者向けメモ

  • atexit を介して登録されているため、プログラム終了時に自動的に実行される。
  • NXThreadYield の呼び出しは、終了処理中に他のスレッドへ制御を譲るための措置である。
