ssl_pphrase_t

定義

  • 種別: enum
  • 定義箇所: modules/ssl/ssl_private.h:506

目的

SSL証明書の秘密鍵を復号するためのパスフレーズ取得方式を識別するために使用されます。

シグネチャまたは形状

typedef enum {
    SSL_PPTYPE_UNSET   = UNSET,
    SSL_PPTYPE_BUILTIN = 0,
    SSL_PPTYPE_FILTER  = 1,
    SSL_PPTYPE_PIPE    = 2
} ssl_pphrase_t;

入力と出力

  • 入力: 該当なし（列挙型定義）
  • 出力: SSL_PPTYPE_UNSET, SSL_PPTYPE_BUILTIN, SSL_PPTYPE_FILTER, SSL_PPTYPE_PIPE のいずれかの値。

前提条件と不変条件

  • UNSET マクロが定義されている環境下で利用される必要があります。
  • 各列挙値は、パスフレーズ取得の具体的な実装戦略（組み込み、フィルタ、パイプ経由など）に対応します。

アルゴリズムまたは振る舞い

  • SSL_PPTYPE_BUILTIN: 標準的な組み込みのパスフレーズ取得処理を示唆します。
  • SSL_PPTYPE_FILTER: フィルタを介したパスフレーズ取得処理を示唆します。
  • SSL_PPTYPE_PIPE: 外部パイプを介したパスフレーズ取得処理を示唆します。

呼び出し元と呼び出し先

  • 呼び出し元: modules/ssl/ssl_private.h:793 にて、設定構造体内のメンバ pphrase_dialog_type の型として参照されています。
  • 呼び出し先: 情報なし。

関連データ構造

  • ssl_engine_pphrase.c: 本列挙型を利用したパスフレーズ処理の実装が含まれることが示唆されます。

失敗モードと境界条件

  • SSL_PPTYPE_UNSET が設定されている場合、パスフレーズ取得方式が未構成であることを示します。

保守担当者向けメモ

  • パスフレーズ取得方式を追加・変更する場合は、本列挙型に新しい識別子を追加し、ssl_engine_pphrase.c 内のロジックを更新してください。
