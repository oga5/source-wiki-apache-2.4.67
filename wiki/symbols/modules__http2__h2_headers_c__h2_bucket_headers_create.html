h2_bucket_headers_create

定義

  • 種別: function
  • 定義箇所: modules/http2/h2_headers.c:73

目的

HTTP/2 レスポンスヘッダーを保持するカスタムバケット（H2HEADERS 型）を生成し、Apache のバケットチェーンに組み込める形式で初期化します。

シグネチャまたは形状

apr_bucket *h2_bucket_headers_create(apr_bucket_alloc_t *list, h2_headers *r);

入力と出力

  • **入力**:
  • list: バケット割り当て用の apr_bucket_alloc_t インスタンス。
  • r: バケットに格納する h2_headers 構造体へのポインタ。
  • **出力**:
  • 初期化済みの apr_bucket 構造体へのポインタを返します。

前提条件と不変条件

  • list は有効な apr_bucket_alloc_t である必要があります。
  • 生成されたバケットは h2_bucket_headers_make を内部で呼び出し、H2HEADERS 型として適切に構成されます。

アルゴリズムまたは振る舞い

1. apr_bucket_alloc を使用して、指定された list から apr_bucket 構造体分のメモリを確保します。 2. APR_BUCKET_INIT マクロでバケットの初期化を行います。 3. バケットの解放関数 b->free に apr_bucket_free を設定します。 4. h2_bucket_headers_make を呼び出し、渡された h2_headers データをバケットに紐付けます。 5. 構成されたバケットポインタを返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/http2/h2_c2_filter.c (複数箇所)
  • modules/http2/h2_headers.c:115
  • modules/http2/h2_stream.c:287
  • modules/http2/h2_ws.c:172, 191
  • **呼び出し先**:
  • apr_bucket_alloc (APR)
  • h2_bucket_headers_make (同モジュール内)

関連データ構造

  • apr_bucket: Apache のバケットチェーンを構成する基本単位。
  • h2_headers: HTTP/2 レスポンスヘッダー情報を保持する構造体。

失敗モードと境界条件

  • apr_bucket_alloc がメモリ不足により失敗した場合、呼び出し元で適切なエラーハンドリングが必要です（本関数自体は戻り値のチェックを行いません）。

保守担当者向けメモ

  • 本関数は H2HEADERS 型バケットのファクトリです。バケットのライフサイクルは apr_bucket の参照カウント管理に依存します。
  • AP_HAS_RESPONSE_BUCKETS が定義されている環境では、この実装は無効化される可能性があります。
