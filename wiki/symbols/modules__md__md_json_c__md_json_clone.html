md_json_clone

定義

  • 種別: function
  • 定義箇所: modules/md/md_json.c:116

目的

既存の md_json_t オブジェクトを、指定された APR プール上で完全に複製（ディープコピー）した新しい md_json_t インスタンスを作成します。

シグネチャまたは形状

md_json_t *md_json_clone(apr_pool_t *pool, const md_json_t *json);

入力と出力

  • **入力**:
  • pool: 新しいオブジェクトのメモリ管理に使用する apr_pool_t。
  • json: 複製元となる md_json_t オブジェクト。
  • **出力**:
  • 複製された新しい md_json_t へのポインタ。

前提条件と不変条件

  • json 引数は有効な md_json_t 構造体である必要があります。
  • 内部で json_deep_copy を呼び出すため、JSON ツリー全体が再帰的に複製されます。
  • 作成されたオブジェクトは、指定された pool の生存期間に紐付きます。

アルゴリズムまたは振る舞い

1. 渡された json オブジェクトから内部の json_t ポインタ（json->j）を取り出します。 2. json_deep_copy を使用して、JSON ツリーの完全な複製を作成します。 3. json_create を呼び出し、複製された JSON ツリーと pool を紐付けた新しい md_json_t を生成して返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/md/md_acme.c:509
  • modules/md/md_acme_acct.c:456
  • modules/md/md_json.c:717, 723
  • modules/md/md_jws.c:52, 122
  • modules/md/md_result.c:107, 223
  • **呼び出し先**:
  • json_deep_copy (Jansson ライブラリ)
  • json_create (modules/md/md_json.c)

関連データ構造

  • md_json_t: 本関数が生成・操作するラッパー構造体。
  • json_t: Jansson ライブラリの JSON ノード構造体。
  • apr_pool_t: メモリ管理用プール。

失敗モードと境界条件

  • メモリ不足により json_deep_copy または json_create が失敗した場合、NULL が返される可能性があります。

保守担当者向けメモ

  • 本関数はディープコピーを行うため、元の JSON ツリーに変更を加えても複製先には影響しません。
  • 呼び出し元で pool を適切に管理し、不要になった際にプールが破棄されることでメモリリークを防ぐ設計となっています。
