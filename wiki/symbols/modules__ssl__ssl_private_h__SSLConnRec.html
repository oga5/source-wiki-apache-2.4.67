SSLConnRec

定義

  • 種別: struct
  • 定義箇所: modules/ssl/ssl_private.h:606

目的

Apache HTTP Server の mod_ssl において、個々の接続（Connection）に関連付けられた SSL/TLS の状態を保持・管理する。

シグネチャまたは形状

typedef struct {
    SSL *ssl;
    const char *client_dn;
    X509 *client_cert;
    ssl_shutdown_type_e shutdown_type;
    const char *verify_info;
    const char *verify_error;
    int verify_depth;
    int disabled;
    enum { ... } non_ssl_request;
#ifndef SSL_OP_NO_RENEGOTIATION
    modssl_reneg_state reneg_state;
#endif
    server_rec *server;
    SSLDirConfigRec *dc;
    const char *cipher_suite;
    int service_unavailable;
    int vhost_found;
} SSLConnRec;

入力と出力

本構造体は関数ではなくデータ構造であるため、入力および出力の概念は適用されない。各フィールドは接続のライフサイクルを通じて、mod_ssl の各フック関数によって読み書きされる。

前提条件と不変条件

  • SSL *ssl は、OpenSSL の接続コンテキストを指す。
  • server_rec *server および SSLDirConfigRec *dc は、当該接続が属するサーバーおよびディレクトリの設定を保持する。
  • non_ssl_request 列挙型は、SSL 以外のリクエストやエラーハンドリングの状態を追跡する。

アルゴリズムまたは振る舞い

  • **接続状態の保持**: 接続ごとの証明書情報（client_cert）、検証結果（verify_info, verify_error）、再ネゴシエーション状態（reneg_state）を保持する。
  • **SNI 追跡**: vhost_found フラグにより、SNI（Server Name Indication）による仮想ホストの特定が完了しているかを管理する。
  • **シャットダウン管理**: shutdown_type を通じて、SSL 接続の終了処理の挙動を制御する。

呼び出し元と呼び出し先

  • **呼び出し元（参照元）**:
  • modules/ssl/mod_ssl.c: ssl_init_connection_ctx, ssl_engine_status 等
  • modules/ssl/ssl_engine_io.c: ssl_io_filter_disable 等
  • modules/arch/netware/mod_nw_ssl.c
  • **呼び出し先**:
  • 本構造体はデータ保持用であり、直接的な関数呼び出しは行わない。ただし、内部の SSL *ssl フィールドを通じて OpenSSL ライブラリの関数が呼び出される。

関連データ構造

  • SSLDirConfigRec: ディレクトリごとの SSL 設定。
  • server_rec: Apache のサーバー設定構造体。
  • modssl_reneg_state: 再ネゴシエーションの状態管理用。
  • ssl_shutdown_type_e: シャットダウンの種類を定義する列挙型。

失敗モードと境界条件

  • disabled フラグが設定されている場合、SSL 処理がスキップされる可能性がある。
  • service_unavailable が真の場合、SSL ネゴシエーションが成功してもリクエストの処理は行われない。
  • OpenSSL のバージョン（< 1.1.1 か否か）により、reneg_state の有無や再ネゴシエーションの制御方法が分岐する。

保守担当者向けメモ

  • 本構造体は myConnConfig(c) マクロを介して conn_rec から取得される。
  • OpenSSL 3.0 への移行に伴い、reneg_state 等の条件付きコンパイル部分の保守には注意が必要である。
  • 構造体のフィールド追加・変更時は、mod_ssl 内の全参照箇所（特に ssl_engine_io.c や mod_ssl.c）への影響を考慮すること。
