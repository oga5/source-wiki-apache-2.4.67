slotmem_slot_size

定義

  • 種別: function
  • 定義箇所: modules/slotmem/mod_slotmem_plain.c:234

目的

指定されたスロットメモリインスタンスが管理する、個々のスロットのサイズ（バイト単位）を返します。

シグネチャまたは形状

apr_size_t slotmem_slot_size(ap_slotmem_instance_t *slot)

入力と出力

  • **入力**: slot - 情報を取得する ap_slotmem_instance_t 構造体へのポインタ。
  • **出力**: apr_size_t 型のサイズ値。

前提条件と不変条件

  • slot 引数は有効な ap_slotmem_instance_t 構造体を指している必要があります。
  • インスタンス内の size フィールドは、インスタンス作成時に確定され、以降変更されないものとして扱われます。

アルゴリズムまたは振る舞い

1. 引数として受け取った slot 構造体のメンバ size を参照します。 2. その値をそのまま呼び出し元へ返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/slotmem/mod_slotmem_plain.c:313 (プロバイダー構造体への登録)
  • modules/slotmem/mod_slotmem_shm.c:742 (プロバイダー構造体への登録)
  • **呼び出し先**:
  • 特になし（構造体メンバへの直接アクセスのみ）。

関連データ構造

  • ap_slotmem_instance_t: スロットメモリの管理単位であり、本関数が参照する size フィールドを保持します。

失敗モードと境界条件

  • slot が NULL の場合、セグメンテーションフォールトが発生する可能性があります。本関数内での明示的な NULL チェックは行われていません。

保守担当者向けメモ

  • 本関数は ap_slotmem_provider_t インターフェースの一部として公開されています。
  • mod_slotmem_plain.c および mod_slotmem_shm.c の両方で同じシグネチャと実装がプロバイダーとして登録されています。
