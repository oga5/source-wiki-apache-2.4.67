ptoken_char

定義

  • 種別: function
  • 定義箇所: modules/http2/h2_push.c:95

目的

HTTP/2 の Link ヘッダー等の解析において、トークンの一部として許容される文字であるかを判定します。

シグネチャまたは形状

int ptoken_char(char c)

入力と出力

  • **入力**: char c（判定対象の文字）
  • **出力**: int 型。対象文字がトークンの一部として有効な場合は 1、それ以外の場合は 0 を返します。

前提条件と不変条件

  • apr_isalnum を内部で使用しているため、実行環境のロケール設定に依存する可能性があります。

アルゴリズムまたは振る舞い

1. 引数 c が以下の記号セットに含まれるかを確認します。 !, #, $, &, ', (, ), *, +, -, ., /, :, <, =, >, ?, @, [, ], ^, _, ` `, {, |, }, ~ 2. 上記の記号であれば 1 を返します。 3. 記号以外の場合、apr_isalnum(c) を呼び出し、英数字であれば 1 を、それ以外であれば 0` を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/http2/h2_proxy_util.c:857
  • modules/http2/h2_push.c:187
  • modules/md/md_util.c:1366
  • **呼び出し先**:
  • apr_isalnum (APRライブラリ関数)

関連データ構造

  • 特になし。

失敗モードと境界条件

  • 入力が char 型の範囲外である場合の挙動は、apr_isalnum の実装に依存します。

保守担当者向けメモ

  • 本関数はトークンの境界を特定するための走査処理（for ループの条件式）で頻繁に使用されています。
  • 記号の追加や除外を行う際は、HTTP/2 の仕様（RFC 7540）および関連する Link ヘッダーの構文定義との整合性に注意してください。
