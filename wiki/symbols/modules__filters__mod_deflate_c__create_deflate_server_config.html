create_deflate_server_config

定義

  • 種別: function
  • 定義箇所: modules/filters/mod_deflate.c:209

目的

mod_deflate モジュールのサーバー単位の設定構造体（deflate_filter_config）を初期化し、デフォルト値を割り当てる。

シグネチャまたは形状

static void *create_deflate_server_config(apr_pool_t *p, server_rec *s)

入力と出力

  • **入力**:
  • apr_pool_t *p: 設定の割り当てに使用するメモリプール。
  • server_rec *s: 設定対象のサーバーレコード（本関数内では未使用）。
  • **出力**:
  • void *: 初期化された deflate_filter_config 構造体へのポインタ。

前提条件と不変条件

  • apr_pcalloc により、構造体内の全メンバは初期状態でゼロクリアされる。
  • 戻り値の構造体は、Apache の設定フェーズにおいてサーバー単位の設定として保持される。

アルゴリズムまたは振る舞い

1. apr_pcalloc を使用して、メモリプール p 上に deflate_filter_config 構造体分のメモリを確保する。 2. 以下のデフォルト値を構造体の各メンバに代入する。

  • memlevel: DEFAULT_MEMLEVEL
  • windowSize: DEFAULT_WINDOWSIZE
  • bufferSize: DEFAULT_BUFFERSIZE
  • compressionlevel: DEFAULT_COMPRESSION

    • 3. 初期化済みの構造体ポインタを返す。

呼び出し元と呼び出し先

  • **呼び出し元**: modules/filters/mod_deflate.c:1949（サーバー設定の登録時に呼び出される）。
  • **呼び出し先**: apr_pcalloc（APRライブラリ関数）。

関連データ構造

  • deflate_filter_config: 本関数が初期化するサーバー設定構造体。

失敗モードと境界条件

  • メモリ不足時には apr_pcalloc が例外を発生させる可能性がある（Apache の標準的なメモリ管理動作に依存）。

保守担当者向けメモ

  • 本関数で設定されるデフォルト値（DEFAULT_* マクロ）を変更する場合は、mod_deflate.c 内の定義を確認すること。
  • server_rec *s 引数は現在使用されていないが、将来的にサーバーごとの動的な設定調整が必要になった場合に備えてシグネチャに含まれている。
