dav_get_options_providers

定義

  • 種別: function
  • 定義箇所: modules/dav/main/providers.c:42

目的

指定された名前の DAV オプションプロバイダーを検索し、対応する構造体を取得します。

シグネチャまたは形状

const dav_options_provider *dav_get_options_providers(const char *name)

入力と出力

  • **入力**: name (検索対象のプロバイダー名を示す文字列)
  • **出力**: 見つかった場合は dav_options_provider 構造体へのポインタ、見つからない場合は NULL を返します。

前提条件と不変条件

  • プロバイダーの検索には DAV_OPTIONS_EXTENSION_GROUP グループ識別子が使用されます。
  • プロバイダーのバージョンは "0" に固定されています。

アルゴリズムまたは振る舞い

Apache の汎用プロバイダー検索 API である ap_lookup_provider を呼び出し、DAV オプション拡張グループ内から指定された名前のプロバイダーを検索して返します。

呼び出し元と呼び出し先

  • **呼び出し元**: modules/dav/main/mod_dav.c (1766行目および1926行目) から呼び出されています。
  • **呼び出し先**: Apache Core の ap_lookup_provider を呼び出します。

関連データ構造

  • dav_options_provider: DAV オプション拡張を定義する構造体。
  • DAV_OPTIONS_EXTENSION_GROUP: プロバイダー検索時に使用されるグループ識別子。

失敗モードと境界条件

  • 指定された name に該当するプロバイダーが登録されていない場合、ap_lookup_provider の仕様に従い NULL が返されます。

保守担当者向けメモ

  • 本関数は ap_provider API の薄いラッパーです。
  • プロバイダーのバージョン管理は "0" に固定されているため、将来的にバージョン管理が必要となった場合はこの固定値を変更する必要があります。
