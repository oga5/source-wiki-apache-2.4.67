dav_fs_compare_locktoken

定義

  • 種別: function
  • 定義箇所: modules/dav/fs/lock.c:288

目的

2つのロックトークン（dav_locktoken）が同一であるかを比較します。

シグネチャまたは形状

static int dav_fs_compare_locktoken(
    const dav_locktoken *lt1,
    const dav_locktoken *lt2)

入力と出力

  • **入力**: 比較対象となる2つの dav_locktoken 構造体へのポインタ (lt1, lt2)。
  • **出力**: 比較結果を示す整数値を返します。

前提条件と不変条件

  • lt1 および lt2 は有効な dav_locktoken 構造体を指している必要があります。

アルゴリズムまたは振る舞い

dav_compare_locktoken 関数を呼び出し、その結果をそのまま返します。

呼び出し元と呼び出し先

  • **呼び出し元**: modules/dav/fs/lock.c 内の 284 行目および 1431 行目から参照されています。
  • **呼び出し先**: dav_compare_locktoken を呼び出します。

関連データ構造

  • dav_locktoken: ロックを識別するための UUID をラップした構造体。

失敗モードと境界条件

  • 入力ポインタが NULL の場合の挙動については、本ソースコード内では明示されていません。

保守担当者向けメモ

  • 本関数は dav_compare_locktoken への単なるラッパーです。将来的にファイルシステム固有の比較ロジックが必要になった場合に備えて定義されています。
