dav_fs_create_server_config

定義

  • 種別: function
  • 定義箇所: modules/dav/fs/mod_dav_fs.c:40

目的

Apache HTTP Server の WebDAV ファイルシステムモジュールにおいて、サーバー単位の設定構造体 (dav_fs_server_conf) をメモリ上に割り当て、初期化するために使用されます。

シグネチャまたは形状

static void *dav_fs_create_server_config(apr_pool_t *p, server_rec *s)

入力と出力

  • **入力**:
  • p: 設定のライフサイクルを管理するための APR プール。
  • s: 設定が適用される対象のサーバーレコード。
  • **出力**:
  • ゼロクリアされた dav_fs_server_conf 構造体へのポインタを返します。

前提条件と不変条件

  • apr_pcalloc を使用しているため、返される構造体の全メンバは初期状態でゼロ（または NULL）であることが保証されます。

アルゴリズムまたは振る舞い

1. 指定されたメモリプール p から、dav_fs_server_conf 構造体のサイズ分のメモリを確保します。 2. 確保したメモリ領域をゼロクリアして返します。

呼び出し元と呼び出し先

  • **呼び出し元**: modules/dav/fs/mod_dav_fs.c:104 において、サーバー設定作成用フックとして登録されています。
  • **呼び出し先**: APR のメモリ割り当て関数 apr_pcalloc を呼び出します。

関連データ構造

  • dav_fs_server_conf: 本関数が割り当てるサーバー設定構造体。ロックデータベースのパス (lockdb_path) を保持します。

失敗モードと境界条件

  • メモリ不足時には apr_pcalloc の挙動に従い、プロセスが異常終了する可能性があります。

保守担当者向けメモ

  • 本関数は mod_dav_fs の設定ライフサイクルの起点です。設定値の変更や構造体の拡張を行う場合は、dav_fs_server_conf の定義と併せて確認してください。
