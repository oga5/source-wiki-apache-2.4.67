xml2enc_hooks

定義

  • 種別: function
  • 定義箇所: modules/filters/mod_xml2enc.c:589

目的

mod_xml2enc モジュールのフィルタ登録および、コンテンツ解析に必要な正規表現の事前コンパイルを行う。

シグネチャまたは形状

static void xml2enc_hooks(apr_pool_t* pool)

入力と出力

  • **入力**: apr_pool_t* pool（メモリ割り当て用のプール）
  • **出力**: なし（副作用としてフィルタ登録とグローバル変数の初期化を行う）

前提条件と不変条件

  • Apache のフィルタ登録関数 ap_register_output_filter_protocol が利用可能であること。
  • pool はリクエストまたはモジュールのライフサイクルに紐付いた有効なメモリプールであること。

アルゴリズムまたは振る舞い

1. ap_register_output_filter_protocol を呼び出し、xml2enc フィルタを AP_FTYPE_RESOURCE 型で登録する。この際、メイン処理関数として xml2enc_ffunc を、初期化関数として xml2enc_filter_init を指定する。 2. APR_REGISTER_OPTIONAL_FN を使用し、xml2enc_filter および xml2enc_charset をオプション関数として登録する。 3. ap_pregcomp を使用し、HTML の <meta> タグおよび charset 属性を抽出するための正規表現をコンパイルし、それぞれ seek_meta_ctype および seek_charset に格納する。

呼び出し元と呼び出し先

  • **呼び出し元**: modules/filters/mod_xml2enc.c:675（モジュール登録プロセスの一部として呼び出される）
  • **呼び出し先**:
  • ap_register_output_filter_protocol
  • APR_REGISTER_OPTIONAL_FN
  • ap_pregcomp

関連データ構造

  • apr_pool_t: メモリ管理に使用。
  • seek_meta_ctype, seek_charset: コンパイルされた正規表現を保持するグローバル変数。

失敗モードと境界条件

  • ap_pregcomp による正規表現のコンパイルが失敗した場合、Apache のエラーログ出力およびプロセス終了の可能性がある（ap_pregcomp の内部実装に依存）。

保守担当者向けメモ

  • 本関数はモジュールの初期化時に一度だけ実行される。
  • 正規表現のパターンを変更する場合は、本関数内の ap_pregcomp 引数を更新すること。
  • PROTO_FLAGS は本ソースコード内で定義されている定数であり、フィルタのプロトコル特性を決定する。
