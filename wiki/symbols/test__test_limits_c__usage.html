usage

定義

  • 種別: function
  • 定義箇所: test/test_limits.c:71

目的

test_limits ユーティリティのコマンドライン引数の使用方法を標準エラー出力に表示し、プログラムを異常終了させるために使用されます。

シグネチャまたは形状

void usage(void)

入力と出力

  • 入力: なし
  • 出力: 標準エラー出力 (stderr) に使用方法の文字列を出力し、プロセスを終了コード 1 で終了させます。

前提条件と不変条件

  • 本関数が呼び出されると、プログラムは必ず終了します。

アルゴリズムまたは振る舞い

1. fprintf を使用して、サポートされているコマンドラインオプション（-t, -a, -p, -n）を含む使用方法のメッセージを stderr に書き出します。 2. exit(1) を呼び出し、呼び出し元へ制御を戻さずにプロセスを終了します。

呼び出し元と呼び出し先

  • 呼び出し元: 提供されたソースコード内では明示されていません。
  • 呼び出し先: fprintf (標準ライブラリ), exit (標準ライブラリ)
  • 注記: ツールによる解析では、他のファイル（include/ 配下など）に存在する同名のシンボルや文字列との混同が見られますが、本関数 test/test_limits.c:71 を直接呼び出している具体的な箇所は提供された情報からは特定できません。

関連データ構造

  • なし

失敗モードと境界条件

  • 本関数自体はエラーを発生させず、エラー発生時の通知および終了処理を担います。

保守担当者向けメモ

  • 本関数は test_limits ツール専用のヘルパーです。コマンドライン引数の仕様を変更した場合は、必ずこのメッセージを更新してください。
