ap_expr_lookup_default

定義

  • 種別: function
  • 定義箇所: server/util_expr_eval.c:374

目的

式評価エンジンにおいて、変数や関数の検索を行うためのデフォルトのフック関数を提供します。

シグネチャまたは形状

int ap_expr_lookup_default(ap_expr_lookup_parms *parms)

入力と出力

  • **入力**: ap_expr_lookup_parms *parms（検索対象の識別子やコンテキスト情報を含む構造体）
  • **出力**: int（検索結果のステータスコードを返します）

前提条件と不変条件

  • ap_run_expr_lookup が正常に機能する環境である必要があります。

アルゴリズムまたは振る舞い

内部で ap_run_expr_lookup(parms) を呼び出し、登録されている式検索プロバイダーのフックチェーンを実行します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • server/util_expr_eval.c:398（デフォルトの検索関数として割り当て）
  • modules/aaa/mod_authz_core.c:1072
  • **呼び出し先**:
  • ap_run_expr_lookup（フック関数）

関連データ構造

  • ap_expr_lookup_parms: 検索パラメータを保持する構造体。

失敗モードと境界条件

  • ap_run_expr_lookup の戻り値に依存します。検索対象が見つからない場合や、フック実行中にエラーが発生した場合の挙動は、登録されている各プロバイダーの実装に依存します。

保守担当者向けメモ

  • 本関数は ap_run_expr_lookup の単なるラッパーです。検索ロジックのカスタマイズが必要な場合は、本関数を直接修正するのではなく、ap_hook_expr_lookup を使用して新しいプロバイダーを登録してください。
