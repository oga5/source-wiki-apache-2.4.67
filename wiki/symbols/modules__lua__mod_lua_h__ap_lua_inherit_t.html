ap_lua_inherit_t

定義

  • 種別: enum
  • 定義箇所: modules/lua/mod_lua.h:96

目的

Apache HTTP Server の Lua モジュールにおいて、設定の継承ルールを定義するための列挙型です。

シグネチャまたは形状

typedef enum {
    AP_LUA_INHERIT_UNSET        = -1,
    AP_LUA_INHERIT_NONE         =  0,
    AP_LUA_INHERIT_PARENT_FIRST =  1,
    AP_LUA_INHERIT_PARENT_LAST  =  2
} ap_lua_inherit_t;

入力と出力

  • 入力: なし（列挙値の定義）
  • 出力: なし

前提条件と不変条件

  • 各列挙値は、設定の継承順序や有効性を表す整数値として扱われます。
  • AP_LUA_INHERIT_UNSET は、設定が明示的に指定されていない状態を示します。

アルゴリズムまたは振る舞い

  • AP_LUA_INHERIT_NONE: 親設定の継承を行いません。
  • AP_LUA_INHERIT_PARENT_FIRST: 親の設定を先に適用し、その後に現在の設定を適用します。
  • AP_LUA_INHERIT_PARENT_LAST: 現在の設定を先に適用し、その後に親の設定を適用します。

呼び出し元と呼び出し先

  • 呼び出し元: modules/lua/mod_lua.h:141 にて、ap_lua_dir_cfg 構造体のメンバ inherit の型として参照されています。
  • 呼び出し先: 情報なし。

関連データ構造

  • ap_lua_dir_cfg: ディレクトリ単位の設定を保持する構造体であり、本型をメンバとして保持します。

失敗モードと境界条件

  • 該当なし。

保守担当者向けメモ

  • 本型は、設定の継承ロジックを制御する重要なフラグです。新しい継承戦略を追加する場合は、この列挙型に値を追加し、設定読み込み処理側で対応するロジックを実装する必要があります。
