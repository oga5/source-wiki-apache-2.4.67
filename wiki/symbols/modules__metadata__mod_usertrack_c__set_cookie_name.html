set_cookie_name

定義

  • 種別: function
  • 定義箇所: modules/metadata/mod_usertrack.c:366

目的

Apache の設定ディレクティブ CookieName 等を通じて指定された Cookie 名を cookie_dir_rec 構造体に保存し、その名前を用いてリクエストから Cookie を検索するための正規表現を生成・検証します。

シグネチャまたは形状

static const char *set_cookie_name(cmd_parms *cmd, void *mconfig, const char *name)

入力と出力

  • **入力**:
  • cmd: 設定コマンドのパラメータ構造体。
  • mconfig: ディレクトリごとの設定構造体 (cookie_dir_rec)。
  • name: 設定される Cookie 名の文字列。
  • **出力**:
  • 成功時: NULL。
  • 失敗時: エラーメッセージを含む文字列（const char *）。

前提条件と不変条件

  • mconfig は cookie_dir_rec 型としてキャスト可能であること。
  • set_and_comp_regexp により生成される正規表現は、NUM_SUBS（3）と一致するキャプチャグループ数を持つ必要があります。

アルゴリズムまたは振る舞い

1. mconfig を cookie_dir_rec としてキャストし、cookie_name フィールドに name を代入します。 2. set_and_comp_regexp を呼び出し、指定された name に基づく正規表現をコンパイルして dcfg->regexp に格納します。 3. 正規表現のコンパイルに失敗した場合はエラーを返します。 4. コンパイルされた正規表現のキャプチャグループ数（re_nsub + 1）が NUM_SUBS と一致するか確認します。一致しない場合は、無効な Cookie 名としてエラーを返します。 5. 正常終了時は NULL を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/metadata/mod_usertrack.c:477 (AP_INIT_TAKE1 "CookieName")
  • modules/session/mod_session_cookie.c:258 (AP_INIT_RAW_ARGS "SessionCookieName")
  • modules/session/mod_session_dbd.c:626 (AP_INIT_RAW_ARGS "SessionDBDCookieName")
  • **呼び出し先**:
  • set_and_comp_regexp (正規表現の構築)
  • apr_pstrcat (エラーメッセージ生成時)

関連データ構造

  • cookie_dir_rec: ディレクトリごとの設定を保持する構造体。cookie_name および regexp を含みます。
  • cmd_parms: Apache 設定コマンドのコンテキスト。

失敗モードと境界条件

  • 正規表現のコンパイル失敗: dcfg->regexp が NULL の場合に発生。
  • キャプチャグループ数の不整合: re_nsub + 1 != NUM_SUBS の場合に発生し、設定エラーとして扱われます。

保守担当者向けメモ

  • 本関数は mod_usertrack だけでなく、mod_session_cookie や mod_session_dbd からも参照されています。変更時はこれらのモジュールへの影響を考慮してください。
  • NUM_SUBS は正規表現の構造に直結する定数であるため、変更時は set_and_comp_regexp との整合性を維持してください。
