h2_proxy_ihash_empty

定義

  • 種別: function
  • 定義箇所: modules/http2/h2_proxy_util.c:92

目的

指定された h2_proxy_ihash_t ハッシュテーブルが空であるかどうかを判定する。

シグネチャまたは形状

int h2_proxy_ihash_empty(h2_proxy_ihash_t *ih)

入力と出力

  • **入力**: ih (ハッシュテーブル h2_proxy_ihash_t へのポインタ)
  • **出力**: ハッシュテーブルが空であれば非ゼロ（真）、そうでなければ 0（偽）を返す。

前提条件と不変条件

  • ih は有効な h2_proxy_ihash_t 構造体を指している必要がある。
  • 内部の apr_hash_t 構造体が初期化されていること。

アルゴリズムまたは振る舞い

内部的に apr_hash_count を呼び出し、ハッシュテーブル内の要素数が 0 であるかを比較してその結果を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: modules/http2/h2_proxy_session.c において、セッション内のストリーム管理状態を確認するために複数回呼び出されている。
  • **呼び出し先**: apr_hash_count (APR ライブラリ関数)

関連データ構造

  • h2_proxy_ihash_t: 整数 ID をキーとするハッシュテーブル構造体。

失敗モードと境界条件

  • ih が NULL の場合、プログラムはクラッシュする可能性がある（ソースコード上に明示的なガードは存在しない）。

保守担当者向けメモ

  • 本関数は h2_proxy_ihash_t の状態確認を行う軽量なラッパーである。
  • h2_proxy_ihash_t の内部構造（apr_hash_t）に変更があった場合、本関数の実装も影響を受ける可能性がある。
