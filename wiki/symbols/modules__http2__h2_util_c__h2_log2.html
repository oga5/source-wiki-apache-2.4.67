h2_log2

定義

  • 種別: function
  • 定義箇所: modules/http2/h2_util.c:35

目的

与えられた符号なし整数 n に対し、その対数（底は2）を計算します。主に n が 2 のべき乗である場合に、その指数を求めるために使用されます。

シグネチャまたは形状

unsigned char h2_log2(unsigned int n)

入力と出力

  • **入力**: unsigned int n （対象となる整数）
  • **出力**: unsigned char （計算された対数値）

前提条件と不変条件

  • 入力が 0 の場合、関数は 0 を返します。
  • 呼び出し側のコメントによれば、本関数は n が 2 のべき乗である場合に正しく機能することが想定されています。

アルゴリズムまたは振る舞い

1. n が 0 の場合は直ちに 0 を返します。 2. ビットシフトとマスク演算を用いて、最上位ビット（MSB）の位置を特定します。 3. 16ビット、8ビット、4ビット、2ビット、1ビットの順で段階的にシフトし、先行ゼロ（leading zeros）の数 lz をカウントします。 4. 最終的に 31 - lz を計算して返します。これは、32ビット整数における最上位ビットのインデックスを求める処理に相当します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/http2/h2_push.c:818: log2n = h2_log2(N);
  • modules/http2/h2_push.c:824: log2pmax = h2_log2(ceil_power_of_2(maxP));
  • **呼び出し先**:
  • 本関数は標準ライブラリや他のモジュール関数を呼び出さず、算術演算のみで完結します。

関連データ構造

  • 特になし。

失敗モードと境界条件

  • n = 0: 0 を返します。
  • n = 1: 31 - 31 = 0 を返します（$2^0 = 1$）。
  • n = 2^31: 31 - 0 = 31 を返します。

保守担当者向けメモ

  • 本関数は h2_util.h で宣言されており、HTTP/2 モジュール全体で利用可能です。
  • 内部ロジックは 32 ビット整数を前提としています。
  • 呼び出し元のコメントにある通り、n が 2 のべき乗でない場合の戻り値は定義されていません。利用時には入力値の妥当性を確認してください。
