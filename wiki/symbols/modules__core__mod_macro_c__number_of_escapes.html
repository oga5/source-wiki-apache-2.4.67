number_of_escapes

定義

  • 種別: function
  • 定義箇所: modules/core/mod_macro.c:313

目的

指定された文字列内において、エスケープ文字（ESCAPE）および特定の区切り文字（delim）が出現する回数をカウントします。

シグネチャまたは形状

static int number_of_escapes(const char delim, const char *str)

入力と出力

  • **入力**:
  • delim: カウント対象とする区切り文字。
  • str: 走査対象のヌル終端文字列。
  • **出力**:
  • int: 文字列内に含まれる ESCAPE 文字と delim 文字の合計出現回数。

前提条件と不変条件

  • str は有効なヌル終端文字列である必要があります。
  • ESCAPE マクロが定義されていることが前提です。

アルゴリズムまたは振る舞い

1. カウンタ nesc を 0 に初期化します。 2. 文字列 str の先頭から終端まで 1 文字ずつ走査します。 3. 現在の文字が ESCAPE または delim と一致する場合、nesc をインクリメントします。 4. 走査終了後、デバッグ用として標準エラー出力にカウント結果と対象文字列を出力します。 5. 最終的な nesc の値を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: modules/core/mod_macro.c:340 において、置換処理に関連する計算の一部として呼び出されています。
  • **呼び出し先**: 標準ライブラリの fprintf（デバッグ出力時）。

関連データ構造

  • 特になし。

失敗モードと境界条件

  • str が空文字列の場合、ループは実行されず 0 を返します。
  • str が NULL の場合、未定義の動作（セグメンテーション違反）を引き起こす可能性があります。

保守担当者向けメモ

  • 本関数はデバッグ出力を伴うため、高頻度で呼び出される場合はパフォーマンスへの影響を考慮してください。
  • ESCAPE の定義値については、mod_macro.c 内の他の箇所と整合性が取れていることを確認してください。
