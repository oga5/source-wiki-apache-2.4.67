h2_srv_config_geti64

定義

  • 種別: function
  • 定義箇所: modules/http2/h2_config.c:277

目的

HTTP/2 サーバー設定構造体から、指定された列挙型変数に対応する 64 ビット整数型の設定値を取得する。

シグネチャまたは形状

apr_int64_t h2_srv_config_geti64(const h2_config *conf, h2_config_var_t var)

入力と出力

  • **入力**:
  • conf: 読み取り対象の h2_config 構造体へのポインタ。
  • var: 取得対象の設定項目を示す h2_config_var_t 型の列挙値。
  • **出力**:
  • 指定された設定項目の値（apr_int64_t）。該当する設定がない、または未設定の場合は DEF_VAL (-1) を返す。

前提条件と不変条件

  • conf は有効な h2_config 構造体を指している必要がある。
  • 設定値の解決には静的インスタンス defconf が使用される。

アルゴリズムまたは振る舞い

1. var に基づき switch 文で分岐する。 2. 各ケースにおいて H2_CONFIG_GET マクロを実行し、conf 内の値が未設定（DEF_VAL）であれば defconf の値を、そうでなければ conf の値を返す。 3. switch 文で定義されていない var が渡された場合は、デフォルト値 DEF_VAL を返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/http2/h2_config.c:516
  • modules/http2/h2_config.c:521
  • modules/http2/h2_config.c:526
  • modules/http2/h2_config.c:531
  • **呼び出し先**:
  • H2_CONFIG_GET マクロ（定義元は h2_config.h と推測されるが、本ソース excerpt には詳細なし）。

関連データ構造

  • h2_config: サーバーレベルの設定を保持する構造体。
  • h2_config_var_t: 設定項目を識別する列挙型。
  • defconf: 未設定時に参照されるデフォルト値の静的インスタンス。

失敗モードと境界条件

  • 未定義の var が渡された場合、エラーを発生させず DEF_VAL (-1) を返す。
  • conf が NULL の場合の挙動については、本ソース excerpt 内に明示的なガード句は存在しない。

保守担当者向けメモ

  • 本関数は h2_config_var_t に新しい設定項目を追加するたびに、switch 文へのケース追加が必要となる。
  • H2_CONFIG_GET マクロの挙動に依存しているため、マージロジックを変更する際は当該マクロの定義を確認すること。
