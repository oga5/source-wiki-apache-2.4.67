dav_get_provider_name

定義

  • 種別: function
  • 定義箇所: modules/dav/main/mod_dav.c:209

目的

現在処理中のリクエストに関連付けられた WebDAV プロバイダーの名前を取得します。

シグネチャまたは形状

const char *dav_get_provider_name(request_rec *r)

入力と出力

  • **入力**: request_rec *r (Apache のリクエストレコード)
  • **出力**: const char * (プロバイダー名を示す文字列、または設定が存在しない場合は NULL)

前提条件と不変条件

  • r は有効なリクエストレコードである必要があります。
  • dav_module がサーバーにロードされ、ディレクトリ設定が適切に初期化されていることを前提とします。

アルゴリズムまたは振る舞い

1. ap_get_module_config を使用して、リクエストのディレクトリ設定 (r->per_dir_config) から dav_dir_conf 構造体を取得します。 2. 取得した設定構造体が NULL でない場合、その構造体内の provider_name メンバを返します。 3. 設定構造体が NULL の場合は NULL を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: ソースコード内の直接的な呼び出し元は提供されていません。
  • **呼び出し先**: ap_get_module_config (Apache API) を呼び出します。

関連データ構造

  • dav_dir_conf: ディレクトリごとの設定を保持する構造体。provider_name を格納します。
  • request_rec: Apache のリクエスト情報を保持する構造体。

失敗モードと境界条件

  • r->per_dir_config に dav_module の設定が含まれていない場合、ap_get_module_config は NULL を返し、本関数も NULL を返します。

保守担当者向けメモ

  • 本関数は mod_dav.h で宣言されており、外部モジュールから利用可能です。
  • dav_dir_conf 構造体の定義は mod_dav.c 内にあります。
