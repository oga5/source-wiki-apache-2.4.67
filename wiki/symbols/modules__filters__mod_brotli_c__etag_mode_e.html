etag_mode_e

定義

  • 種別: enum
  • 定義箇所: modules/filters/mod_brotli.c:26

目的

Brotli 圧縮が適用されたレスポンスにおける ETag ヘッダーの取り扱い方針を定義する。

シグネチャまたは形状

typedef enum {
    ETAG_MODE_ADDSUFFIX = 0,
    ETAG_MODE_NOCHANGE = 1,
    ETAG_MODE_REMOVE = 2
} etag_mode_e;

入力と出力

  • 入力: なし（列挙型定義）
  • 出力: ETAG_MODE_ADDSUFFIX (0), ETAG_MODE_NOCHANGE (1), ETAG_MODE_REMOVE (2)

前提条件と不変条件

  • brotli_server_config_t 構造体内の etag_mode メンバとして使用される。
  • 各値は、圧縮によるコンテンツの変化をクライアントにどう通知するかというポリシーに対応する。

アルゴリズムまたは振る舞い

  • ETAG_MODE_ADDSUFFIX: 既存の ETag に -br サフィックスを付与する。
  • ETAG_MODE_NOCHANGE: ETag を変更しない。
  • ETAG_MODE_REMOVE: ETag を削除する。

呼び出し元と呼び出し先

  • 呼び出し元: modules/filters/mod_brotli.c 内の set_etag_mode 関数が、設定値に応じてこの列挙型を brotli_server_config_t に格納する。
  • 呼び出し先: フィルタ処理の過程で、ETag の調整ロジックがこの値を参照する。

関連データ構造

  • brotli_server_config_t: この列挙型を保持するサーバー設定構造体。

失敗モードと境界条件

  • 定義外の整数値が設定された場合の挙動については、ソースコード内に明示的なバリデーションロジックの記述なし。

保守担当者向けメモ

  • mod_deflate.c の ETag 処理と設計が類似しているため、挙動の変更が必要な場合は同モジュールの実装を参照すること。
