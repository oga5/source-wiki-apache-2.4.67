modssl_X509_getBC

定義

  • 種別: function
  • 定義箇所: modules/ssl/ssl_util_ssl.c:159

目的

X.509 証明書から Basic Constraints 拡張情報を抽出し、CA フラグおよびパス長制限（Path Length Constraint）を取得します。

シグネチャまたは形状

BOOL modssl_X509_getBC(X509 *cert, int *ca, int *pathlen)

入力と出力

  • **入力**:
  • cert: 情報を抽出する対象の X509 証明書構造体。
  • ca: CA フラグ（bc->ca）を格納するための整数ポインタ。
  • pathlen: パス長制限を格納するための整数ポインタ。
  • **出力**:
  • 成功時は TRUE を返し、ca および pathlen に値を設定します。
  • 失敗時は FALSE を返します。

前提条件と不変条件

  • cert は有効な X509 構造体である必要があります。
  • ca および pathlen は呼び出し元で確保された有効なメモリ領域を指している必要があります。
  • pathlen は、拡張が存在しない場合や値が設定されていない場合、-1（無制限）として扱われます。

アルゴリズムまたは振る舞い

1. X509_get_ext_d2i を使用して NID_basic_constraints 拡張を取得します。取得できない場合は FALSE を返します。 2. ca フィールドを bc->ca からコピーします。 3. pathlen をデフォルト値 -1 で初期化します。 4. bc->pathlen が存在する場合:

  • ASN1_INTEGER_to_BN で BIGNUM に変換し、さらに BN_bn2dec で 10 進文字列に変換します。
  • atoi で整数値に変換して pathlen に格納します。
  • 変換に使用した BIGNUM および一時文字列を解放します。

    • 5. 最後に BASIC_CONSTRAINTS_free(bc) を呼び出し、拡張構造体を解放して TRUE を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/ssl/ssl_engine_init.c:1433
  • **呼び出し先**:
  • OpenSSL API: X509_get_ext_d2i, ASN1_INTEGER_to_BN, BN_bn2dec, BN_free, OPENSSL_free, BASIC_CONSTRAINTS_free, atoi

関連データ構造

  • X509: OpenSSL の証明書構造体。
  • BASIC_CONSTRAINTS: X.509 Basic Constraints 拡張を保持する OpenSSL 内部構造体。

失敗モードと境界条件

  • 証明書に Basic Constraints 拡張が含まれていない場合、FALSE を返します。
  • ASN1_INTEGER から BIGNUM への変換、または BIGNUM から文字列への変換に失敗した場合、メモリを適切に解放した上で FALSE を返します。

保守担当者向けメモ

  • 本関数は OpenSSL の BASIC_CONSTRAINTS 構造体を一時的に確保するため、エラー発生時のメモリリークを防ぐための解放処理が各分岐に含まれています。修正時は BASIC_CONSTRAINTS_free の呼び出し漏れに注意してください。
