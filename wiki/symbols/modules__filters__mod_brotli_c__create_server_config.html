create_server_config

定義

  • 種別: function
  • 定義箇所: modules/filters/mod_brotli.c:42

目的

Brotli 圧縮フィルタ用のサーバー単位設定構造体（brotli_server_config_t）を生成し、デフォルト値を設定する。

シグネチャまたは形状

static void *create_server_config(apr_pool_t *p, server_rec *s)

入力と出力

  • **入力**:
  • p: 設定構造体の割り当てに使用するメモリプール。
  • s: 設定が適用されるサーバーレコード（本関数内では未使用）。
  • **出力**:
  • brotli_server_config_t*: 初期化済みの設定構造体へのポインタ。

前提条件と不変条件

  • apr_pcalloc により、構造体はゼロクリアされた状態で割り当てられる。
  • 設定値は mod_deflate との互換性を考慮したデフォルト値で初期化される。

アルゴリズムまたは振る舞い

1. apr_pcalloc を使用して brotli_server_config_t サイズのメモリを確保。 2. 以下のデフォルト値を設定:

  • quality: 5
  • lgwin: 18
  • lgblock: 0 (自動選択)
  • etag_mode: ETAG_MODE_ADDSUFFIX

    • 3. 初期化された構造体を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: Apache HTTP Server の設定フェーズにおいて、モジュール構造体を通じて呼び出される（modules/filters/mod_brotli.c:604 にて登録）。
  • **呼び出し先**: apr_pcalloc (APR ライブラリ)。

関連データ構造

  • brotli_server_config_t: 本関数が初期化するサーバー設定構造体。

失敗モードと境界条件

  • メモリ不足時には apr_pcalloc が例外を発生させる可能性がある。

保守担当者向けメモ

  • 本関数で設定されるデフォルト値は、Brotli の圧縮効率とメモリ使用量のバランスを考慮して選定されている。変更時は mod_deflate との挙動の差異に留意すること。
