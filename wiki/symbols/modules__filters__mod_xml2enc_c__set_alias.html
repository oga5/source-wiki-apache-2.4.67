set_alias

定義

  • 種別: function
  • 定義箇所: modules/filters/mod_xml2enc.c:602

目的

Apache HTTP Server の設定において、特定の文字エンコーディング名に対して別名（エイリアス）を登録する。

シグネチャまたは形状

static const char* set_alias(cmd_parms* cmd, void* CFG, const char* charset, const char* alias)

入力と出力

  • **入力**:
  • cmd: 設定コマンドのパラメータ構造体。
  • CFG: モジュールの設定構造体（本関数内では未使用）。
  • charset: 既存のエンコーディング名。
  • alias: 新たに割り当てるエイリアス名。
  • **出力**:
  • 成功時は NULL を返す。
  • 失敗時はエラーメッセージ文字列を返す。

前提条件と不変条件

  • 本関数は GLOBAL_ONLY なコンテキスト（サーバー全体の設定）でのみ呼び出し可能である。

アルゴリズムまたは振る舞い

1. ap_check_cmd_context を呼び出し、現在の設定コンテキストが GLOBAL_ONLY であるか検証する。 2. コンテキストが不適切な場合はエラーメッセージを返す。 3. xmlAddEncodingAlias(charset, alias) を呼び出し、libxml2 のエンコーディングエイリアスリストに登録を試みる。 4. 登録が成功（戻り値が 0）すれば NULL を返し、失敗した場合はエラーメッセージを返す。

呼び出し元と呼び出し先

  • **呼び出し元**: modules/filters/mod_xml2enc.c:642 にて AP_INIT_ITERATE2 マクロを通じて xml2EncAlias ディレクティブのハンドラとして登録されている。
  • **呼び出し先**:
  • ap_check_cmd_context (Apache API)
  • xmlAddEncodingAlias (libxml2 ライブラリ)

関連データ構造

  • cmd_parms: 設定コマンドの実行時パラメータを保持する構造体。

失敗モードと境界条件

  • コンテキストがサーバー全体（GLOBAL_ONLY）でない場合、エラーが発生する。
  • xmlAddEncodingAlias が失敗した場合、「Error setting charset alias」というエラーメッセージが返される。

保守担当者向けメモ

  • 本関数は libxml2 のグローバルなエイリアス管理テーブルを操作するため、スレッドセーフティについては libxml2 の仕様に依存する。
  • CFG 引数は現在の実装では使用されていない。
