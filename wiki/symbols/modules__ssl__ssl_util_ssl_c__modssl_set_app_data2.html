modssl_set_app_data2

定義

  • 種別: function
  • 定義箇所: modules/ssl/ssl_util_ssl.c:65

目的

OpenSSL の SSL オブジェクトに対し、mod_ssl 固有のアプリケーションデータを関連付けるためのラッパー関数です。

シグネチャまたは形状

void modssl_set_app_data2(SSL *ssl, void *arg)

入力と出力

  • **入力**:
  • ssl: データを設定する対象の SSL オブジェクトへのポインタ。
  • arg: SSL オブジェクトに関連付ける任意のデータ（通常は request_rec 構造体など）。
  • **出力**: なし（void）。

前提条件と不変条件

  • app_data2_idx が事前に初期化されている必要があります。
  • SSL オブジェクトは有効なポインタである必要があります。

アルゴリズムまたは振る舞い

1. OpenSSL の SSL_set_ex_data 関数を呼び出します。 2. app_data2_idx をインデックスとして使用し、arg を char * にキャストして SSL オブジェクトの拡張データ領域に格納します。 3. 処理完了後、直ちに制御を戻します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/ssl/mod_ssl.c:639
  • modules/ssl/ssl_engine_kernel.c:248
  • modules/ssl/ssl_engine_kernel.c:1022
  • modules/ssl/ssl_engine_kernel.c:1031
  • **呼び出し先**:
  • SSL_set_ex_data (OpenSSL API)

関連データ構造

  • SSL (OpenSSL 構造体)
  • app_data2_idx (モジュール内で管理される拡張データ用インデックス)

失敗モードと境界条件

  • ssl が NULL の場合、OpenSSL の SSL_set_ex_data の挙動に依存します。
  • app_data2_idx が初期化されていない状態で呼び出された場合、OpenSSL 側のエラーハンドリングが適用されます。

保守担当者向けメモ

  • 本関数は modssl_get_app_data2 と対になる操作を提供します。
  • arg に NULL を渡すことで、既存のアプリケーションデータの関連付けを解除（クリア）する用途にも使用されています。
