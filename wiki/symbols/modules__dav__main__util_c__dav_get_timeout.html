dav_get_timeout

定義

  • 種別: function
  • 定義箇所: modules/dav/main/util.c:542

目的

HTTP リクエストの Timeout ヘッダーを解析し、絶対時刻（time_t）として返却します。

シグネチャまたは形状

time_t dav_get_timeout(request_rec *r)

入力と出力

  • **入力**: request_rec *r (Apache HTTP リクエスト構造体)
  • **出力**: time_t 型の期限時刻。解析不能またはヘッダー不在の場合は DAV_TIMEOUT_INFINITE を返します。

前提条件と不変条件

  • r->headers_in テーブルから Timeout ヘッダーを取得します。
  • r->pool を使用してメモリを確保するため、呼び出し元で個別の解放は不要です。

アルゴリズムまたは振る舞い

1. Timeout ヘッダーが存在しない場合、DAV_TIMEOUT_INFINITE を返します。 2. ヘッダー値を空白区切りでトークン化し、順次評価します。 3. Infinite トークンを検出した場合、DAV_TIMEOUT_INFINITE を返します。 4. Second- で始まるトークンを検出した場合、続く数値を秒数として取得し、現在時刻 (time(NULL)) に加算した値を返します。 5. 認識可能なトークンが見つからない場合は、最終的に DAV_TIMEOUT_INFINITE を返します。

関連データ構造

  • request_rec: HTTP リクエスト情報を保持する Apache の主要構造体。
  • DAV_TIMEOUT_INFINITE: タイムアウトが無期限であることを示す定数。

失敗モードと境界条件

  • Timeout ヘッダーが空または解析不能な形式の場合、デフォルトで DAV_TIMEOUT_INFINITE を返します。

保守担当者向けメモ

  • RFC 4918 に準拠した Timeout ヘッダーの解析ロジックです。
  • Second- 後の数値解析において、オーバーフロー対策が必要である旨のコメントがソース内に存在します。
