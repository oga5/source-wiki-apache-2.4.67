md_json_create

定義

  • 種別: function
  • 定義箇所: modules/md/md_json.c:92

目的

空のJSONオブジェクトを保持する md_json_t インスタンスを生成し、指定されたAPRプールに関連付けます。

シグネチャまたは形状

md_json_t *md_json_create(apr_pool_t *pool)

入力と出力

  • **入力**: pool (メモリ管理に使用される apr_pool_t へのポインタ)
  • **出力**: 新規作成された md_json_t 構造体へのポインタ。

前提条件と不変条件

  • pool は有効なAPRプールである必要があります。
  • 内部で json_object() を呼び出し、空のJSONオブジェクトを初期値として割り当てます。

アルゴリズムまたは振る舞い

内部関数 json_create を呼び出し、pool と json_object() の戻り値を渡してインスタンスを初期化します。これにより、APRプールの生存期間とJSONオブジェクトのライフサイクルが紐付けられます。

呼び出し元と呼び出し先

  • **呼び出し先**: json_create (内部関数)
  • **呼び出し元**:
  • modules/md/md_acme.c
  • modules/md/md_acme_acct.c
  • modules/md/md_acme_authz.c
  • modules/md/md_acme_drive.c
  • modules/md/md_acme_order.c
  • modules/md/md_core.c

関連データ構造

  • md_json_t: json_t をラップし、APRプールへのポインタを保持する構造体。
  • json_t: JanssonライブラリのJSON型。

失敗モードと境界条件

  • メモリ不足時には json_create 内でアボートまたはエラーが発生する可能性があります（Janssonのメモリ管理依存）。

保守担当者向けメモ

  • 本関数は md_json_t のライフサイクル管理の起点です。生成されたオブジェクトは、関連付けられたAPRプールが破棄される際に自動的にクリーンアップされます。
