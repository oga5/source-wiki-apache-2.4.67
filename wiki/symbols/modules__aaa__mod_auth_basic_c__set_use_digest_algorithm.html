set_use_digest_algorithm

定義

  • 種別: function
  • 定義箇所: modules/aaa/mod_auth_basic.c:183

目的

Apache HTTP Server の設定ディレクティブ AuthBasicUseDigestAlgorithm を処理し、Basic 認証において Digest アルゴリズムを使用するかどうか、およびそのアルゴリズムの種類を設定する。

シグネチャまたは形状

static const char *set_use_digest_algorithm(cmd_parms *cmd, void *config, const char *alg)

入力と出力

  • **入力**:
  • cmd: 設定コマンドのパラメータ構造体。
  • config: ディレクトリごとの設定構造体 (auth_basic_config_rec)。
  • alg: 設定値として渡される文字列（"Off" または "MD5"）。
  • **出力**:
  • 成功時: NULL を返す。
  • 失敗時: エラーメッセージを含む文字列（apr_pstrcat で生成）を返す。

前提条件と不変条件

  • config 引数は auth_basic_config_rec 型にキャスト可能であること。
  • alg には "Off" または "MD5" 以外の値が指定された場合、エラーとして処理される。

アルゴリズムまたは振る舞い

1. config を auth_basic_config_rec 型として取得する。 2. alg が "Off" でも "MD5" でもない場合、エラーメッセージを生成して返す。 3. 妥当な値であれば、conf->use_digest_algorithm に alg を格納する。 4. conf->use_digest_algorithm_set フラグを 1 に設定する。 5. NULL を返し、設定処理を正常終了する。

呼び出し元と呼び出し先

  • **呼び出し元**: modules/aaa/mod_auth_basic.c:211 にて AP_INIT_TAKE1 マクロを通じてディレクティブ AuthBasicUseDigestAlgorithm のハンドラとして登録されている。
  • **呼び出し先**: apr_pstrcat（エラー発生時のみ）。

関連データ構造

  • auth_basic_config_rec: ディレクトリごとの設定を保持する構造体。本関数はこの構造体の use_digest_algorithm および use_digest_algorithm_set メンバを更新する。

失敗モードと境界条件

  • alg に "Off" または "MD5" 以外の文字列が指定された場合、apr_pstrcat を用いてエラーメッセージを生成し、設定の読み込みを失敗させる。

保守担当者向けメモ

  • 本関数は AuthBasicUseDigestAlgorithm ディレクティブ専用のハンドラである。
  • メモリ割り当ては cmd->pool を使用しているため、明示的な解放は不要である。
