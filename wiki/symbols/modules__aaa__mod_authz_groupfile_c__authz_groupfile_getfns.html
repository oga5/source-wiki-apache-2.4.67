authz_groupfile_getfns

定義

  • 種別: function
  • 定義箇所: modules/aaa/mod_authz_groupfile.c:306

目的

mod_authz_owner モジュールが提供するオプション関数 authz_owner_get_file_group を取得し、ローカルの関数ポインタ変数に格納します。

シグネチャまたは形状

static void authz_groupfile_getfns(void)

入力と出力

  • 入力: なし
  • 出力: なし（副作用としてグローバルな関数ポインタ変数を更新）

前提条件と不変条件

  • APR_RETRIEVE_OPTIONAL_FN マクロを使用して、外部モジュールから関数ポインタを安全に取得します。
  • 本関数は、mod_authz_owner がロードされている場合にのみ有効な関数ポインタを取得します。

アルゴリズムまたは振る舞い

1. APR_RETRIEVE_OPTIONAL_FN(authz_owner_get_file_group) を呼び出します。 2. 取得した関数ポインタを、モジュール内で定義された authz_owner_get_file_group 変数に代入します。

呼び出し元と呼び出し先

  • 呼び出し元: modules/aaa/mod_authz_groupfile.c:321 において、ap_hook_optional_fn_retrieve に登録されています。
  • 呼び出し先: APR_RETRIEVE_OPTIONAL_FN（APRのオプション関数取得マクロ）。

関連データ構造

  • authz_owner_get_file_group: 取得した関数ポインタを保持する変数。

失敗モードと境界条件

  • mod_authz_owner がロードされていない場合、取得される関数ポインタは NULL となります。その後の利用時に NULL チェックが必要です。

保守担当者向けメモ

  • 本関数はフック ap_hook_optional_fn_retrieve を通じて実行されます。
  • file-group 認可機能を利用する際、この関数によるポインタ取得が成功していることが前提となります。
