table_bytes_ctx

定義

  • 種別: struct
  • 定義箇所: modules/http2/h2_util.c:1140

目的

HTTP/2 関連の処理において、テーブル（ヘッダー等）のバイトサイズを計算・集計する際のコンテキストを保持するために使用されます。

シグネチャまたは形状

typedef struct {
    apr_size_t bytes;
    apr_size_t pair_extra;
} table_bytes_ctx;

入力と出力

  • bytes: 集計されたバイト数の合計値を格納します。
  • pair_extra: ヘッダーペアごとの追加オーバーヘッド分などを保持するために使用されます。

前提条件と不変条件

  • 本構造体は、apr_table_do 等の反復処理において、コールバック関数へ渡されるコンテキストとして利用されることを想定しています。

アルゴリズムまたは振る舞い

  • modules/http2/h2_util.c 内の処理において、テーブル内の要素を走査する際にこの構造体のインスタンスが初期化され、各要素のサイズを累積加算するために使用されます。

呼び出し元と呼び出し先

  • 呼び出し元: modules/http2/h2_util.c 内のテーブル走査関数（apr_table_do 等のコールバックを伴う処理）。
  • 呼び出し先: 本構造体自体はデータ保持用であり、関数を呼び出すことはありません。

関連データ構造

  • apr_table_t: 本コンテキストがサイズ計算の対象とする Apache Portable Runtime のテーブル構造体。

失敗モードと境界条件

  • apr_size_t の範囲を超える加算が行われた場合のオーバーフローについては、本構造体自体には保護機能はありません。

保守担当者向けメモ

  • 本構造体は h2_util.c 内の局所的な集計処理に特化しています。
  • 構造体のメンバー追加や変更を行う際は、このコンテキストを利用している apr_table_do のコールバック関数の実装と整合性を取る必要があります。
