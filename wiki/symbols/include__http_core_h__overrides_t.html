overrides_t

定義

  • 種別: typedef
  • 定義箇所: include/http_core.h:483

目的

Apache HTTP Server における設定のオーバーライド許可状態を保持するための型定義です。

シグネチャまたは形状

typedef unsigned int overrides_t;

入力と出力

  • 入力: unsigned int 型のビットマスク値。
  • 出力: なし（型定義）。

前提条件と不変条件

  • unsigned int として定義されており、ビット演算によるフラグ管理を前提としています。

アルゴリズムまたは振る舞い

  • 設定ファイル内の AllowOverride ディレクティブ等に関連する設定状態を保持するために使用されます。
  • 構造体内でビットマスクとして機能し、特定のオーバーライド許可設定を表現します。

呼び出し元と呼び出し先

  • 呼び出し元: include/http_core.h:543 および server/request.c:609, 610 にて、この型を用いた変数 override や override_opts が宣言されています。
  • 呼び出し先: 情報なし。

関連データ構造

  • core_dir_config: 本型がメンバとして含まれる設定構造体。

失敗モードと境界条件

  • unsigned int のビット幅を超えるフラグ設定はできません。
  • include/ap_mmn.h:386 によると、本型のサイズ変更はモジュールバイナリ互換性（MMN）に影響を与える可能性があります。

保守担当者向けメモ

  • 本型は core_dir_config 構造体の一部として使用されます。変更を加える際は、ABI 互換性への影響を考慮してください。
