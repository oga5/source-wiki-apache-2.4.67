log_err

定義

  • 種別: function
  • 定義箇所: test/clients/h2ws.c:77

目的

エラーメッセージを標準出力またはログ出力先に記録するためのラッパー関数です。

シグネチャまたは形状

static void log_err(const char *where, const char *msg)

入力と出力

  • where: エラーが発生した箇所やコンテキストを示す文字列。
  • msg: 出力するエラーメッセージの内容。
  • 戻り値: なし（void）。

前提条件と不変条件

  • where および msg は有効なヌル終端文字列である必要があります。

アルゴリズムまたは振る舞い

1. 引数として受け取った where と msg を、固定のラベル "ERROR" と共に log_out 関数へ渡します。 2. log_out を呼び出すことで、ログ出力処理を委譲します。

呼び出し元と呼び出し先

  • 呼び出し元: 提供されたソースコード内には、本関数を呼び出している箇所は明示されていません。
  • 呼び出し先: log_out 関数。
  • 注意: ツールが検出した support/suexec.c 内の log_err は同名の別関数であり、本シンボルとは無関係です。

関連データ構造

  • なし。

失敗モードと境界条件

  • log_out 関数が内部で依存している出力ストリームがクローズされている場合、ログ出力が失敗する可能性があります。

保守担当者向けメモ

  • 本関数は log_out への単なる転送関数です。ログの出力形式を変更する場合は log_out を修正してください。
  • support/suexec.c に存在する同名の関数と混同しないよう注意してください。
