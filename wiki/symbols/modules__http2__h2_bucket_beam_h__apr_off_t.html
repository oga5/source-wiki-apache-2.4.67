apr_off_t

定義

  • 種別: typedef
  • 定義箇所: modules/http2/h2_bucket_beam.h:32

目的

ファイルサイズ、オフセット、またはデータ転送量（バイト数）を表現するための、プラットフォーム依存の符号付き整数型です。

シグネチャまたは形状

typedef /* 符号付き整数型 */ apr_off_t;

入力と出力

  • **入力**: 該当なし（型定義のため）。
  • **出力**: 該当なし（型定義のため）。

前提条件と不変条件

  • Apache Portable Runtime (APR) の標準的な型であり、システム上のファイルオフセットやバイト数を扱うために使用されます。
  • 負の値は通常、エラー状態や無効なオフセットを示すために使用される場合があります。

アルゴリズムまたは振る舞い

  • 本モジュール内では、主に h2_beam_io_callback の引数として、転送または消費されたデータのバイト数を通知するために使用されます。
  • 算術演算や比較演算は、標準的な C 言語の符号付き整数型として振る舞います。

呼び出し元と呼び出し先

  • **呼び出し元**: h2_bucket_beam モジュール内のコールバック関数定義で使用されています。
  • **呼び出し先**: 該当なし（型定義のため）。
  • **補足**: HTTPD コア（http_core.h, httpd.h）の広範な API で使用されており、リクエストボディの制限値や送信済みバイト数の管理に利用されています。

関連データ構造

  • h2_beam_io_callback: この型を引数として受け取るコールバック関数型。
  • apr_bucket: h2_bucket_beam で管理されるデータチャンク。

失敗モードと境界条件

  • **オーバーフロー**: 非常に大きなファイルやデータストリームを扱う場合、プラットフォームの apr_off_t の最大値を超える可能性があります。
  • **符号**: 負の値を渡す際の挙動は、各 API の仕様に依存します。

保守担当者向けメモ

  • 本シンボルは APR によって提供される型であり、h2_bucket_beam.h で再定義されているわけではありません。
  • HTTP/2 モジュール内では、主にデータ消費量の報告（h2_beam_report_consumption 等）におけるバイト数カウントとして一貫して使用してください。
