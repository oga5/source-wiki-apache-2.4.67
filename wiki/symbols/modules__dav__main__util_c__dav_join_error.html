dav_join_error

定義

  • 種別: function
  • 定義箇所: modules/dav/main/util.c:80

目的

2つの dav_error エラーオブジェクトを連結し、単一のエラーチェーン（スタック）を構築します。

シグネチャまたは形状

dav_error* dav_join_error(dav_error *dest, dav_error *src)

入力と出力

  • **入力**:
  • dest: 連結先となるエラーオブジェクト（または NULL）。
  • src: 連結対象となるエラーオブジェクト（または NULL）。
  • **出力**: 連結後のエラーチェーンの先頭ポインタを返します。

前提条件と不変条件

  • dest が NULL の場合、src をそのまま返します。
  • src が NULL の場合、dest をそのまま返します。
  • 連結は dest が指すエラーチェーンの末尾に対して行われます。

アルゴリズムまたは振る舞い

1. src が NULL ならば dest を返して終了します。 2. dest が NULL ならば src を返して終了します。 3. dest から辿れるエラーチェーンの末尾（prev が NULL であるノード）を探索します。 4. 発見した末尾ノードの prev メンバに src を代入し、連結します。 5. 連結後の dest を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/dav/main/mod_dav.c:1160
  • modules/dav/main/mod_dav.c:1178
  • **呼び出し先**:
  • 本関数は直接的な関数呼び出しを行いません。

関連データ構造

  • dav_error: エラー情報を保持する構造体。prev メンバを介してエラーのスタック（連結リスト）を形成します。

失敗モードと境界条件

  • 両方の引数が NULL の場合、NULL を返します。
  • 循環参照については考慮されていません。

保守担当者向けメモ

  • 本関数は dav_error 構造体の prev ポインタを書き換えることで連結を実現しています。エラーチェーンの順序は dest が先頭、src が後続となります。
