ssl_stapling_certid_free

定義

  • 種別: function
  • 定義箇所: modules/ssl/ssl_util_stapling.c:75

目的

APR プール管理下にある OCSP_CERTID オブジェクトを解放するためのクリーンアップ関数です。

シグネチャまたは形状

static apr_status_t ssl_stapling_certid_free(void *data)

入力と出力

  • **入力**: void *data - 解放対象の OCSP_CERTID 構造体へのポインタ。
  • **出力**: apr_status_t - 常に APR_SUCCESS を返します。

前提条件と不変条件

  • 入力 data は OCSP_CERTID 型としてキャスト可能である必要があります。
  • data が NULL の場合、関数は何もせず正常終了します。

アルゴリズムまたは振る舞い

1. 引数 data を OCSP_CERTID *cid にキャストします。 2. cid が NULL でないことを確認します。 3. OpenSSL の OCSP_CERTID_free(cid) を呼び出し、メモリを解放します。 4. APR_SUCCESS を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: modules/ssl/ssl_util_stapling.c:225 において、apr_pool_cleanup_register の引数として登録されています。
  • **呼び出し先**: OpenSSL の OCSP_CERTID_free を呼び出します。

関連データ構造

  • OCSP_CERTID: OpenSSL が定義する OCSP 証明書識別子構造体。

失敗モードと境界条件

  • data が NULL の場合: 安全に無視され、APR_SUCCESS が返されます。
  • OCSP_CERTID_free の内部動作に依存しますが、本関数自体はエラーを報告しません。

保守担当者向けメモ

  • 本関数は apr_pool_cleanup_register を介して呼び出されることを前提としたラッパーです。直接呼び出すことは想定されていません。
