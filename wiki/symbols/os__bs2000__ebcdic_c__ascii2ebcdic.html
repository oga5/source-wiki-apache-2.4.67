ascii2ebcdic

定義

  • 種別: function
  • 定義箇所: os/bs2000/ebcdic.c:203

目的

メモリブロック内のデータを ASCII 文字コードから EBCDIC 文字コードへ変換します。

シグネチャまたは形状

void ascii2ebcdic(unsigned char *dest, const unsigned char *srce, size_t count)

入力と出力

  • dest: 変換後の EBCDIC データを格納するバッファへのポインタ。
  • srce: 変換元の ASCII データが格納されたバッファへのポインタ。
  • count: 変換するバイト数。
  • 戻り値: なし。

前提条件と不変条件

  • dest と srce が指すメモリ領域が重なっていないこと。
  • os_toebcdic テーブルが有効な変換テーブルとして参照可能であること。

アルゴリズムまたは振る舞い

1. count が 0 になるまでループを繰り返します。 2. srce が指す現在のバイトをインデックスとして os_toebcdic テーブルを参照します。 3. 変換後の値を dest が指す位置に書き込みます。 4. dest と srce のポインタをそれぞれインクリメントし、count をデクリメントします。

呼び出し元と呼び出し先

  • 呼び出し元: 提供されたソース情報からは特定の呼び出し元は特定できません。
  • 呼び出し先: os_toebcdic（変換テーブル）を参照します。

関連データ構造

  • os_toebcdic: ASCII から EBCDIC への変換を行うための静的ルックアップテーブル。

失敗モードと境界条件

  • count が 0 の場合、ループは実行されず何も変換されません。
  • メモリ領域が重なっている場合、変換結果は未定義となります。

保守担当者向けメモ

  • 本関数は単純なバイト単位のルックアップ変換です。EBCDIC 環境特有の文字コード不連続性を吸収するために必須の処理です。
  • 変換テーブル os_toebcdic の定義は os/bs2000/ebcdic.c 内に存在します。
