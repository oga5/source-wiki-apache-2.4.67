status_get_certs_json

定義

  • 種別: function
  • 定義箇所: modules/md/md_status.c:197

目的

Managed Domain (MD) に関連付けられた証明書リストを JSON オブジェクトに変換し、各証明書の詳細情報および有効期間の共通範囲を生成する。

シグネチャまたは形状

static apr_status_t status_get_certs_json(md_json_t **pjson, apr_array_header_t *certs,
                                          int from_staging,
                                          const md_t *md, md_reg_t *reg,  
                                          md_ocsp_reg_t *ocsp, int with_logs,
                                          apr_pool_t *p)

入力と出力

  • **入力**:
  • certs: 証明書オブジェクトの配列。
  • from_staging: ステージング領域の証明書を対象とするかのフラグ。
  • md, reg, ocsp: MD 設定、レジストリ、OCSP レジストリへの参照。
  • with_logs: ログ情報を含めるかのフラグ。
  • p: メモリ割り当て用プール。
  • **出力**:
  • pjson: 生成された JSON オブジェクトへのポインタ。成功時は APR_SUCCESS を返し、失敗時は NULL を設定してエラーコードを返す。

前提条件と不変条件

  • certs 配列内の各要素は md_cert_t* 型であること。
  • p は有効な apr_pool_t であること。
  • md_json_create が成功すること。

アルゴリズムまたは振る舞い

1. md_json_create で空の JSON オブジェクトを初期化する。 2. get_cert_count で取得した件数分ループし、各証明書に対して status_get_cert_json_ex を呼び出して詳細 JSON を生成する。 3. 各証明書の有効期間（md_cert_get_valid）を取得し、全証明書の共通有効期間（md_timeperiod_common）を算出する。 4. get_cert_name で生成したキー名を使用して、各証明書の JSON をメインの JSON オブジェクトに格納する。 5. 全証明書の共通有効期間が算出された場合、MD_KEY_VALID キーで JSON に追加する。 6. 成功時は *pjson に結果を格納し、APR_SUCCESS を返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/md/md_status.c:248 (get_staging_certs_json 内)
  • modules/md/md_status.c:274 (status_get_md_json 内)
  • **呼び出し先**:
  • get_cert_count, get_cert_name, status_get_cert_json_ex, md_json_create, md_json_setj, md_json_set_timeperiod, md_cert_get_valid, md_timeperiod_common, APR_ARRAY_IDX

関連データ構造

  • md_json_t: JSON 構造体。
  • md_cert_t: 証明書情報。
  • md_timeperiod_t: 証明書の有効期間範囲。

失敗モードと境界条件

  • status_get_cert_json_ex が失敗した場合、処理を中断しエラーコードを返す。
  • certs 配列内の要素が NULL の場合、その要素はスキップされる。
  • certs_valid は初期化時に {0, 0} となり、証明書が存在しない場合は JSON に有効期間情報は追加されない。

保守担当者向けメモ

  • 本関数は md_status モジュールにおける証明書情報のシリアライズの要所である。
  • MD_KEY_VALID の出力は、証明書が少なくとも1つ存在する場合にのみ行われる。
  • from_staging フラグの切り替えにより、本番用とステージング用の証明書を動的に切り替えて処理する設計となっている。
