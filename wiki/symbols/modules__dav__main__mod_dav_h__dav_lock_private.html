dav_lock_private

定義

  • 種別: struct
  • 定義箇所: modules/dav/main/mod_dav.h:1332

目的

ロックデータベース（lockdb）の実装において、リポジトリ固有のロック情報を保持するための不透明な（opaque）データ構造です。

シグネチャまたは形状

typedef struct dav_lock_private dav_lock_private;

入力と出力

本構造体は型定義のみであり、具体的なメンバー変数は公開されていません。各プロバイダーの実装において、ロックに関連する内部状態を格納するために使用されます。

前提条件と不変条件

  • dav_lock 構造体内の info メンバーとして保持されることを想定しています。
  • 構造体のメモリ管理は、関連付けられたリクエストプール（apr_pool_t）の生存期間に依存します。

アルゴリズムまたは振る舞い

本構造体自体はデータ保持用であり、具体的な振る舞いは各プロバイダー（mod_dav_fs 等）の実装に委ねられます。

呼び出し元と呼び出し先

  • 呼び出し元（参照箇所）:
  • modules/dav/fs/lock.c:205
  • modules/dav/lock/locks.c:201
  • modules/dav/main/mod_dav.h:1411 (dav_lock 構造体内の info メンバーとして定義)
  • 呼び出し先:
  • 不明（各プロバイダーの実装に依存します）

関連データ構造

  • dav_lock: 本構造体を info メンバーとして保持する上位のロック管理構造体。

失敗モードと境界条件

  • 本構造体は不透明なポインタとして扱われるため、プロバイダー間でキャストして使用する場合、型安全性の確保は実装側の責任となります。

保守担当者向けメモ

  • 本構造体は mod_dav のプロバイダーインターフェースの一部であり、具体的な実装は mod_dav のコアではなく、各ストレージプロバイダー（fs や lock モジュール）側に存在します。変更を加える際は、関連するすべてのプロバイダー実装への影響を確認してください。
