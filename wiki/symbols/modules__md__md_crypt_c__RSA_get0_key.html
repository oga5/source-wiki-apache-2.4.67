RSA_get0_key

定義

  • 種別: function
  • 定義箇所: modules/md/md_crypt.c:986

目的

OpenSSL の RSA 構造体から、公開鍵の構成要素（法 n、公開指数 e、秘密指数 d）へのポインタを取得する。

シグネチャまたは形状

static void RSA_get0_key(const RSA *r,
                         const BIGNUM **n, const BIGNUM **e, const BIGNUM **d)

入力と出力

  • **入力**:
  • r: 読み取り対象の RSA 構造体へのポインタ。
  • n, e, d: それぞれ法、公開指数、秘密指数のポインタを格納するためのアドレス。NULL を渡すことで取得をスキップ可能。
  • **出力**: なし（戻り値は void）。引数経由で BIGNUM へのポインタが書き込まれる。

前提条件と不変条件

  • r は有効な RSA 構造体を指している必要がある。
  • 取得対象のポインタ（n, e, d）が NULL でない場合、そのアドレスに RSA 構造体内部の BIGNUM ポインタが代入される。

アルゴリズムまたは振る舞い

1. n が NULL でなければ、r->n を代入する。 2. e が NULL でなければ、r->e を代入する。 3. d が NULL でなければ、r->d を代入する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/md/md_crypt.c:1026
  • modules/md/md_crypt.c:1057
  • **呼び出し先**: なし（OpenSSL 構造体のメンバアクセスのみ）。

関連データ構造

  • RSA: OpenSSL の RSA 鍵構造体。
  • BIGNUM: OpenSSL の多倍長整数構造体。

失敗モードと境界条件

  • r が NULL の場合、プログラムはクラッシュする可能性がある（ガードなし）。
  • n, e, d が NULL の場合、その要素の取得は安全にスキップされる。

保守担当者向けメモ

  • 本関数は OpenSSL のバージョン差異を吸収するためのラッパーとして実装されている。
  • RSA 構造体のメンバに直接アクセスしているため、OpenSSL の API バージョン更新時には構造体の定義変更に注意すること。
