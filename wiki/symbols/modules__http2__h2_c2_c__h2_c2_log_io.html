h2_c2_log_io

定義

  • 種別: function
  • 定義箇所: modules/http2/h2_c2.c:138

目的

セカンダリ接続（c2）において、送信されたバイト数をログ記録用モジュールへ通知する。

シグネチャまたは形状

static void h2_c2_log_io(conn_rec *c2, apr_off_t bytes_sent)

入力と出力

  • c2: 送信元となるセカンダリ接続の conn_rec 構造体。
  • bytes_sent: 送信されたバイト数（apr_off_t）。
  • 出力: なし。

前提条件と不変条件

  • h2_c_logio_add_bytes_out が外部で定義されている場合にのみ実行される。
  • bytes_sent が 0 以外の値である場合にのみ処理が行われる。

アルゴリズムまたは振る舞い

1. bytes_sent が 0 でないか確認する。 2. グローバル関数ポインタ h2_c_logio_add_bytes_out が非 NULL であるか確認する。 3. 上記条件を満たす場合、h2_c_logio_add_bytes_out(c2, bytes_sent) を呼び出し、ログ記録処理を委譲する。

呼び出し元と呼び出し先

  • 呼び出し元:
  • modules/http2/h2_c2.c:152
  • modules/http2/h2_c2.c:164
  • 呼び出し先:
  • h2_c_logio_add_bytes_out（外部のログ記録用関数ポインタ）

関連データ構造

  • conn_rec: Apache HTTP Server の接続構造体。

失敗モードと境界条件

  • h2_c_logio_add_bytes_out が初期化されていない（NULL）場合、ログ記録はスキップされる。
  • bytes_sent が 0 の場合、関数は何も行わずに終了する。

保守担当者向けメモ

  • 本関数は mod_logio 等のログ記録モジュールとの連携を目的としています。
  • h2_c_logio_add_bytes_out は mod_http2 の初期化プロセスで設定される外部シンボルです。
