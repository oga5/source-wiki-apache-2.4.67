X509

定義

  • 種別: struct
  • 定義箇所: modules/ssl/ssl_engine_pphrase.c:899

目的

OpenSSL ライブラリにおける X.509 証明書データを保持するための構造体です。

シグネチャまたは形状

OpenSSL の公開ヘッダーで定義される不透明な構造体（opaque structure）です。本ソースファイル内では、証明書IDと証明書ポインタをペアで管理する params 構造体のメンバとして参照されています。

入力と出力

  • 入力: 該当なし（本ファイル内では型定義およびポインタの保持に使用）。
  • 出力: 該当なし。

前提条件と不変条件

  • OpenSSL ライブラリがリンクされている必要があります。
  • ssl_private.h を通じて OpenSSL のヘッダーが適切にインクルードされている必要があります。

アルゴリズムまたは振る舞い

本ファイル内では、証明書を識別するためのコンテキストの一部として保持されるのみであり、この構造体自体が特定のアルゴリズムを実装しているわけではありません。

呼び出し元と呼び出し先

  • 呼び出し元: modules/ssl/ssl_engine_pphrase.c 内のコードが、証明書オブジェクトを保持するために使用しています。
  • 呼び出し先: 構造体の内部実装については、OpenSSL ライブラリの API に依存します。
  • 補足: modules/md/md_crypt.c 等の他ファイルからも参照されていますが、本ファイル内での具体的な呼び出し関係はソース抜粋からは特定できません。

関連データ構造

  • params (ローカル構造体): cert_id (const char*) と cert (X509*) を保持するペア構造体。

失敗モードと境界条件

  • 構造体自体は型定義であるため、失敗モードはありません。ただし、この型を指すポインタが NULL である場合、OpenSSL API 呼び出し時にセグメンテーションフォールトやエラーが発生する可能性があります。

保守担当者向けメモ

  • 本ファイルにおける X509 は OpenSSL の外部定義を参照しています。
  • modules/ssl/ssl_engine_pphrase.c 内での利用は、証明書とIDの紐付け管理に限定されています。
  • 構造体の詳細なメモリレイアウトや内部メンバへの直接アクセスは、OpenSSL のバージョンアップにより変更される可能性があるため避けてください。
