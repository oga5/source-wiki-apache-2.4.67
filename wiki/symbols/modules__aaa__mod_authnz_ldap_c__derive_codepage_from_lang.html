derive_codepage_from_lang

定義

  • 種別: function
  • 定義箇所: modules/aaa/mod_authnz_ldap.c:120

目的

指定された言語コード文字列から、対応する文字コードセット（エンコーディング名）を導出する。

シグネチャまたは形状

char* derive_codepage_from_lang(apr_pool_t *p, char *language)

入力と出力

  • **入力**:
  • p: メモリ割り当てに使用する apr_pool_t。
  • language: 言語コードを表す文字列（例: "en-US"）。
  • **出力**:
  • 導出された文字コードセット名（文字列）。該当がない場合は NULL。

前提条件と不変条件

  • charset_conversions という名前のグローバルなハッシュテーブルが、言語コードから文字コードセットへのマッピングを保持している必要がある。
  • language が NULL の場合、デフォルト値として "ISO-8859-1" を返す。

アルゴリズムまたは振る舞い

1. language が NULL ならば "ISO-8859-1" を複製して返す。 2. charset_conversions ハッシュテーブルから language をキーとして検索する。 3. 見つからない場合、かつ language が "xx-YY" のような形式（長さ3以上で3文字目が '-'）であれば、最初の2文字（言語部分）のみを抽出し、再度ハッシュテーブルを検索する。 4. 検索結果が見つかった場合、その値を p プール上に複製して返す。見つからなければ NULL を返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/aaa/mod_authnz_ldap.c:159
  • modules/aaa/mod_authnz_ldap.c:1988
  • **呼び出し先**:
  • apr_pstrdup (APR)
  • apr_hash_get (APR)
  • apr_pstrndup (APR)
  • strlen (標準Cライブラリ)

関連データ構造

  • charset_conversions: 言語コードと文字コードセットのマッピングを保持するハッシュテーブル。

失敗モードと境界条件

  • language が NULL の場合はデフォルト値が返される。
  • language に対応するエントリがハッシュテーブルに存在しない場合、NULL が返される。
  • 文字列の長さが3未満で、かつハッシュテーブルに存在しない場合、短縮形へのフォールバックは行われない。

保守担当者向けメモ

  • 本関数は charset_conversions ハッシュテーブルの構造に強く依存している。このテーブルの初期化や内容が変更された場合、本関数の挙動に影響する可能性がある。
