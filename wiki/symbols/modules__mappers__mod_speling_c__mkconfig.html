mkconfig

定義

  • 種別: function
  • 定義箇所: modules/mappers/mod_speling.c:73

目的

spconfig 構造体のメモリを確保し、デフォルト値で初期化する。

シグネチャまたは形状

static void *mkconfig(apr_pool_t *p)

入力と出力

  • **入力**: apr_pool_t *p (メモリ割り当てに使用するプール)
  • **出力**: 初期化済みの spconfig * 構造体へのポインタ

前提条件と不変条件

  • apr_pcalloc により、構造体内の全メンバはゼロ初期化された状態で開始される。

アルゴリズムまたは振る舞い

1. apr_pcalloc を使用して spconfig サイズのメモリをプールから確保する。 2. enabled を 0 (無効) に設定する。 3. check_case_only を 0 (無効) に設定する。 4. check_basename_match を 1 (有効) に設定する。 5. 初期化した構造体ポインタを返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/mappers/mod_speling.c:89
  • modules/mappers/mod_speling.c:97
  • modules/generators/mod_suexec.c:47 (注: ツールによる推測)
  • modules/generators/mod_suexec.c:55 (注: ツールによる推測)
  • **呼び出し先**: apr_pcalloc (APRライブラリ)

関連データ構造

  • spconfig: 本関数で初期化される設定保持用構造体。

失敗モードと境界条件

  • apr_pcalloc がメモリ不足により失敗した場合、APRの規約に従いプロセスが終了する可能性がある。

保守担当者向けメモ
