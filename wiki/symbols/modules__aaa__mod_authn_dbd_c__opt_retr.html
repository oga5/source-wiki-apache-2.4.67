opt_retr

定義

  • 種別: function
  • 定義箇所: modules/aaa/mod_authn_dbd.c:280

目的

認証キャッシュ保存用のオプション関数 ap_authn_cache_store を取得し、モジュール内のグローバル変数 authn_cache_store に格納する。

シグネチャまたは形状

static void opt_retr(void)

入力と出力

  • 入力: なし（内部で APR_RETRIEVE_OPTIONAL_FN を使用）
  • 出力: なし（副作用としてグローバル変数 authn_cache_store を更新）

前提条件と不変条件

  • ap_authn_cache_store が他のモジュールによって提供されている必要がある。
  • ap_hook_optional_fn_retrieve を介して呼び出されることを前提としている。

アルゴリズムまたは振る舞い

1. APR_RETRIEVE_OPTIONAL_FN マクロを使用して、ap_authn_cache_store 関数ポインタを検索する。 2. 取得した関数ポインタを authn_cache_store に代入する。

呼び出し元と呼び出し先

  • 呼び出し元: ap_hook_optional_fn_retrieve により、Apache の初期化フェーズで呼び出される。
  • 呼び出し先: ap_authn_cache_store（オプション関数として解決）。

関連データ構造

  • authn_cache_store: 取得した関数ポインタを保持する変数。

失敗モードと境界条件

  • ap_authn_cache_store が提供されていない場合、authn_cache_store は NULL になる。この場合、後続のキャッシュ保存処理は実行されない。

保守担当者向けメモ

  • 本関数は mod_authn_socache 等が提供するキャッシュ機能との疎結合な連携を実現するためのものです。
  • 同名の opt_retr 関数が mod_authn_dbm.c や mod_authn_file.c にも存在しますが、それぞれが自身のモジュールスコープ内でキャッシュ関数を解決しています。
