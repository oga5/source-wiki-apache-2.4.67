md_ocsp_cert_stat_name

定義

  • 種別: function
  • 定義箇所: modules/md/md_ocsp.c:115

目的

OCSP 証明書ステータスを表す列挙型 md_ocsp_cert_stat_t の値を、ログ出力や JSON 生成に適した人間が読める文字列（"good", "revoked", "unknown"）に変換します。

シグネチャまたは形状

const char *md_ocsp_cert_stat_name(md_ocsp_cert_stat_t stat)

入力と出力

  • **入力**: md_ocsp_cert_stat_t 型のステータス値。
  • **出力**: 対応する文字列リテラル（const char *）。

前提条件と不変条件

  • 入力値が MD_OCSP_CERT_ST_GOOD または MD_OCSP_CERT_ST_REVOKED 以外の場合、デフォルトとして "unknown" を返します。

アルゴリズムまたは振る舞い

  • switch 文を用いて入力値を評価します。
  • MD_OCSP_CERT_ST_GOOD に対しては "good" を返します。
  • MD_OCSP_CERT_ST_REVOKED に対しては "revoked" を返します。
  • それ以外の値に対しては "unknown" を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/md/md_ocsp.c:232
  • modules/md/md_ocsp.c:1017
  • modules/md/md_status.c:165
  • **呼び出し先**:
  • 本関数は単一の変換ロジックであり、他の関数を呼び出しません。

関連データ構造

  • md_ocsp_cert_stat_t（列挙型）

失敗モードと境界条件

  • 定義されていない列挙値が渡された場合、エラーを発生させるのではなく "unknown" を返す安全な設計となっています。

保守担当者向けメモ

  • 新しいステータス値が md_ocsp_cert_stat_t に追加された場合は、本関数の switch 文にもケースを追加してください。
