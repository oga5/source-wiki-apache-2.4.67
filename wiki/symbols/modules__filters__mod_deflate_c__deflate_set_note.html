deflate_set_note

定義

  • 種別: function
  • 定義箇所: modules/filters/mod_deflate.c:261

目的

Apache 設定ディレクティブ DeflateFilterNote を処理し、圧縮統計（圧縮率、入力サイズ、出力サイズ）を記録するためのメモ名を deflate_filter_config 構造体に設定する。

シグネチャまたは形状

static const char *deflate_set_note(cmd_parms *cmd, void *dummy,
                                    const char *arg1, const char *arg2)

入力と出力

  • **入力**:
  • cmd: 設定コマンドのパラメータ構造体。
  • dummy: 使用されない引数。
  • arg1: 設定タイプ（"ratio", "input", "output"）またはメモ名。
  • arg2: メモ名（arg1 がタイプ指定の場合）。
  • **出力**: 成功時は NULL を返し、不明なタイプが指定された場合はエラーメッセージ文字列を返す。

前提条件と不変条件

  • cmd->server->module_config から deflate_module の設定構造体を取得できること。
  • arg1 がタイプ指定（"ratio", "input", "output"）の場合、arg2 はメモ名として解釈される。

アルゴリズムまたは振る舞い

1. ap_get_module_config を使用して、現在のサーバー設定 deflate_filter_config を取得する。 2. arg2 が NULL の場合、arg1 を圧縮率のメモ名 (note_ratio_name) として設定する。 3. arg2 が存在する場合、arg1 を大文字小文字を区別せずに比較する:

  • "ratio": note_ratio_name に arg2 を設定。
  • "input": note_input_name に arg2 を設定。
  • "output": note_output_name に arg2 を設定。

    • 4. いずれにも該当しない場合、apr_psprintf を使用してエラーメッセージを生成し返す。

呼び出し元と呼び出し先

  • **呼び出し元**: modules/filters/mod_deflate.c:1921 において、AP_INIT_TAKE12 マクロ経由で DeflateFilterNote ディレクティブのハンドラとして登録されている。
  • **呼び出し先**: ap_get_module_config（Apache API）、strcasecmp（標準Cライブラリ）、apr_psprintf（APR）。

関連データ構造

  • deflate_filter_config: サーバー全体の設定を保持する構造体。note_ratio_name, note_input_name, note_output_name フィールドを更新する。

失敗モードと境界条件

  • 不明なメモタイプが指定された場合、apr_psprintf によりエラー文字列が返され、設定の読み込みが失敗する。

保守担当者向けメモ

  • 本関数は DeflateFilterNote ディレクティブの引数解析に特化している。
  • arg1 と arg2 の組み合わせによる柔軟な引数指定（DeflateFilterNote name または DeflateFilterNote type name）に対応している。
