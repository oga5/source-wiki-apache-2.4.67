h2_mpm_supported

定義

  • 種別: function
  • 定義箇所: modules/http2/h2_c2.c:125

目的

現在の Apache HTTP Server の MPM（Multi-Processing Module）が、HTTP/2 のワーカーモデル（セカンダリ接続）をサポートしているかどうかを判定する。

シグネチャまたは形状

int h2_mpm_supported(void)

入力と出力

  • 入力: なし
  • 出力: int 型。サポートされている場合は非ゼロ（真）、そうでない場合は 0（偽）を返す。

前提条件と不変条件

  • 内部で check_modules(0) を呼び出し、MPM のサポート状況を初期化・確認する。
  • 戻り値は静的な mpm_supported 変数の状態に依存する。

アルゴリズムまたは振る舞い

1. check_modules(0) を実行し、MPM の互換性チェックを強制する。 2. 内部保持されている mpm_supported の値を返す。

呼び出し元と呼び出し先

  • 呼び出し元（直接的な参照）:
  • modules/http2/h2_switch.c:61
  • modules/http2/h2_switch.c:162
  • modules/http2/mod_http2.c:129
  • 呼び出し先:
  • check_modules（同ファイル内）

関連データ構造

  • mpm_supported: h2_c2.c 内で管理される静的変数。

失敗モードと境界条件

  • check_modules が未実行または失敗した場合、mpm_supported の初期値（通常は 0）が返される。

保守担当者向けメモ

  • 本関数は h2_c2.h で宣言されており、HTTP/2 モジュールの有効化可否を判断するゲートキーパーとして機能する。
  • MPM の判定ロジックを変更する場合は、check_modules の実装を確認すること。
