md_acme_acct_matches_url

定義

  • 種別: function
  • 定義箇所: modules/md/md_acme_acct.c:237

目的

指定された ACME アカウントが、特定の ACME サーバー URL に関連付けられているかを判定します。

シグネチャまたは形状

int md_acme_acct_matches_url(md_acme_acct_t *acct, const char *url)

入力と出力

  • **入力**:
  • acct: 照合対象の md_acme_acct_t 構造体へのポインタ。
  • url: 比較対象となる ACME サーバーの URL 文字列。
  • **出力**:
  • 1: アカウントの ca_url が指定された url と完全に一致する場合。
  • 0: 一致しない場合、または入力が不正な場合。

前提条件と不変条件

  • acct が NULL でないこと。
  • acct->ca_url は、アカウントが登録された ACME サーバーの URL を保持している必要があります。

アルゴリズムまたは振る舞い

1. url が NULL であるか、acct->ca_url が NULL である場合は 0 を返します。 2. strcmp を使用して acct->ca_url と url を比較します。 3. 文字列が完全に一致すれば 1 を返し、そうでなければ 0 を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/md/md_acme.c:556
  • modules/md/md_acme_acct.c:246
  • **呼び出し先**:
  • 標準ライブラリ関数 strcmp を使用しています。

関連データ構造

  • md_acme_acct_t: ACME アカウント情報を保持する構造体。本関数はこの構造体内の ca_url メンバを参照します。

失敗モードと境界条件

  • url または acct->ca_url が NULL の場合、安全に 0 を返します。
  • URL の大文字・小文字や末尾のスラッシュの有無など、文字列として完全一致しない場合は不一致とみなされます。

保守担当者向けメモ

  • 本関数は ACME サーバーの URL による厳密な照合を行います。URL の正規化が必要な場合は、呼び出し元で事前に処理を行う必要があります。
