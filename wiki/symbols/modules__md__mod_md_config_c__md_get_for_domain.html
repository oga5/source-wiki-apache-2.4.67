md_get_for_domain

定義

  • 種別: function
  • 定義箇所: modules/md/mod_md_config.c:1525

目的

指定されたサーバー（server_rec）の設定から、特定のドメイン名を含む管理対象ドメイン（md_t）を検索して返します。

シグネチャまたは形状

const md_t *md_get_for_domain(server_rec *s, const char *domain)

入力と出力

  • **入力**:
  • s: 検索対象の Apache サーバーレコード。
  • domain: 検索するドメイン名（文字列）。
  • **出力**:
  • 指定されたドメインを管理している md_t 構造体へのポインタ。該当するものがない場合は NULL を返します。

前提条件と不変条件

  • s は有効な server_rec である必要があります。
  • md_config_get(s) により取得される md_srv_conf_t 構造体が初期化されている必要があります。

アルゴリズムまたは振る舞い

1. md_config_get(s) を呼び出し、サーバーに関連付けられた設定構造体 sc を取得します。 2. sc->assigned に格納されている管理対象ドメインの配列を順次走査します。 3. 各 md_t 要素に対して md_contains(md, domain, 0) を実行し、指定されたドメインが含まれているか確認します。 4. ドメインが見つかった場合、その md_t へのポインタを返します。 5. 配列の最後まで見つからない場合は NULL を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/md/mod_md.c:1500
  • **呼び出し先**:
  • md_config_get (設定取得)
  • md_contains (ドメイン包含判定)
  • APR_ARRAY_IDX (配列要素アクセス)

関連データ構造

  • md_t: 管理対象ドメインの定義構造体。
  • md_srv_conf_t: サーバー単位の設定構造体。
  • server_rec: Apache HTTP Server のサーバーレコード。

失敗モードと境界条件

  • sc が NULL の場合、または sc->assigned が初期化されていない（NULL）場合、ループは実行されず NULL が返されます。
  • 指定されたドメインがどの管理対象ドメインにも属していない場合、NULL が返されます。

保守担当者向けメモ

  • 本関数は mod_md_config.h で宣言されており、モジュール間でのドメイン検索の標準的な手段として利用されます。
  • md_contains の第3引数に 0 を渡しているため、大文字小文字の区別や特定のフラグ設定については md_contains の実装仕様に依存します。
