dav_get_liveprop_info

定義

  • 種別: function
  • 定義箇所: modules/dav/main/liveprop.c:111

目的

指定されたプロパティIDに対応するライブプロパティの仕様情報（dav_liveprop_spec）を、プロバイダ固有のグループ定義から逆引きし、その名前空間のグローバルインデックスを返します。

シグネチャまたは形状

long dav_get_liveprop_info(int propid,
                           const dav_liveprop_group *group,
                           const dav_liveprop_spec **info)

入力と出力

  • **入力**:
  • propid: 検索対象のプロパティID。
  • group: 検索対象となる dav_liveprop_group 構造体へのポインタ。
  • info: 見つかった仕様情報を格納するための dav_liveprop_spec へのポインタのポインタ。
  • **出力**:
  • 戻り値: 成功時は対応するグローバル名前空間インデックス（long）、見つからない場合は 0。
  • info: 成功時は該当する dav_liveprop_spec 構造体のアドレス、見つからない場合は NULL。

前提条件と不変条件

  • group は有効な dav_liveprop_group 構造体を指している必要があります。
  • group->specs 配列は name メンバが NULL である要素で終端されている必要があります。

アルゴリズムまたは振る舞い

1. group->specs 配列を先頭から順に走査します。 2. 各要素の propid が引数の propid と一致するか確認します。 3. 一致した場合、その要素のアドレスを *info に代入します。 4. group->namespace_uris 配列から、当該プロパティが属する名前空間のインデックスを解決し、dav_get_liveprop_ns_index を呼び出してグローバルインデックスを取得・返却します。 5. 配列の終端に達しても一致しない場合、*info を NULL に設定し、0 を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/dav/fs/repos.c
  • modules/dav/main/std_liveprop.c
  • **呼び出し先**:
  • dav_get_liveprop_ns_index (名前空間インデックスの解決)

関連データ構造

  • dav_liveprop_group: プロパティの仕様セットと名前空間のリストを保持する構造体。
  • dav_liveprop_spec: 個々のライブプロパティの定義（ID、名前、名前空間インデックス等）を保持する構造体。

失敗モードと境界条件

  • 指定された propid が group 内に存在しない場合、*info は NULL となり、戻り値は 0 となります。
  • group->specs が空（name が即座に NULL）の場合、直ちに失敗モードとして処理されます。

保守担当者向けメモ

  • 本関数は線形探索を行うため、group 内のプロパティ数が多い場合はパフォーマンスに影響する可能性があります。
  • 戻り値の 0 は、名前空間インデックスの解決失敗またはプロパティ未検出を意味する予約値として扱われます。
