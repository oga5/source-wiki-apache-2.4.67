dav_generic_format_locktoken

定義

  • 種別: function
  • 定義箇所: modules/dav/lock/locks.c:293

目的

dav_locktoken 構造体に含まれる UUID を WebDAV 標準の opaquelocktoken URI 形式の文字列に変換します。

シグネチャまたは形状

static const char *dav_generic_format_locktoken(apr_pool_t *p,
                                                const dav_locktoken *locktoken)

入力と出力

  • **入力**:
  • p: 文字列の割り当てに使用するメモリプール。
  • locktoken: 変換対象の dav_locktoken 構造体へのポインタ。
  • **出力**:
  • opaquelocktoken: プレフィックスが付与された、UUID 文字列形式の const char *。

前提条件と不変条件

  • locktoken 引数は有効な dav_locktoken 構造体を指している必要があります。
  • p はメモリ割り当てが可能な有効な apr_pool_t である必要があります。

アルゴリズムまたは振る舞い

1. APR_UUID_FORMATTED_LENGTH を用いて、UUID 文字列を格納するためのスタックバッファ buf を確保します。 2. apr_uuid_format を呼び出し、locktoken->uuid を buf に文字列として書き込みます。 3. apr_pstrcat を使用し、"opaquelocktoken:" と buf を連結した新しい文字列をプール p 上に生成して返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/dav/lock/locks.c:289 (直接の参照)
  • modules/dav/lock/locks.c:1222
  • **呼び出し先**:
  • apr_uuid_format (APR ライブラリ)
  • apr_pstrcat (APR ライブラリ)

関連データ構造

  • dav_locktoken: ロックを一意に識別するための UUID ラッパー構造体。

失敗モードと境界条件

  • メモリ不足が発生した場合、apr_pstrcat の内部動作によりメモリ割り当てエラーが発生する可能性があります。

保守担当者向けメモ

  • 本関数は dav_locktoken の内部表現を外部公開用の URI 形式に変換する唯一のロジックです。フォーマットの変更が必要な場合は、WebDAV 仕様との整合性に注意してください。
