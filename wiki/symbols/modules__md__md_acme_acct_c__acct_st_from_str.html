acct_st_from_str

定義

  • 種別: function
  • 定義箇所: modules/md/md_acme_acct.c:73

目的

ACME サーバーから取得したアカウント状態を表す文字列を、内部的な列挙型定数（md_acme_acct_st）に変換します。

シグネチャまたは形状

static md_acme_acct_st acct_st_from_str(const char *s)

入力と出力

  • **入力**: s (ACME サーバーから返された状態文字列、または NULL)
  • **出力**: md_acme_acct_st 型の列挙値。

前提条件と不変条件

  • 入力文字列 s が NULL の場合、関数は MD_ACME_ACCT_ST_UNKNOWN を返します。

アルゴリズムまたは振る舞い

1. 入力文字列 s が存在する場合、以下の文字列比較を行い、対応する列挙値を返します。

  • "valid" → MD_ACME_ACCT_ST_VALID
  • "deactivated" → MD_ACME_ACCT_ST_DEACTIVATED
  • "revoked" → MD_ACME_ACCT_ST_REVOKED

    • 2. 上記のいずれにも一致しない場合、または s が NULL の場合は MD_ACME_ACCT_ST_UNKNOWN を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/md/md_acme_acct.c:132
  • modules/md/md_acme_acct.c:444
  • **呼び出し先**:
  • strcmp (標準ライブラリ)

関連データ構造

  • md_acme_acct_st: アカウントの状態を定義する列挙型。

失敗モードと境界条件

  • 未知の文字列が渡された場合、エラーを発生させるのではなく MD_ACME_ACCT_ST_UNKNOWN を返して安全に処理を継続します。

保守担当者向けメモ

  • 本関数は ACME プロトコル仕様で定義されるアカウント状態の文字列と、内部実装の列挙値の橋渡しを行います。新しい状態が ACME 仕様に追加された場合は、本関数の条件分岐に追加が必要です。
