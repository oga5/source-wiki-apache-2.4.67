dav_lookup_provider

定義

  • 種別: function
  • 定義箇所: modules/dav/main/providers.c:30

目的

指定された名前の DAV プロバイダーをグローバルなレジストリから検索し、対応するプロバイダー構造体を取得します。

シグネチャまたは形状

const dav_provider *dav_lookup_provider(const char *name)

入力と出力

  • **入力**: name (検索対象のプロバイダー名を示す文字列)
  • **出力**: 該当する dav_provider 構造体へのポインタ。見つからない場合は NULL が返されます。

前提条件と不変条件

  • DAV_PROVIDER_GROUP 定数で定義されたグループ内での検索が行われます。
  • プロバイダーのバージョンは "0" に固定されています。

アルゴリズムまたは振る舞い

Apache の汎用プロバイダー API である ap_lookup_provider を呼び出し、DAV 固有のグループ識別子と指定された名前、および固定のバージョン文字列 "0" を使用してプロバイダーを検索します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/dav/main/mod_dav.c:280 (設定構造体内のプロバイダー解決に使用)
  • **呼び出し先**:
  • ap_lookup_provider (Apache Core のプロバイダー検索 API)

関連データ構造

  • dav_provider (プロバイダーのインターフェースを定義する構造体)

失敗モードと境界条件

  • 指定した name に一致するプロバイダーが登録されていない場合、ap_lookup_provider は NULL を返し、本関数もそのまま NULL を返します。

保守担当者向けメモ

  • 本関数は ap_provider API の薄いラッパーです。
  • バージョン "0" は固定値であり、将来的なバージョン管理が必要な場合は API の変更が伴います。
