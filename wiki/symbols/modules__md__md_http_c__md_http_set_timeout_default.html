md_http_set_timeout_default

定義

  • 種別: function
  • 定義箇所: modules/md/md_http.c:129

目的

HTTP セッションにおけるデフォルトの全体タイムアウト値を設定します。

シグネチャまたは形状

void md_http_set_timeout_default(md_http_t *http, apr_time_t timeout)

入力と出力

  • http: 設定対象となる md_http_t セッション構造体へのポインタ。
  • timeout: 設定するタイムアウト時間（apr_time_t 型）。
  • 戻り値: なし。

前提条件と不変条件

  • http ポインタは有効な md_http_t インスタンスを指している必要があります。

アルゴリズムまたは振る舞い

渡された timeout 値を、http 構造体内の timeout.overall フィールドに代入します。

呼び出し元と呼び出し先

  • 呼び出し元:
  • modules/md/md_acme.c:799
  • modules/md/md_ocsp.c:920
  • 呼び出し先:
  • 外部関数呼び出しはありません。

関連データ構造

  • md_http_t: HTTP クライアントのコンテキストを保持する構造体。

失敗モードと境界条件

  • http が NULL の場合、プログラムは異常終了（セグメンテーション違反）する可能性があります。

保守担当者向けメモ

  • 本関数は md_http_t の設定値を直接更新する単純なアクセサです。
  • タイムアウトの単位は apr_time_t に準拠します。
