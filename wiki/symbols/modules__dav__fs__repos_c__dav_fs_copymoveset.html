dav_fs_copymoveset

定義

  • 種別: function
  • 定義箇所: modules/dav/fs/repos.c:554

目的

WebDAV の COPY または MOVE 操作において、リソースに関連付けられたプロパティ（DBM ファイル）をコピーまたは移動する。

シグネチャまたは形状

static dav_error *dav_fs_copymoveset(int is_move, apr_pool_t *p,
                                     const dav_resource *src,
                                     const dav_resource *dst,
                                     dav_buffer *pbuf);

入力と出力

  • is_move: 操作が移動（1）かコピー（0）かを示すフラグ。
  • p: メモリ割り当て用プール。
  • src, dst: コピー元およびコピー先のリソース。
  • pbuf: 操作中に使用するバッファ。
  • 戻り値: 成功時は NULL、失敗時はエラー情報を含む dav_error 構造体へのポインタ。

前提条件と不変条件

  • dav_fs_dir_file_name が成功し、リソースのディレクトリとファイル名が取得可能であること。
  • dav_dbm_get_statefiles により取得される状態ファイル（DBM）のペア数が、コピー元とコピー先で一致していること（DAV_DEBUG 有効時）。

アルゴリズムまたは振る舞い

1. dav_fs_dir_file_name を使用して、ソースおよびデスティネーションのディレクトリとファイル名を特定する。 2. dav_dbm_get_statefiles を呼び出し、各リソースに関連する DBM 状態ファイルを特定する。 3. dav_fs_copymove_state を呼び出し、プライマリ状態ファイルをコピーまたは移動する。 4. セカンダリ状態ファイルが存在する場合、同様に dav_fs_copymove_state を呼び出す。 5. セカンダリ処理でエラーが発生した場合、HTTP 500 エラーを返し、システムが不整合な状態にあることを示すエラー記述を生成する。

呼び出し元と呼び出し先

  • 呼び出し元:
  • modules/dav/fs/repos.c:1283
  • modules/dav/fs/repos.c:1372
  • 呼び出し先:
  • dav_fs_dir_file_name
  • dav_dbm_get_statefiles
  • dav_fs_copymove_state
  • dav_new_error (デバッグ時)

関連データ構造

  • dav_resource: WebDAV リソース情報を保持。
  • dav_error: エラー状態を管理。
  • dav_buffer: 操作用の一時バッファ。

失敗モードと境界条件

  • dav_dbm_get_statefiles の結果に不整合がある場合、DAV_DEBUG モードでは HTTP_INTERNAL_SERVER_ERROR を返す。
  • プロパティのコピー/移動処理中にエラーが発生した場合、部分的な処理によりシステムが不整合な状態になる可能性がある。

保守担当者向けメモ

  • 本関数はファイルシステム上の物理ファイルではなく、プロパティを格納する DBM ファイルの操作に特化している。
  • エラー発生時のクリーンアップ処理は実装されておらず、不整合が発生した旨をエラー構造体に含めるのみである。
