md_log_level_name

定義

  • 種別: function
  • 定義箇所: modules/md/md_log.c:44

目的

ログレベルを表す列挙型値を、人間が読み取り可能な文字列（ログレベル名）に変換します。

シグネチャまたは形状

const char *md_log_level_name(md_log_level_t level)

入力と出力

  • **入力**: md_log_level_t level - 変換対象のログレベル。
  • **出力**: const char * - 対応するログレベルの名称文字列。

前提条件と不変条件

  • level は level_names 配列の有効なインデックス範囲内である必要があります。

アルゴリズムまたは振る舞い

1. 引数として受け取った level をインデックスとして使用し、静的配列 level_names から対応する文字列を取得します。 2. 取得した文字列ポインタを返します。

呼び出し元と呼び出し先

  • **呼び出し元**: ソースコード内の直接的な呼び出し元は提供されていません。
  • **呼び出し先**: level_names（ファイルスコープの静的配列）を参照します。
  • **宣言**: modules/md/md_log.h:45 にてプロトタイプ宣言されています。

関連データ構造

  • md_log_level_t: ログレベルを定義する列挙型。
  • level_names: md_log.c 内で定義されている、ログレベルに対応する文字列の配列。

失敗モードと境界条件

  • level が level_names 配列の範囲外である場合、未定義のメモリ領域を参照する可能性があります。

保守担当者向けメモ

  • 本関数は level_names 配列への単なるラッパーです。ログレベルの定義を変更する場合は、level_names 配列との整合性を維持してください。
