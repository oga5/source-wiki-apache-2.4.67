md_ocsp_cert_stat_t

定義

  • 種別: enum
  • 定義箇所: modules/md/md_ocsp.h:27

目的

OCSP（Online Certificate Status Protocol）による証明書の失効状態を表現するために使用されます。

シグネチャまたは形状

typedef enum {
    MD_OCSP_CERT_ST_UNKNOWN,
    MD_OCSP_CERT_ST_GOOD,
    MD_OCSP_CERT_ST_REVOKED,
} md_ocsp_cert_stat_t;

入力と出力

  • 入力: なし（列挙型定義）
  • 出力: MD_OCSP_CERT_ST_UNKNOWN（不明）、MD_OCSP_CERT_ST_GOOD（有効）、MD_OCSP_CERT_ST_REVOKED（失効）のいずれかの状態値。

前提条件と不変条件

  • OCSP レスポンダから取得したレスポンスを解析した結果、またはキャッシュされた状態を保持するために使用されます。

アルゴリズムまたは振る舞い

  • 証明書の検証結果を抽象化し、モジュール内で一貫したステータス管理を可能にします。

呼び出し元と呼び出し先

  • 呼び出し元（参照箇所）: modules/md/md_ocsp.c 内の各関数（ostat_set, ostat_from_json, ostat_to_json, ocsp_status_save, ocsp_get_meta, md_ocsp_get_meta など）。
  • 呼び出し先: 情報なし。

関連データ構造

  • md_ocsp_status_t: この列挙型を内部で保持するステータス構造体。
  • md_json_t: JSON 形式への変換・復元時に使用されます。

失敗モードと境界条件

  • 該当なし。

保守担当者向けメモ

  • 本列挙型は md_ocsp.c におけるステータスのシリアライズ（JSON変換）およびログ出力の基盤となります。新しい状態を追加する場合は、md_ocsp_cert_stat_name および md_ocsp_cert_stat_value の更新が必要です。
