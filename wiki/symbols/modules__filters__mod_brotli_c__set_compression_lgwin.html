set_compression_lgwin

定義

  • 種別: function
  • 定義箇所: modules/filters/mod_brotli.c:109

目的

Apache HTTP Server の設定ディレクティブ BrotliCompressionWindow の値を解析し、サーバー設定構造体 brotli_server_config_t に適用する。

シグネチャまたは形状

static const char *set_compression_lgwin(cmd_parms *cmd, void *dummy, const char *arg)

入力と出力

  • **入力**:
  • cmd: コマンドパラメータ構造体。
  • dummy: 使用されていない引数。
  • arg: 設定値（文字列）。
  • **出力**:
  • 成功時: NULL を返す。
  • 失敗時: エラーメッセージ（文字列）を返す。

前提条件と不変条件

  • arg は整数値として解釈可能であること。
  • 設定される値 val は 10 以上 24 以下である必要がある。

アルゴリズムまたは振る舞い

1. ap_get_module_config を使用して、現在のサーバー設定 brotli_server_config_t を取得する。 2. atoi(arg) で入力文字列を整数に変換する。 3. 値が 10 から 24 の範囲外である場合、エラーメッセージを返して終了する。 4. 範囲内であれば、conf->lgwin に値を格納し、NULL を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: modules/filters/mod_brotli.c:585 において、AP_INIT_TAKE1 マクロ経由で BrotliCompressionWindow ディレクティブのハンドラとして登録されている。
  • **呼び出し先**: ap_get_module_config (Apache API), atoi (標準ライブラリ)。

関連データ構造

  • brotli_server_config_t: サーバー単位の Brotli 設定を保持する構造体。

失敗モードと境界条件

  • 入力値が 10 未満または 24 を超える場合、設定の適用に失敗し、エラーメッセージが返される。
  • atoi が非数値文字列を処理した場合の挙動は標準ライブラリの仕様に依存する。

保守担当者向けメモ

  • 本関数は mod_brotli の設定フェーズでのみ実行される。
  • lgwin の値は Brotli エンコーダのウィンドウサイズ（2^lgwin）を決定する。
