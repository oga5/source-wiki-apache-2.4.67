h2_stream_state_str

定義

  • 種別: function
  • 定義箇所: modules/http2/h2_stream.c:76

目的

HTTP/2 ストリームの現在の状態を、デバッグログ出力等に適した人間が読める文字列に変換する。

シグネチャまたは形状

const char *h2_stream_state_str(const h2_stream *stream)

入力と出力

  • **入力**: h2_stream 構造体への定数ポインタ。
  • **出力**: ストリームの状態を表す文字列（const char *）。

前提条件と不変条件

  • 引数 stream は有効な h2_stream オブジェクトを指している必要がある。
  • stream->state には、h2_stream_state_t で定義された有効な状態値が格納されていること。

アルゴリズムまたは振る舞い

1. 引数として受け取った stream 構造体から state メンバを参照する。 2. h2_ss_str() 関数を呼び出し、状態値に対応する文字列を取得して返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/http2/h2_stream.h:340 (ログ出力用マクロ内での参照を確認)
  • **呼び出し先**:
  • h2_ss_str (同モジュール内の状態文字列変換関数)

関連データ構造

  • h2_stream: ストリームのライフサイクルと状態を保持する構造体。
  • h2_stream_state_t: ストリームの状態定義（IDLE, OPEN, CLOSED 等）。

失敗モードと境界条件

  • stream が NULL の場合、ポインタ参照によるクラッシュの可能性がある（ソースコード上に明示的なガードは存在しない）。

保守担当者向けメモ

  • 本関数は h2_ss_str のラッパーである。状態定義の文字列を変更する場合は h2_ss_str の実装を確認すること。
