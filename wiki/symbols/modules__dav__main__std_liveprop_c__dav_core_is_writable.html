dav_core_is_writable

定義

  • 種別: function
  • 定義箇所: modules/dav/main/std_liveprop.c:175

目的

指定されたライブプロパティが書き込み可能かどうかを判定します。

シグネチャまたは形状

static int dav_core_is_writable(const dav_resource *resource, int propid)

入力と出力

  • **入力**:
  • resource: 対象となる dav_resource 構造体へのポインタ。
  • propid: 判定対象のプロパティを識別する整数値。
  • **出力**:
  • int: プロパティが書き込み可能であれば非ゼロ、そうでなければ 0 を返します。

前提条件と不変条件

  • dav_core_liveprop_group が初期化済みである必要があります。
  • propid は dav_core_liveprop_group に登録されている有効なプロパティ ID である必要があります。

アルゴリズムまたは振る舞い

1. dav_get_liveprop_info を呼び出し、propid に対応する dav_liveprop_spec 構造体を取得します。 2. 取得した仕様構造体内の is_writable フィールドの値を返します。 3. resource 引数は現在のシグネチャに含まれていますが、この関数内では直接使用されていません。

呼び出し元と呼び出し先

  • **呼び出し元**: modules/dav/main/std_liveprop.c:196（dav_core_liveprop_group 構造体内のフックとして参照されている可能性が高い）。
  • **呼び出し先**: dav_get_liveprop_info（liveprop.c で定義されているライブプロパティ情報取得関数）。

関連データ構造

  • dav_resource: WebDAV リソースの抽象表現。
  • dav_liveprop_spec: ライブプロパティの仕様を保持する構造体。
  • dav_core_liveprop_group: DAV: 名前空間のライブプロパティ群を管理する構造体。

失敗モードと境界条件

  • dav_get_liveprop_info が失敗した場合の戻り値やエラーハンドリングについては、ソースコード上に明示的な分岐処理が存在しません。

保守担当者向けメモ

  • 本関数は DAV: 名前空間の標準プロパティに対する書き込み可否を返すためのラッパーです。
  • resource 引数が未使用であるため、将来的にリソースの状態（コレクションか否か等）に応じた動的な書き込み判定が必要になった場合は、この引数を利用してロジックを拡張してください。
