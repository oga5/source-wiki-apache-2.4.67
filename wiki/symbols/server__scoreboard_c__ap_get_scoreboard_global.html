ap_get_scoreboard_global

定義

  • 種別: function
  • 定義箇所: server/scoreboard.c:726

目的

スコアボード内のグローバル統計情報構造体へのアクセスを提供します。

シグネチャまたは形状

global_score *ap_get_scoreboard_global(void)

入力と出力

  • 入力: なし
  • 出力: global_score 構造体へのポインタ。

前提条件と不変条件

  • ap_scoreboard_image が初期化済みである必要があります。
  • 本関数は、共有メモリ上のグローバル領域を直接参照します。

アルゴリズムまたは振る舞い

グローバル変数 ap_scoreboard_image が保持するポインタから、global メンバを返します。

呼び出し元と呼び出し先

  • 呼び出し元: modules/generators/mod_status.c:262
  • 呼び出し先: なし（直接的なポインタ参照のみ）

関連データ構造

  • global_score: スコアボードのグローバル統計情報を保持する構造体。
  • ap_scoreboard_image: スコアボード全体を管理するグローバルポインタ。

失敗モードと境界条件

  • ap_scoreboard_image が NULL の場合、セグメンテーションフォールトが発生します。

保守担当者向けメモ

  • 本関数はロックを行いません。共有メモリの読み取りは、初期化後の静的なメモリ配置に依存しています。
