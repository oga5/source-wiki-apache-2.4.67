val_len_check_ctx

定義

  • 種別: struct
  • 定義箇所: modules/http2/h2_stream.c:850

目的

HTTP/2 ストリーム処理において、値の長さ制限を検証する際のコンテキスト情報を保持するために使用されます。

シグネチャまたは形状

typedef struct {
    apr_size_t maxlen;
    const char *failed_key;
} val_len_check_ctx;

入力と出力

  • maxlen: 検証対象となる最大許容長（入力）。
  • failed_key: 検証に失敗した際のキー名（出力/状態保持）。

前提条件と不変条件

  • maxlen は検証の閾値として使用されます。
  • 検証失敗時には failed_key に該当するキー名が格納されます。

アルゴリズムまたは振る舞い

本構造体は、検証処理のバトン（baton）として渡され、検証ロジック内で現在の最大長制限と、失敗が発生した場合の識別子を追跡するために利用されます。

呼び出し元と呼び出し先

  • 呼び出し元: modules/http2/h2_stream.c 内の検証ロジック（baton として参照）。
  • 呼び出し先: 情報なし。

関連データ構造

  • apr_size_t: APR ライブラリで定義されたサイズ型。

失敗モードと境界条件

  • 検証対象の値が maxlen を超過した場合、failed_key が設定されます。

保守担当者向けメモ

  • 本構造体は h2_stream.c 内の局所的な検証処理で使用されています。変更を加える際は、この構造体を利用している検証関数のシグネチャと整合性を確認してください。
