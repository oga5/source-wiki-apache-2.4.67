util_ald_sgl_dup

定義

  • 種別: function
  • 定義箇所: modules/ldap/util_ldap_cache_mgr.c:145

目的

util_compare_subgroup_t 構造体のディープコピーを作成し、指定されたキャッシュ専用メモリ領域（util_ald_cache_t）へ割り当てる。

シグネチャまたは形状

util_compare_subgroup_t *util_ald_sgl_dup(util_ald_cache_t *cache, util_compare_subgroup_t *sgl_in)

入力と出力

  • **入力**:
  • cache: メモリ割り当てに使用するキャッシュ管理構造体。
  • sgl_in: 複製対象の util_compare_subgroup_t 構造体へのポインタ。
  • **出力**:
  • 成功時: 複製された util_compare_subgroup_t 構造体へのポインタ。
  • 失敗時: NULL。

前提条件と不変条件

  • sgl_in が NULL の場合、関数は直ちに NULL を返す。
  • 割り当てには util_ald_alloc および util_ald_strdup を使用し、共有メモリ環境下では apr_rmm を介したオフセット管理が行われる。

アルゴリズムまたは振る舞い

1. sgl_in が有効な場合、util_ald_alloc を用いて util_compare_subgroup_t 構造体本体を確保する。 2. 構造体内の subgroupDNs 配列用に、sgl_in->len 分のメモリを確保する。 3. ループ処理により、各 DN 文字列を util_ald_strdup で複製する。 4. 文字列の複製に失敗した場合、それまでに確保したすべての文字列と構造体本体を util_ald_free で解放し、NULL を返す（ロールバック処理）。 5. 全ての複製が成功した場合、len を設定して構造体を返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/ldap/util_ldap.c:1624
  • modules/ldap/util_ldap_cache.c:265
  • **呼び出し先**:
  • util_ald_alloc
  • util_ald_strdup
  • util_ald_free

関連データ構造

  • util_compare_subgroup_t: サブグループの DN リストと長さを保持する構造体。
  • util_ald_cache_t: キャッシュ管理およびメモリ割り当てのコンテキスト。

失敗モードと境界条件

  • メモリ不足（SHM 枯渇等）により util_ald_alloc または util_ald_strdup が失敗した場合、関数は部分的に確保したメモリを解放し、NULL を返す。
  • sgl_in->len が 0 の場合、配列の割り当ては行われないが、構造体本体は確保される。

保守担当者向けメモ

  • 本関数は共有メモリ環境を前提としているため、標準の malloc や strdup を使用してはならない。
  • 失敗時のロールバック処理において、ループ変数のインクリメントとデクリメントの整合性に注意すること。
