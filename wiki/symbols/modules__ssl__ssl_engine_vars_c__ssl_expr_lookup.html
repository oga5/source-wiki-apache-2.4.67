ssl_expr_lookup

定義

  • 種別: function
  • 定義箇所: modules/ssl/ssl_engine_vars.c:101

目的

Apache HTTP Server の式評価エンジン（ap_expr）に対し、SSL 関連の変数、関数、およびリスト操作を解決するためのフック関数を提供します。

シグネチャまたは形状

static int ssl_expr_lookup(ap_expr_lookup_parms *parms)

入力と出力

  • **入力**: ap_expr_lookup_parms *parms
  • parms->type: 解決対象の型（AP_EXPR_FUNC_VAR, AP_EXPR_FUNC_STRING, AP_EXPR_FUNC_LIST）。
  • parms->name: 解決対象の識別子。
  • **出力**: int
  • 成功時: OK
  • 該当なし: DECLINED
  • **副作用**: 成功時、parms->func に対応するハンドラ関数を、parms->data に解決に必要なコンテキスト情報を格納します。

前提条件と不変条件

  • ap_expr の構文解析フェーズで呼び出されることを前提としています。
  • SSL_ で始まる変数は、プレフィックスを除いた残りの文字列が parms->data として渡されます。

アルゴリズムまたは振る舞い

1. parms->type に基づき分岐します。

  • AP_EXPR_FUNC_VAR: SSL_ で始まる名前の場合、expr_var_fn を関数として設定し、プレフィックスを除いた名前をデータとして渡します。
  • AP_EXPR_FUNC_STRING: 名前が SSL の場合、expr_func_fn を関数として設定します。
  • AP_EXPR_FUNC_LIST: 名前が PeerExtList の場合、expr_peer_ext_list_fn を関数として設定します。

    • 2. いずれにも該当しない場合は DECLINED を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: ap_hook_expr_lookup を介して Apache の式評価エンジンから呼び出されます（modules/ssl/ssl_engine_vars.c:159 にて登録）。
  • **呼び出し先**:
  • expr_var_fn（変数解決用）
  • expr_func_fn（関数実行用）
  • expr_peer_ext_list_fn（リスト取得用）

関連データ構造

  • ap_expr_lookup_parms: 式評価のルックアップパラメータを保持する構造体。

失敗モードと境界条件

  • 該当する変数や関数名が見つからない場合は DECLINED を返し、他のモジュールによる解決を許可します。
  • SSL_ プレフィックスの判定は、文字列の先頭 4 文字に対して行われます。

保守担当者向けメモ

  • 本関数は APR_HOOK_LAST として登録されており、他のモジュールによる解決が優先された後に実行されます。
  • SSL_ プレフィックスの処理は最適化の余地があるとソースコード内にコメントされています。
