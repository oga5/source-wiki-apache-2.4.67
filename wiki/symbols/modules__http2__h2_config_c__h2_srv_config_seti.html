h2_srv_config_seti

定義

  • 種別: function
  • 定義箇所: modules/http2/h2_config.c:334

目的

サーバー設定構造体（h2_config）の特定のフィールドに対し、指定された列挙値に基づいて整数値を設定する。

シグネチャまたは形状

static void h2_srv_config_seti(h2_config *conf, h2_config_var_t var, int val)

入力と出力

  • conf: 更新対象の h2_config 構造体へのポインタ。
  • var: 設定対象の項目を指定する h2_config_var_t 型の列挙値。
  • val: 設定する整数値。
  • 出力: なし（副作用として conf の内容を更新）。

前提条件と不変条件

  • conf は有効な h2_config 構造体を指している必要がある。
  • H2_CONFIG_SET マクロを使用して、指定された var に対応する構造体メンバへ値を代入する。

アルゴリズムまたは振る舞い

1. var 引数に対して switch 文を実行し、対応する設定項目を特定する。 2. 一致するケースにおいて、H2_CONFIG_SET マクロを呼び出し、conf 内の該当メンバに val を書き込む。 3. H2_CONF_MAX_STREAMS から H2_CONF_MAX_STREAM_ERRORS まで、定義されたすべての設定項目を網羅的に処理する。 4. 定義されていない var が渡された場合は、何も行わず終了する。

呼び出し元と呼び出し先

  • 呼び出し元:
  • modules/http2/h2_config.c:420
  • modules/http2/h2_config.c:480
  • 呼び出し先:
  • H2_CONFIG_SET マクロ（内部的なメンバ代入処理）

関連データ構造

  • h2_config: サーバーレベルの設定を保持する構造体。
  • h2_config_var_t: 設定項目を識別するための列挙型。

失敗モードと境界条件

  • 未定義の var が渡された場合、default ケースにより無視される。
  • conf が NULL の場合、H2_CONFIG_SET マクロの実装に依存して未定義動作となる可能性がある。

保守担当者向けメモ

  • 本関数は h2_config 構造体のメンバ更新を一元管理している。新しい設定項目を追加する際は、h2_config_var_t への列挙値追加と併せて本関数の switch 文にケースを追加すること。
