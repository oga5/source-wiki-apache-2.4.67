dav_liveprop_group

定義

  • 種別: struct
  • 定義箇所: modules/dav/main/mod_dav.h:1039

目的

WebDAV ライブプロパティのグループを定義し、関連するプロパティ仕様、名前空間 URI、およびそれらを操作するためのフックをひとまとめにするために使用されます。

シグネチャまたは形状

typedef struct {
    const dav_liveprop_spec *specs;
    const char * const *namespace_uris;
    const dav_hooks_liveprop *hooks;
} dav_liveprop_group;

入力と出力

  • **入力**: 構造体の各フィールドに、プロパティ仕様の配列、名前空間 URI のリスト、および操作用フック関数テーブルを保持します。
  • **出力**: ライブプロパティの管理および検索を行うための統合されたインターフェースを提供します。

前提条件と不変条件

  • specs は、ライブプロパティの仕様を定義する配列を指す必要があります。
  • namespace_uris は、このグループが使用する XML 名前空間 URI のリストである必要があります。
  • hooks は、このグループのライブプロパティを処理するための関数ポインタテーブルを提供する必要があります。

アルゴリズムまたは振る舞い

この構造体は、特定のライブプロパティセットを登録および検索するためのコンテナとして機能します。mod_dav はこのグループを参照することで、リクエストされたプロパティがどの名前空間に属し、どのフックを使用して処理すべきかを特定します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/dav/fs/repos.c: dav_fs_liveprop_group の定義にて使用。
  • modules/dav/main/std_liveprop.c: dav_core_liveprop_group の定義にて使用。
  • **呼び出し先**:
  • modules/dav/main/liveprop.c: この構造体を受け取り、ライブプロパティの検索や管理を行う関数群で使用されます。
  • modules/dav/main/mod_dav.h: ライブプロパティ関連の関数宣言において引数として使用されます。

関連データ構造

  • dav_liveprop_spec: 個々のライブプロパティの仕様を定義します。
  • dav_hooks_liveprop: ライブプロパティ操作のためのフック関数テーブルです。

失敗モードと境界条件

  • 各フィールドに NULL が設定された場合、対応する機能（プロパティ検索やフック実行）が正しく動作しない可能性があります。
  • namespace_uris のリストが適切に終端されていない場合、名前空間の解決時にメモリ境界外アクセスが発生するリスクがあります。

保守担当者向けメモ

  • 本構造体は mod_dav のライブプロパティ拡張の基盤です。新しいライブプロパティセットを追加する場合は、この構造体をインスタンス化し、適切なフックと仕様を割り当ててください。
  • liveprop.c 内の関数がこの構造体をどのように走査しているかを確認することで、プロパティ解決のロジックを理解できます。
