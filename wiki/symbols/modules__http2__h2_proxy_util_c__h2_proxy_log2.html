h2_proxy_log2

定義

  • 種別: function
  • 定義箇所: modules/http2/h2_proxy_util.c:37

目的

与えられた符号なし整数 n に対して、その値の対数（底は2）を計算する。

シグネチャまたは形状

unsigned char h2_proxy_log2(unsigned int n)

入力と出力

  • **入力**: unsigned int n （対象となる整数）
  • **出力**: unsigned char （n の2を底とする対数）

前提条件と不変条件

  • n が 0 の場合、関数は 0 を返す。
  • ヘッダーファイル modules/http2/h2_proxy_util.h のコメントによれば、n が2の累乗である場合に正しい対数値を返すことが想定されている。

アルゴリズムまたは振る舞い

1. n が 0 の場合は即座に 0 を返す。 2. ビットシフトとマスク演算を用いて、最上位ビットの位置を特定する。 3. 16ビット、8ビット、4ビット、2ビット、1ビットの順で上位ビットを検査し、lz（leading zeros に相当するオフセット）を累積する。 4. 最終的に 31 - lz を計算して返す。これは、32ビット整数における最上位ビットのインデックスを求める処理に相当する。

呼び出し元と呼び出し先

  • **呼び出し元**: modules/http2/mod_proxy_http2.c:272 にて ctx->req_buffer_size を引数として呼び出されている。
  • **呼び出し先**: 外部関数への依存はない。

関連データ構造

  • 特になし。

失敗モードと境界条件

  • n = 0: 0 を返す。
  • n が 2 の累乗でない場合: アルゴリズムの性質上、最上位ビットの位置に基づいた値が返される。

保守担当者向けメモ

  • 本関数は h2_proxy_util.h にて宣言されている。
  • 32ビット整数を前提としたビット操作を行っているため、環境の整数サイズに依存する可能性がある。
