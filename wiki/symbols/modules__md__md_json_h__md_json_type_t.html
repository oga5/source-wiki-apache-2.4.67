md_json_type_t

定義

  • 種別: enum
  • 定義箇所: modules/md/md_json.h:31

目的

JSON データの型を識別するための列挙型です。mod_md モジュール内で JSON オブジェクトの構造を走査・検証する際に使用されます。

シグネチャまたは形状

typedef enum {
    MD_JSON_TYPE_OBJECT,
    MD_JSON_TYPE_ARRAY,
    MD_JSON_TYPE_STRING,
    MD_JSON_TYPE_REAL,
    MD_JSON_TYPE_INT,
    MD_JSON_TYPE_BOOL,
    MD_JSON_TYPE_NULL,
} md_json_type_t;

入力と出力

本シンボルは値の型定義であり、関数ではないため直接的な入出力はありません。ただし、md_json_is 関数の引数として型指定に使用されます。

前提条件と不変条件

  • jansson ライブラリの型定義と一対一で対応しています。
  • 列挙値は JSON の標準的なデータ型を網羅しています。

アルゴリズムまたは振る舞い

JSON ノードの型チェックを行う際に、対象ノードの型と本列挙値を比較することで、期待されるデータ構造であるかを判定します。

呼び出し元と呼び出し先

  • **呼び出し元**: modules/md/md_json.c:294 の md_json_is 関数にて、JSON ノードの型判定に使用されています。
  • **呼び出し先**: 本シンボルは列挙型であるため、呼び出し先はありません。

関連データ構造

  • md_json_t: JSON オブジェクトを保持する不透明な構造体。本列挙型は、この構造体内のノードの型を特定するために利用されます。

失敗モードと境界条件

  • 該当なし（型定義のため）。

保守担当者向けメモ

  • 新たな JSON 型のサポートが必要になった場合は、jansson ライブラリの仕様を確認した上で本列挙型を拡張してください。
  • md_json_is 関数以外で型判定を行うユーティリティを追加する場合も、この列挙型を基準として使用してください。
