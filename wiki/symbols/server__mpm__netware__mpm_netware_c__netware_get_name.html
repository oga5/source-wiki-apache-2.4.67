netware_get_name

定義

  • 種別: function
  • 定義箇所: server/mpm/netware/mpm_netware.c:254

目的

Apache HTTP Server の MPM（マルチプロセッシングモジュール）名として "NetWare" を返す。

シグネチャまたは形状

const char *netware_get_name(void)

入力と出力

  • 入力: なし
  • 出力: 文字列リテラル "NetWare" へのポインタを返す。

前提条件と不変条件

  • 常に静的な文字列 "NetWare" を返す。
  • 副作用はなく、呼び出しごとに同一の値を返す。

アルゴリズムまたは振る舞い

1. 関数が呼び出される。 2. 文字列 "NetWare" を即座にリターンする。

呼び出し元と呼び出し先

  • 呼び出し元: server/mpm/netware/mpm_netware.c:1101 にて、ap_hook_mpm_get_name を介して登録されている。
  • 呼び出し先: なし。

関連データ構造

  • なし。

失敗モードと境界条件

  • 失敗モード: なし。常に成功する。
  • 境界条件: なし。

保守担当者向けメモ

  • 本関数は Apache の MPM フック機構の一部であり、サーバーが現在使用している MPM の識別子を提供するために使用される。変更の必要性は極めて低い。
