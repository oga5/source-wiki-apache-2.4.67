md_duration_format

定義

  • 種別: function
  • 定義箇所: modules/md/md_time.c:154

目的

期間（apr_interval_time_t）を人間が読みやすい文字列形式に変換します。

シグネチャまたは形状

const char *md_duration_format(apr_pool_t *p, apr_interval_time_t duration)

入力と出力

  • **入力**:
  • p: メモリ確保に使用する apr_pool_t。
  • duration: 変換対象の期間（マイクロ秒単位の apr_interval_time_t）。
  • **出力**:
  • 変換された期間を表す文字列（const char *）。メモリは指定されたプールから確保されます。

前提条件と不変条件

  • p は有効な apr_pool_t である必要があります。
  • 内部で duration_format 関数を呼び出しているため、その関数が要求する前提条件に準拠します。

アルゴリズムまたは振る舞い

1. 引数として受け取った p と duration をそのまま内部関数 duration_format に渡します。 2. duration_format の戻り値をそのまま呼び出し元へ返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/md/md_acme_drive.c:874
  • modules/md/mod_md_config.c:1239
  • **呼び出し先**:
  • duration_format (同一ファイル内の静的関数)

関連データ構造

  • apr_pool_t: APR のメモリプール。
  • apr_interval_time_t: APR で定義される時間間隔（マイクロ秒単位）。

失敗モードと境界条件

  • apr_pool_t のメモリ不足が発生した場合、APR の規約に従いアボートまたはエラーが発生する可能性があります。
  • duration の値が負数や極端な値である場合の挙動は、内部の duration_format の実装に依存します。

保守担当者向けメモ

  • 本関数は duration_format への単なるラッパーです。ロジックの変更が必要な場合は、呼び出し先の duration_format を確認してください。
  • md_time.h に宣言が存在するため、外部モジュールからのインターフェースとして機能しています。
