on_frame_recv

定義

  • 種別: function
  • 定義箇所: modules/http2/h2_stream.c:162

目的

HTTP/2 ストリームにおいて、受信したフレーム種別に基づき状態遷移を決定・実行します。

シグネチャまたは形状

static int on_frame_recv(h2_stream_state_t state, int frame_type)

入力と出力

  • **入力**:
  • state: 現在のストリーム状態 (h2_stream_state_t)。
  • frame_type: 受信した HTTP/2 フレームの種別。
  • **出力**:
  • 遷移後の新しいストリーム状態を int 型で返します。

前提条件と不変条件

  • trans_on_recv 配列が、現在の状態とフレーム種別に対する有効な遷移定義を保持している必要があります。

アルゴリズムまたは振る舞い

1. trans_on_recv テーブルと、そのサイズ (H2_ALEN(trans_on_recv)) を参照します。 2. 内部ヘルパー関数 on_frame を呼び出し、現在の状態とフレーム種別に対応する遷移ロジックを実行して結果を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/http2/h2_stream.c:481
  • **呼び出し先**:
  • on_frame (同一ファイル内)
  • **注記**: nghttp2_session_callbacks_set_on_frame_recv_callback および NGH2_SET_CALLBACK を介して nghttp2 ライブラリから参照される可能性があります（h2_proxy_session.c:766, h2_session.c:725）。

関連データ構造

  • h2_stream_state_t: ストリーム状態の定義。
  • trans_on_recv: 受信フレームに対する状態遷移テーブル。

失敗モードと境界条件

  • trans_on_recv テーブル内に該当する遷移定義が存在しない場合、on_frame の挙動に依存します。

保守担当者向けメモ

  • 本関数は on_frame への単なるラッパーです。遷移ロジックの変更が必要な場合は、trans_on_recv テーブルの定義を確認してください。
