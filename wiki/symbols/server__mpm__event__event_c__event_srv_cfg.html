event_srv_cfg

定義

  • 種別: struct
  • 定義箇所: server/mpm/event/event.c:200

目的

Event MPM におけるサーバーごとの設定情報を保持するための型定義です。

シグネチャまたは形状

typedef struct event_srv_cfg_s event_srv_cfg;

※ event_srv_cfg_s 構造体のエイリアスとして定義されています。

入力と出力

  • 入力: 構成ファイル（pconf）から apr_pcalloc を介してメモリが割り当てられます。
  • 出力: サーバー設定の保持および参照に使用されます。

前提条件と不変条件

  • event_srv_cfg_s 構造体の定義に依存します。
  • サーバーの構成フェーズにおいて初期化されることが前提となります。

アルゴリズムまたは振る舞い

本シンボル自体は型定義であり、具体的なロジックは保持しません。設定値の保持コンテナとして機能します。

呼び出し元と呼び出し先

  • 呼び出し元:
  • server/mpm/event/event.c:233 (ポインタ変数としての宣言)
  • server/mpm/event/event.c:3724 (メモリ割り当て時の型指定)
  • 呼び出し先:
  • 提供されたソース情報からは、本型定義が直接呼び出す関数や、本型を呼び出す具体的な関数名は特定できません。

関連データ構造

  • event_srv_cfg_s: 本型の実体となる構造体定義。

失敗モードと境界条件

  • メモリ割り当て時（apr_pcalloc）の失敗が境界条件となります。

保守担当者向けメモ

  • 本シンボルは event_srv_cfg_s 構造体への型エイリアスです。構造体の詳細なフィールド構成を確認する場合は event_srv_cfg_s を参照してください。
