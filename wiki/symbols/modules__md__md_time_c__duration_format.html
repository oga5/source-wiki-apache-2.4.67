duration_format

定義

  • 種別: function
  • 定義箇所: modules/md/md_time.c:118

目的

apr_interval_time_t 型の期間を、人間が読みやすい形式の文字列（例: "3d", "5mi", "10s"）に変換する。

シグネチャまたは形状

const char *duration_format(apr_pool_t *p, apr_interval_time_t duration)

入力と出力

  • **入力**:
  • p: メモリ割り当てに使用する apr_pool_t。
  • duration: 変換対象の期間（apr_interval_time_t）。
  • **出力**:
  • 変換された文字列を指す const char *。メモリは p から割り当てられる。

前提条件と不変条件

  • p は有効な apr_pool_t であること。
  • 戻り値の文字列は、指定されたプール p の生存期間に依存する。

アルゴリズムまたは振る舞い

期間の長さに応じて、以下の優先順位で単位を選択し、文字列を生成する。 1. **日単位**: duration が日単位で割り切れる場合、"%dd" 形式。 2. **時間単位**: 日単位で割り切れない場合、時間単位で割り切れるなら "%dh" 形式。 3. **分単位**: 時間単位で割り切れない場合、分単位で割り切れるなら "%dmi" 形式。 4. **秒単位**: 分単位で割り切れない場合、ミリ秒単位で割り切れるなら "%ds" 形式。 5. **ミリ秒単位**: 上記いずれにも該当しない場合、"%dms" 形式。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/md/md_time.c:156 (md_duration_print からの呼び出しと推測される)
  • modules/md/md_time.c:283 (md_timeslice_format からの呼び出しと推測される)
  • **呼び出し先**:
  • apr_psprintf (文字列生成)
  • apr_time_sec / apr_time_msec (期間の分解)

関連データ構造

  • apr_interval_time_t: APR で定義される期間を表す型。
  • apr_pool_t: メモリ管理用プール。

失敗モードと境界条件

  • duration が 0 の場合、"0" という文字列を返す。
  • apr_psprintf がメモリ不足で失敗した場合の挙動は、APR の実装に依存する。

保守担当者向けメモ

  • 本関数は md_duration_print 等のラッパー関数から利用されることを想定している。
  • 単位の判定ロジックは固定されており、より細かい精度や異なる単位（週など）の追加にはロジックの変更が必要。
