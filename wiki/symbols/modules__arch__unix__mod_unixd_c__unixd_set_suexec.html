unixd_set_suexec

定義

  • 種別: function
  • 定義箇所: modules/arch/unix/mod_unixd.c:265

目的

Apache HTTP Server の設定ファイルにおける Suexec ディレクティブを処理し、suEXEC 機能の有効化または無効化を制御します。

シグネチャまたは形状

const char *unixd_set_suexec(cmd_parms *cmd, void *dummy, int arg)

入力と出力

  • **入力**:
  • cmd: 設定コマンドのパラメータ構造体。
  • dummy: 使用されません。
  • arg: Suexec ディレクティブの値（On なら 1、Off なら 0）。
  • **出力**:
  • 成功時は NULL を返します。
  • 失敗時はエラーメッセージ文字列を返します。

前提条件と不変条件

  • 本関数はグローバルコンテキスト（GLOBAL_ONLY）でのみ呼び出し可能です。
  • ap_unixd_config 構造体の suexec_enabled および suexec_disabled_reason フィールドを更新します。

アルゴリズムまたは振る舞い

1. ap_check_cmd_context を呼び出し、設定がグローバルスコープで行われているか検証します。 2. arg が有効（On）かつ、システム側で suEXEC がサポートされていない場合、ap_unixd_config.suexec_disabled_reason に格納された理由と共にエラーを返します。 3. arg が無効（Off）の場合、ap_unixd_config.suexec_disabled_reason を "Suexec directive is Off" に設定します。 4. ap_unixd_config.suexec_enabled に arg の値を代入し、設定を反映します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/arch/unix/mod_unixd.c:419 にて AP_INIT_FLAG マクロを通じて設定ディレクティブとして登録されています。
  • **呼び出し先**:
  • ap_check_cmd_context (Apache 内部 API)
  • apr_pstrcat (APR API)

関連データ構造

  • cmd_parms: 設定コマンドの引数とコンテキストを保持する構造体。
  • ap_unixd_config: mod_unixd の設定を保持するグローバル構造体（unixd.h 定義）。

失敗モードと境界条件

  • **コンテキストエラー**: グローバルスコープ以外で Suexec ディレクティブが使用された場合、ap_check_cmd_context がエラーを返します。
  • **機能非対応**: システム環境で suEXEC がサポートされていない状態で Suexec On を指定した場合、初期化時のチェック結果に基づきエラーを返します。

保守担当者向けメモ

  • 本関数は Suexec ディレクティブのフラグ設定のみを担当します。実際の suEXEC バイナリの検証は unixd_pre_config で行われます。
  • ap_unixd_config はモジュール全体で共有されるため、設定変更はサーバー起動時の構成フェーズで行われる必要があります。
