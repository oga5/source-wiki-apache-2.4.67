log_debug

定義

  • 種別: function
  • 定義箇所: test/clients/h2ws.c:88

目的

デバッグレベルのメッセージを標準出力へ条件付きで出力する。

シグネチャまたは形状

static void log_debug(const char *where, const char *msg)

入力と出力

  • where: ログの発生箇所を示す文字列。
  • msg: 出力するデバッグメッセージの内容。
  • 出力: 戻り値なし。条件を満たす場合、log_out を介して標準出力へ書き出す。

前提条件と不変条件

  • グローバル変数 verbose が定義されている必要がある。
  • log_out 関数が利用可能であること。

アルゴリズムまたは振る舞い

1. グローバル変数 verbose の値を確認する。 2. verbose > 1 の場合のみ、log_out 関数を呼び出し、タグとして "DEBUG" を付与して where と msg を出力する。 3. verbose <= 1 の場合は何も行わない。

呼び出し元と呼び出し先

  • 呼び出し元: test/clients/h2ws.c:136 にて呼び出しを確認。
  • 呼び出し先: log_out 関数を呼び出す。

関連データ構造

  • なし。

失敗モードと境界条件

  • where または msg が NULL の場合、log_out の実装に依存する（本関数内での明示的なチェックはない）。

保守担当者向けメモ

  • 本関数は verbose レベルによるフィルタリングを行うラッパーである。詳細なログ出力が必要な場合は、実行時に verbose の値を調整すること。
