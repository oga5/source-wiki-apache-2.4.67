set_default

定義

  • 種別: function
  • 定義箇所: modules/filters/mod_xml2enc.c:614

目的

Apache HTTP Server の設定ディレクティブ xml2EncDefault を処理し、モジュールのデフォルト文字エンコーディングを設定する。

シグネチャまたは形状

static const char* set_default(cmd_parms* cmd, void* CFG, const char* charset)

入力と出力

  • **入力**:
  • cmd: コマンドパラメータ構造体。
  • CFG: ディレクトリ設定構造体 (xml2cfg*)。
  • charset: 設定する文字エンコーディング名（文字列）。
  • **出力**:
  • 成功時: NULL。
  • 失敗時: エラーメッセージ（文字列）。

前提条件と不変条件

  • CFG は xml2cfg 型のポインタとしてキャスト可能であること。
  • charset は libxml2 の xmlParseCharEncoding 関数で解析可能な形式であること。

アルゴリズムまたは振る舞い

1. CFG を xml2cfg 構造体としてキャストし、default_charset に引数の charset を格納する。 2. xmlParseCharEncoding(charset) を呼び出し、内部的なエンコーディング識別子を default_encoding に格納する。 3. 識別子の値を判定する:

  • XML_CHAR_ENCODING_NONE: "Default charset not found" を返す。
  • XML_CHAR_ENCODING_ERROR: "Invalid or unsupported default charset" を返す。
  • その他: 正常終了として NULL を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: modules/filters/mod_xml2enc.c:640 において、AP_INIT_TAKE1 マクロ経由で xml2EncDefault ディレクティブのハンドラとして登録されている。
  • **呼び出し先**: xmlParseCharEncoding (libxml2 ライブラリ関数)。

関連データ構造

  • xml2cfg: モジュールのディレクトリごとの設定を保持する構造体。

失敗モードと境界条件

  • 指定された charset が libxml2 でサポートされていない場合、エラーメッセージを返し設定は適用されない。

保守担当者向けメモ

  • 本関数は xml2EncDefault ディレクティブのコールバックとしてのみ使用される。
  • libxml2 のエンコーディングサポート状況に依存するため、ライブラリのバージョンアップ時に挙動が変化する可能性がある。
