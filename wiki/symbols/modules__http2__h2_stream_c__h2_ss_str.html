h2_ss_str

定義

  • 種別: function
  • 定義箇所: modules/http2/h2_stream.c:52

目的

HTTP/2 ストリームの状態（h2_stream_state_t）を、デバッグログや表示に適した人間が読み取り可能な文字列に変換する。

シグネチャまたは形状

const char *h2_ss_str(const h2_stream_state_t state)

入力と出力

  • **入力**: state (h2_stream_state_t 型) - 変換対象のストリーム状態。
  • **出力**: const char * - 状態に対応する文字列リテラル。

前提条件と不変条件

  • 入力された state が定義済みの h2_stream_state_t 列挙値のいずれかであること。
  • 戻り値は静的な文字列リテラルを指すため、呼び出し側で解放してはならない。

アルゴリズムまたは振る舞い

switch 文を用いて h2_stream_state_t の各列挙値を評価し、対応する文字列を返す。定義されていない値が渡された場合は "UNKNOWN" を返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/http2/h2_stream.c:78
  • modules/http2/h2_stream.c:357
  • **呼び出し先**: なし（標準的な制御フローのみ）。

関連データ構造

  • h2_stream_state_t: ストリームの状態を定義する列挙型。

失敗モードと境界条件

  • 未定義の state 値が渡された場合、デフォルトケースとして "UNKNOWN" を返す安全な設計となっている。

保守担当者向けメモ

  • 新しいストリーム状態が h2_stream_state_t に追加された場合は、本関数の switch 文にも必ずケースを追加すること。
