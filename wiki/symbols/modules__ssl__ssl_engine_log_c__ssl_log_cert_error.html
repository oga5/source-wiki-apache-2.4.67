ssl_log_cert_error

定義

  • 種別: function
  • 定義箇所: modules/ssl/ssl_engine_log.c:126

目的

X.509 証明書に関連するエラー発生時に、証明書の詳細情報（サブジェクト、発行者、シリアル番号、有効期限）を整形してログメッセージに付与し、Apache のログシステムへ出力する。

シグネチャまたは形状

static void ssl_log_cert_error(const char *file, int line, int level,
                               apr_status_t rv, const server_rec *s,
                               const conn_rec *c, const request_rec *r,
                               apr_pool_t *p, X509 *cert, const char *format,
                               va_list ap)

入力と出力

  • **入力**: ログ出力位置（file, line）、ログレベル、ステータスコード、サーバー/接続/リクエストの各コンテキスト、メモリプール、対象の X509 証明書オブジェクト、フォーマット文字列、および可変引数リスト。
  • **出力**: なし（戻り値は void）。副作用として Apache のログシステムへメッセージを出力する。

前提条件と不変条件

  • cert が NULL の場合、証明書情報は「-not available-」として出力される。
  • 証明書情報の文字列化には、呼び出し元から提供された apr_pool_t *p を使用する。
  • ログ行の最大長は HUGE_STRING_LEN に制限されており、サブジェクトおよび発行者の DN 文字列長は、メッセージ全体がこの制限を超えないよう計算される。

アルゴリズムまたは振る舞い

1. apr_vsnprintf を使用して、提供されたフォーマットと引数から基本メッセージを生成する。 2. cert が存在する場合、BIO_s_mem() を用いてメモリ上に一時バッファを作成する。 3. modssl_X509_NAME_to_string を呼び出し、サブジェクトおよび発行者名をバッファに書き込む。 4. i2a_ASN1_INTEGER および ASN1_TIME_print を使用して、シリアル番号、有効期限（notBefore/notAfter）をバッファに追加する。 5. 生成された証明書情報を基本メッセージと結合する。 6. r（リクエスト）、c（接続）、s（サーバー）の順に優先度を判定し、対応する ap_log_*error 関数を呼び出してログを出力する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/ssl/ssl_engine_log.c:218 (ssl_log_xerror 経由)
  • modules/ssl/ssl_engine_log.c:230 (ssl_log_cxerror 経由)
  • modules/ssl/ssl_engine_log.c:242 (ssl_log_rxerror 経由)
  • **呼び出し先**:
  • modssl_X509_NAME_to_string
  • i2a_ASN1_INTEGER
  • ASN1_TIME_print
  • ap_log_rerror / ap_log_cerror / ap_log_error

関連データ構造

  • X509: OpenSSL の証明書構造体。
  • BIO: OpenSSL の入出力抽象化レイヤー。
  • server_rec, conn_rec, request_rec: Apache のサーバー/接続/リクエスト管理構造体。

失敗モードと境界条件

  • BIO_new が失敗した場合、証明書情報の付与はスキップされる。
  • modssl_X509_NAME_to_string が空文字列を返した場合、「-empty-」という文字列がログに記録される。
  • i2a_ASN1_INTEGER が失敗した場合、「(ERROR)」という文字列が記録される。

保守担当者向けメモ

  • 本関数は BIO を使用してメモリを確保するため、関数終了時に BIO_free で適切に解放している。
  • ログ行の最大長計算において、HUGE_STRING_LEN を基準とした安全なバッファ操作が行われている。
