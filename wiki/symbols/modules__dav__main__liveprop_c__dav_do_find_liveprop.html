dav_do_find_liveprop

定義

  • 種別: function
  • 定義箇所: modules/dav/main/liveprop.c:83

目的

指定された名前空間 URI とプロパティ名に基づき、dav_liveprop_group 内から該当するライブプロパティを検索し、そのプロパティ ID と処理用フックを特定する。

シグネチャまたは形状

int dav_do_find_liveprop(const char *ns_uri, const char *name,
                         const dav_liveprop_group *group,
                         const dav_hooks_liveprop **hooks)

入力と出力

  • **入力**:
  • ns_uri: 検索対象のプロパティ名前空間 URI。
  • name: 検索対象のプロパティ名。
  • group: 検索対象となるライブプロパティグループの定義。
  • hooks: 該当プロパティの処理用フックを格納するためのポインタのポインタ。
  • **出力**:
  • 戻り値: 該当プロパティが見つかった場合はその propid (正の整数)、見つからない場合は 0。
  • hooks: 該当プロパティが見つかった場合、group->hooks が設定される。

前提条件と不変条件

  • group 引数は有効な dav_liveprop_group 構造体を指している必要がある。
  • group->namespace_uris は NULL で終端されている必要がある。
  • group->specs は name フィールドが NULL である要素で終端されている必要がある。

アルゴリズムまたは振る舞い

1. **名前空間の特定**: group->namespace_uris 配列を線形走査し、ns_uri と一致するインデックス ns を特定する。一致しない場合は 0 を返す。 2. **プロパティの検索**: group->specs 配列を走査し、以下の条件を満たすエントリを探す。

  • scan->ns が手順1で特定したインデックス ns と一致する。
  • scan->name が引数の name と一致する。

    • 3. **結果の返却**: 一致するエントリが見つかった場合、*hooks に group->hooks を代入し、scan->propid を返す。一致するものがない場合は 0 を返す。

呼び出し元と呼び出し先

  • **呼び出し先**:

関連データ構造

  • dav_liveprop_group: ライブプロパティの集合とフックを保持する構造体。
  • dav_liveprop_spec: 個別のライブプロパティの仕様（名前、名前空間インデックス、ID）を定義する構造体。
  • dav_hooks_liveprop: ライブプロパティの操作フック群。

失敗モードと境界条件

  • 指定された名前空間が group 内に存在しない場合、0 を返す。
  • 名前空間は一致するが、指定されたプロパティ名が group 内に存在しない場合、0 を返す。
  • group が空（namespace_uris が即座に NULL）の場合、常に 0 を返す。

保守担当者向けメモ

  • 線形探索を行っているため、プロパティ数が多いグループではパフォーマンスに影響する可能性がある。
  • ns インデックスは group->namespace_uris 配列内での位置に依存するため、登録順序の変更には注意が必要である。
