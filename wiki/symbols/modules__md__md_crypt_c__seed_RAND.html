seed_RAND

定義

  • 種別: function
  • 定義箇所: modules/md/md_crypt.c:85

目的

OpenSSL の乱数生成器（PRNG）を初期化するためのシードを供給する。

シグネチャまたは形状

static void seed_RAND(int pid)

入力と出力

  • **入力**: pid (int型。現在のプロセスID。関数内で (void)pid として明示的に無視される)
  • **出力**: なし（副作用として OpenSSL の内部状態を更新）

前提条件と不変条件

  • arc4random_buf が利用可能な環境であること。
  • OpenSSL の RAND_seed が呼び出し可能であること。

アルゴリズムまたは振る舞い

1. 128バイトのバッファ seed を確保する。 2. arc4random_buf を使用して、システムから取得したランダムなデータでバッファを埋める。 3. RAND_seed を呼び出し、生成したデータを OpenSSL の乱数生成器にシードとして投入する。

呼び出し元と呼び出し先

  • **呼び出し元**: modules/md/md_crypt.c 内の md_crypt_init から呼び出される。
  • **呼び出し先**: arc4random_buf (システムライブラリ), RAND_seed (OpenSSL)

関連データ構造

  • char seed[128]: 乱数シードを保持するための一時バッファ。

失敗モードと境界条件

  • arc4random_buf が失敗した場合の挙動については、本関数内では明示的なエラーハンドリングを行っていない。

保守担当者向けメモ

  • 引数 pid は現在使用されていないが、将来的な拡張やプラットフォーム固有のシード生成ロジックの変更に備えてシグネチャに含まれている可能性がある。
  • 本関数は static であり、md_crypt.c 内でのみ使用される。
