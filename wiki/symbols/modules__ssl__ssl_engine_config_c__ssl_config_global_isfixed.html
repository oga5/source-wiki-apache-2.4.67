ssl_config_global_isfixed

定義

  • 種別: function
  • 定義箇所: modules/ssl/ssl_engine_config.c:101

目的

グローバルな SSL 設定構造体（SSLModConfigRec）が固定（fixed）状態にあるかどうかを判定する。

シグネチャまたは形状

BOOL ssl_config_global_isfixed(SSLModConfigRec *mc)

入力と出力

  • **入力**: mc (SSLModConfigRec *) - 判定対象のグローバル設定構造体へのポインタ。
  • **出力**: BOOL - 設定が固定されていれば真（非ゼロ）、そうでなければ偽（ゼロ）を返す。

前提条件と不変条件

  • 引数 mc は有効な SSLModConfigRec 構造体を指している必要がある。

アルゴリズムまたは振る舞い

  • 構造体 mc 内のメンバ変数 bFixed の値をそのまま返す。

呼び出し元と呼び出し先

  • **呼び出し元**: modules/ssl/ssl_engine_config.c:656 にて、設定の変更可否をチェックするために呼び出されている。
  • **呼び出し先**: なし。

関連データ構造

  • SSLModConfigRec: mod_ssl のグローバル設定を保持する構造体（ssl_private.h で定義）。

失敗モードと境界条件

  • mc が NULL の場合、プログラムはクラッシュする可能性がある（ソースコード上にガード句は存在しない）。

保守担当者向けメモ

  • 本関数は ssl_private.h:901 にて宣言されている。
  • bFixed フラグがいつ、どのような条件で設定されるかについては、本関数単体では定義されていないため、ssl_config_global_fix 等の関連関数を参照すること。
