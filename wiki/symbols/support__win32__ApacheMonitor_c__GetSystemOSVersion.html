GetSystemOSVersion

定義

  • 種別: function
  • 定義箇所: support/win32/ApacheMonitor.c:286

目的

実行中の Windows OS のバージョンを判定し、アプリケーションがサポートする特定の OS バージョン（Windows 2000 またはそれ以前の Windows NT）に分類して呼び出し元へ通知する。

シグネチャまたは形状

BOOL GetSystemOSVersion(LPDWORD dwVersion)

入力と出力

  • **入力**: dwVersion (OS バージョン識別子を格納するための DWORD 型へのポインタ)
  • **出力**:
  • 成功時: TRUE を返し、dwVersion に OS_VERSION_WIN2K または OS_VERSION_WINNT を書き込む。
  • 失敗時: FALSE を返し、dwVersion は変更されないか、あるいは 0 が設定される。

前提条件と不変条件

  • GetVersionEx API がシステム上で利用可能であること。
  • dwVersion は有効なメモリ領域を指している必要がある。

アルゴリズムまたは振る舞い

1. OSVERSIONINFO 構造体をゼロ初期化し、サイズを設定する。 2. GetVersionEx を呼び出し、OS 情報を取得する。失敗した場合は FALSE を返す。 3. dwPlatformId を評価する:

  • VER_PLATFORM_WIN32_NT の場合: dwMajorVersion が 5 以上であれば OS_VERSION_WIN2K を、それ未満であれば OS_VERSION_WINNT を dwVersion に代入する。
  • その他のプラットフォーム（Windows 9x 系など）の場合: dwVersion に 0 を代入し、FALSE を返す。

    • 4. 判定が完了すれば TRUE を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: support/win32/ApacheMonitor.c:1556 (WinMain 内で初期化時に呼び出される)
  • **呼び出し先**: GetVersionEx (Windows API)

関連データ構造

  • OSVERSIONINFO (Windows SDK 定義の構造体)

失敗モードと境界条件

  • GetVersionEx が失敗した場合、関数は直ちに FALSE を返す。
  • Windows NT 系以外のプラットフォーム（VER_PLATFORM_WIN32_WINDOWS 等）が検出された場合、サポート対象外として FALSE を返す。

保守担当者向けメモ

  • 本関数は Windows 2000 以前の古い OS 判定ロジックに基づいている。現代の Windows OS 環境では dwMajorVersion の判定ロジックが意図通りに動作しない可能性があるため、OS バージョンチェックの要件を再確認すること。
