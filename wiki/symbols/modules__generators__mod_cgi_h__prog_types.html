prog_types

定義

  • 種別: enum
  • 定義箇所: modules/generators/mod_cgi.h:31

目的

CGI スクリプトの実行モードを識別するために使用されます。具体的には、リクエストが Server Side Includes (SSI) 経由であるか、直接の CGI 実行であるかを区別します。

シグネチャまたは形状

typedef enum {
    RUN_AS_SSI,
    RUN_AS_CGI
} prog_types;

入力と出力

  • 入力: なし（列挙型定義）
  • 出力: RUN_AS_SSI または RUN_AS_CGI のいずれかの値。

前提条件と不変条件

  • cgi_exec_info_t 構造体の一部として使用されることが想定されています。

アルゴリズムまたは振る舞い

  • RUN_AS_SSI: SSI コンテキスト下での実行であることを示します。
  • RUN_AS_CGI: 標準的な CGI 実行であることを示します。

呼び出し元と呼び出し先

  • 呼び出し元: modules/generators/mod_cgi.h:40 において、cgi_exec_info_t 構造体のメンバ prog_type の型として参照されています。
  • 呼び出し先: 情報なし。

関連データ構造

  • cgi_exec_info_t: 本列挙型をメンバとして保持する構造体。

失敗モードと境界条件

  • 列挙型であるため、定義された値以外の値が代入された場合の挙動は未定義です。

保守担当者向けメモ

  • 本シンボルは CGI 実行の準備フェーズにおけるモード判定に使用されます。
  • mod_cgi および mod_cgid の両モジュールで共通して利用されるため、変更時は影響範囲に注意してください。
