ef_init

定義

  • 種別: function
  • 定義箇所: modules/filters/mod_ext_filter.c:351

目的

モジュールの初期化処理として、メインサーバーの情報を静的変数に保持する。

シグネチャまたは形状

int ef_init(apr_pool_t *p, apr_pool_t *plog, apr_pool_t *ptemp, server_rec *main_s)

入力と出力

  • 入力:
  • p: 設定用プール
  • plog: ログ用プール
  • ptemp: 一時プール
  • main_s: メインサーバーの構成情報 (server_rec)
  • 出力:
  • 常に OK を返す。

前提条件と不変条件

  • Apache の設定読み込みフェーズで呼び出される。
  • main_server グローバル変数に main_s が格納される。

アルゴリズムまたは振る舞い

1. 引数として渡された main_s を、モジュール内のグローバル変数 main_server に代入する。 2. OK を返して終了する。

呼び出し元と呼び出し先

  • 呼び出し元: ap_hook_post_config により、Apache の設定完了後のフックとして登録されている。
  • 呼び出し先: なし。

関連データ構造

  • server_rec: Apache のサーバー構成情報を保持する構造体。

失敗モードと境界条件

  • 特になし。常に成功する設計となっている。

保守担当者向けメモ

  • 本関数は post_config フックで実行されるため、サーバー起動時の初期化にのみ関与する。
  • main_server 変数は、このモジュール内で外部プロセス管理や設定参照を行う際に利用される。
