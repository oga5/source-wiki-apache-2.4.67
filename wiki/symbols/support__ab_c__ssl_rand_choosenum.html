ssl_rand_choosenum

定義

  • 種別: function
  • 定義箇所: support/ab.c:617

目的

指定された範囲 [l, h] 内の擬似乱数を生成し、返却する。

シグネチャまたは形状

static int ssl_rand_choosenum(int l, int h)

入力と出力

  • **入力**:
  • l (int): 乱数範囲の下限値。
  • h (int): 乱数範囲の上限値。
  • **出力**:
  • int: l 以上 h 以下の範囲に収まる整数値。

前提条件と不変条件

  • srand() を time(NULL) で初期化するため、呼び出し間隔が非常に短い場合、同一の乱数列が生成される可能性がある。
  • 戻り値は常に l <= return_value <= h を満たすよう、計算後に境界チェックが行われる。

アルゴリズムまたは振る舞い

1. srand((unsigned int)time(NULL)) を呼び出し、乱数生成器を初期化する。 2. rand() を用いて 0 から 1 の範囲の浮動小数点数を生成し、範囲幅 (h - l) を乗算する。 3. 結果を文字列バッファに %.0f で書き込み、atoi() で整数に変換する。 4. 変換値に 1 を加算する。 5. 最終的な値が l 未満であれば l に、h を超えていれば h に補正して返却する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • support/ab.c:655
  • modules/ssl/ssl_engine_rand.c:118 (ツールによる推論)
  • **呼び出し先**:
  • srand, time, apr_snprintf, rand, atoi (標準ライブラリおよび APR)

関連データ構造

  • なし

失敗モードと境界条件

  • l > h の場合、補正ロジックにより h が返される可能性がある。
  • time(NULL) の精度に依存するため、秒単位以下の高速な連続呼び出しでは同じ値が返される特性がある。

保守担当者向けメモ

  • 本関数は support/ab.c 内で定義されているが、modules/ssl/ssl_engine_rand.c からも参照されている。修正時は両方のコンテキストに影響がないか確認すること。
  • rand() および srand() はスレッドセーフではない可能性があるため、マルチスレッド環境での利用には注意が必要である。
