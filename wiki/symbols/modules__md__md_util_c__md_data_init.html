md_data_init

定義

  • 種別: function
  • 定義箇所: modules/md/md_util.c:95

目的

md_data_t 構造体を初期化し、指定されたデータポインタと長さを設定します。

シグネチャまたは形状

void md_data_init(md_data_t *d, const char *data, apr_size_t len)

入力と出力

  • **入力**:
  • d: 初期化対象となる md_data_t 構造体へのポインタ。
  • data: 格納するデータへのポインタ。
  • len: 格納するデータの長さ。
  • **出力**: なし（戻り値なし）。

前提条件と不変条件

  • d は有効なメモリ領域を指している必要があります。
  • 内部で md_data_null(d) が呼び出されるため、既存のデータポインタや状態はリセットされます。

アルゴリズムまたは振る舞い

1. md_data_null(d) を呼び出し、構造体を初期状態（NULLポインタおよび長さ0）にリセットします。 2. 引数 len を d->len に代入します。 3. 引数 data を d->data に代入します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/md/md_curl.c (複数箇所)
  • modules/md/md_jws.c (複数箇所)
  • modules/md/md_util.c:104
  • **呼び出し先**:
  • md_data_null

関連データ構造

  • md_data_t: データポインタと長さを保持する構造体。

失敗モードと境界条件

  • d が NULL の場合、プログラムは異常終了（セグメンテーションフォールト）する可能性があります。
  • data が指すメモリ領域の有効性については、呼び出し元で保証する必要があります。

保守担当者向けメモ

  • 本関数はデータの所有権をコピーするのではなく、ポインタの代入のみを行います。データのライフサイクル管理が必要な場合は、md_data_assign_pcopy 等の他のユーティリティとの使い分けに注意してください。
