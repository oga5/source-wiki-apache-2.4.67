ssl_rssrc_t

定義

  • 種別: enum
  • 定義箇所: modules/ssl/ssl_private.h:546

目的

SSL/TLS における乱数生成源（Random Seed Source）の種類を識別するために使用されます。

シグネチャまたは形状

typedef enum {
    SSL_RSSRC_BUILTIN = 1,
    SSL_RSSRC_FILE    = 2,
    SSL_RSSRC_EXEC    = 3,
    SSL_RSSRC_EGD     = 4
} ssl_rssrc_t;

入力と出力

  • 入力: 該当なし（列挙型定義）
  • 出力: 乱数生成源を特定する整数値（1〜4）

前提条件と不変条件

  • 各列挙値は、mod_ssl がサポートする特定の乱数生成メカニズムに対応しています。
  • 値は 1 から 4 の範囲で定義されており、これ以外の値は想定されていません。

アルゴリズムまたは振る舞い

  • SSL_RSSRC_BUILTIN: OpenSSL 組み込みの乱数生成機能を使用します。
  • SSL_RSSRC_FILE: 指定されたファイルから乱数シードを読み込みます。
  • SSL_RSSRC_EXEC: 外部コマンドを実行し、その出力を乱数シードとして使用します。
  • SSL_RSSRC_EGD: Entropy Gathering Daemon (EGD) を介して乱数を取得します。

呼び出し元と呼び出し先

  • 呼び出し元: modules/ssl/ssl_private.h:554 にて ssl_rssrc_t nSrc; として構造体メンバの型定義に使用されています。
  • 呼び出し先: 提供されたソース情報からは、具体的な関数呼び出し関係は特定できません。

関連データ構造

  • ssl_randseed_t: 乱数生成の設定を保持する構造体（本列挙型をメンバとして含む可能性が高い）。

失敗モードと境界条件

  • 該当なし（列挙型定義のため）。

保守担当者向けメモ

  • 本列挙型は、ssl_engine_rand.c 等の乱数生成ロジックにおいて、どのソースからシードを取得するかを決定する際のフラグとして利用されます。
  • 新しい乱数生成手法を追加する場合は、本列挙型に値を追加し、対応する処理を ssl_engine_rand.c に実装する必要があります。
