md_http_set_connect_timeout_default

定義

  • 種別: function
  • 定義箇所: modules/md/md_http.c:139

目的

HTTP セッションにおけるデフォルトの接続タイムアウト値を設定します。

シグネチャまたは形状

void md_http_set_connect_timeout_default(md_http_t *http, apr_time_t timeout)

入力と出力

  • http: 設定対象となる md_http_t セッションへのポインタ。
  • timeout: 設定する接続タイムアウト値（apr_time_t 型）。
  • 戻り値: なし。

前提条件と不変条件

  • http ポインタは有効な md_http_t インスタンスを指している必要があります。

アルゴリズムまたは振る舞い

引数として渡された timeout 値を、md_http_t 構造体内の timeout.connect メンバに代入します。

呼び出し元と呼び出し先

  • 呼び出し元:
  • modules/md/md_acme.c:800
  • modules/md/md_ocsp.c:921
  • 呼び出し先:
  • ソースコード上での直接的な関数呼び出しはありません。

関連データ構造

  • md_http_t: HTTP クライアントのコンテキストを保持する構造体。
  • apr_time_t: APR で定義された時間表現型。

失敗モードと境界条件

  • http が NULL の場合、セグメンテーション違反が発生する可能性があります（ソースコード内に明示的なガード条件はありません）。

保守担当者向けメモ

  • 本関数は md_http_t の内部状態を直接更新します。設定値は後続の HTTP リクエスト実行時に使用されます。
