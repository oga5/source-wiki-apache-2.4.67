stapling_set_response

定義

  • 種別: function
  • 定義箇所: modules/ssl/ssl_util_stapling.c:374

目的

OCSP レスポンスを DER 形式に変換し、現在の SSL 接続の TLS 拡張（Certificate Status Request）として設定します。

シグネチャまたは形状

int stapling_set_response(SSL *ssl, OCSP_RESPONSE *rsp)

入力と出力

  • **入力**:
  • ssl: 現在の SSL 接続を指す SSL オブジェクト。
  • rsp: 設定対象の OCSP_RESPONSE オブジェクト。
  • **出力**:
  • 成功時は 1 を返し、失敗時は 0 を返します。

前提条件と不変条件

  • rsp は有効な OCSP_RESPONSE オブジェクトである必要があります。
  • i2d_OCSP_RESPONSE による DER 変換が成功する必要があります。

アルゴリズムまたは振る舞い

1. i2d_OCSP_RESPONSE を使用して、OCSP_RESPONSE オブジェクトを DER 形式のバイト列にシリアライズします。 2. 変換に失敗（rspderlen <= 0）した場合は、直ちに 0 を返して終了します。 3. 変換成功時、SSL_set_tlsext_status_ocsp_resp を呼び出し、生成された DER バイト列を SSL 接続に関連付けます。 4. 正常終了時に 1 を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: modules/ssl/ssl_util_stapling.c:912 にて呼び出されています。
  • **呼び出し先**:
  • i2d_OCSP_RESPONSE (OpenSSL API)
  • SSL_set_tlsext_status_ocsp_resp (OpenSSL API)

関連データ構造

  • SSL: OpenSSL の接続状態を保持する構造体。
  • OCSP_RESPONSE: OCSP レスポンスを保持する OpenSSL の ASN.1 構造体。

失敗モードと境界条件

  • i2d_OCSP_RESPONSE が DER 変換に失敗した場合、関数は 0 を返し、SSL 接続へのレスポンス設定は行われません。

保守担当者向けメモ

  • 本関数は OpenSSL の SSL_set_tlsext_status_ocsp_resp をラップしており、メモリ管理は OpenSSL 側の仕様に依存します。
  • rspder バッファの解放については、OpenSSL の API 仕様に従い、適切にハンドリングされていることを確認してください。
