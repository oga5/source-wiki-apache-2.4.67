md_pkey_ec_params_t

定義

  • **種別**: struct
  • **定義箇所**: modules/md/md_crypt.h:59

目的

楕円曲線暗号（EC）鍵を生成する際のパラメータを保持するための構造体です。

シグネチャまたは形状

typedef struct md_pkey_ec_params_t {
    const char *curve;
} md_pkey_ec_params_t;

入力と出力

  • **入力**: curve メンバに、使用する楕円曲線の名称（例: "prime256v1" 等）を文字列として格納します。
  • **出力**: なし（本構造体自体はデータ保持用です）。

前提条件と不変条件

  • curve は、OpenSSL がサポートする有効な楕円曲線名を示す文字列である必要があります。

アルゴリズムまたは振る舞い

  • 本構造体は、md_pkey_spec_t 構造体の一部として、鍵生成の仕様を定義するために使用されます。
  • 実際の鍵生成処理において、このパラメータが OpenSSL の鍵生成関数へ渡されることで、指定された曲線に基づいた秘密鍵が生成されます。

呼び出し元と呼び出し先

  • **呼び出し元**: modules/md/md_crypt.h:67 にて、md_pkey_spec_t 構造体のメンバ ec として参照されています。
  • **呼び出し先**: 提供されたソースコードの範囲内では、本構造体を利用する具体的な関数呼び出しの詳細は確認できません。

関連データ構造

  • md_pkey_spec_t: 鍵生成の仕様全体を定義する構造体であり、本構造体をメンバとして含みます。

失敗モードと境界条件

  • curve に無効な文字列や OpenSSL が認識できない曲線名が設定された場合、後続の鍵生成処理（md_pkey_gen 等）が失敗する可能性があります。

保守担当者向けメモ

  • 本構造体は md_crypt.h で定義されており、mod_md における暗号化操作の抽象化レイヤーの一部です。
  • 構造体内の curve は const char * であるため、文字列の寿命管理には注意してください。
