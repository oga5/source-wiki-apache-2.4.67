h2_proxy_ihash_count

定義

  • 種別: function
  • 定義箇所: modules/http2/h2_proxy_util.c:87

目的

整数識別子をキーとして管理するハッシュテーブル（h2_proxy_ihash_t）に格納されている要素の総数を取得する。

シグネチャまたは形状

size_t h2_proxy_ihash_count(h2_proxy_ihash_t *ih)

入力と出力

  • **入力**: ih - 対象となる h2_proxy_ihash_t 構造体へのポインタ。
  • **出力**: ハッシュテーブル内に現在格納されている要素数（size_t）。

前提条件と不変条件

  • ih は有効な h2_proxy_ihash_t 構造体を指している必要がある。
  • 内部の apr_hash_t は初期化済みであること。

アルゴリズムまたは振る舞い

内部的に保持している apr_hash_t に対して apr_hash_count() を呼び出し、その結果をそのまま返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/http2/h2_proxy_session.c:195
  • modules/http2/h2_proxy_session.c:1694
  • modules/http2/h2_proxy_session.c:1719
  • **呼び出し先**:
  • apr_hash_count (APR ライブラリ関数)

関連データ構造

  • h2_proxy_ihash_t: 整数 ID をキーとするハッシュテーブルをラップした構造体。

失敗モードと境界条件

  • ih が NULL の場合、内部の apr_hash_count 呼び出しによりセグメンテーション違反が発生する可能性がある。

保守担当者向けメモ

  • 本関数は apr_hash_count の単純なラッパーである。
  • h2_proxy_ihash_t の内部構造（hash メンバー）が変更されない限り、この実装は不変である。
