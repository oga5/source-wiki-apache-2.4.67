looks_like_an_argument

定義

  • 種別: function
  • 定義箇所: modules/core/mod_macro.c:222

目的

文字列がマクロの引数として有効なプレフィックス（$, %, @）で開始されているかを判定する。

シグネチャまたは形状

static int looks_like_an_argument(const char *word)

入力と出力

  • **入力**: word (判定対象の文字列へのポインタ)
  • **出力**: int 型。引数プレフィックスで始まる場合は非ゼロ（真）、それ以外は 0（偽）を返す。

前提条件と不変条件

  • word は有効なヌル終端文字列であること。
  • ARG_PREFIX マクロが定義されており、比較対象の文字セットとして機能していること。

アルゴリズムまたは振る舞い

1. word の先頭文字（*word）を取得する。 2. ap_strchr を使用し、その文字が ARG_PREFIX に含まれるかを確認する。 3. 含まれていれば真、含まれていなければ偽を返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/core/mod_macro.c:249: 引数リストの検証時に使用。
  • modules/core/mod_macro.c:760: マクロ使用時の引数名チェックに使用。
  • **呼び出し先**:
  • ap_strchr (Apache Portable Runtime / Core API)

関連データ構造

  • なし

失敗モードと境界条件

  • word が空文字列（""）の場合、*word はヌル文字 \0 となり、ap_strchr の結果は 0 となるため、正しく偽として処理される。

保守担当者向けメモ

  • 本関数はマクロ引数の構文解析における基本的な述語として機能する。
  • ARG_PREFIX の定義が変更された場合、この関数の判定ロジックも自動的に追従する。
