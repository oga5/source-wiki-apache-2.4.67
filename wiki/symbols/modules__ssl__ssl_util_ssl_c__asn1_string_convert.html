asn1_string_convert

定義

  • 種別: function
  • 定義箇所: modules/ssl/ssl_util_ssl.c:205

目的

OpenSSL の ASN1_STRING 構造体を、Apache のメモリ管理下にある文字列（char *）へ変換します。

シグネチャまたは形状

static char *asn1_string_convert(apr_pool_t *p, ASN1_STRING *asn1str, int raw)

入力と出力

  • **入力**:
  • p: メモリ割り当てに使用する apr_pool_t。
  • asn1str: 変換対象の OpenSSL ASN1_STRING 構造体。
  • raw: 0 の場合、UTF-8 変換フラグを有効にします。
  • **出力**:
  • 変換後の文字列を返します。メモリは p に紐付けられます。
  • 失敗した場合は NULL を返します。

前提条件と不変条件

  • modssl_bio_free_read が内部で呼び出されるため、返り値の文字列は呼び出し側で解放する必要はありません。
  • BIO の生成に失敗した場合、関数は即座に NULL を返します。

アルゴリズムまたは振る舞い

1. BIO_s_mem() を用いてメモリベースの BIO を作成します。 2. raw が 0 の場合、ASN1_STRFLGS_UTF8_CONVERT フラグを付与し、制御文字のエスケープフラグ (ASN1_STRFLGS_ESC_CTRL) を設定します。 3. ASN1_STRING_print_ex を使用して、指定されたフラグに基づき BIO へ文字列を出力します。 4. modssl_bio_free_read を呼び出し、BIO の内容を apr_pool_t 上の文字列として取得し、BIO を解放して結果を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/ssl/ssl_util_ssl.c:220 (マクロ asn1_string_to_utf8 経由)
  • modules/ssl/ssl_util_ssl.c:226 (modssl_X509_NAME_ENTRY_to_string 内)
  • **呼び出し先**:
  • BIO_new, BIO_s_mem (OpenSSL)
  • ASN1_STRING_print_ex (OpenSSL)
  • modssl_bio_free_read (本ファイル内のヘルパー)

関連データ構造

  • ASN1_STRING: OpenSSL の ASN.1 文字列型。
  • apr_pool_t: Apache のメモリプール。
  • BIO: OpenSSL の I/O 抽象化レイヤー。

失敗モードと境界条件

  • BIO_new がメモリ不足等で失敗した場合、NULL を返します。
  • ASN1_STRING_print_ex の処理結果が空の場合、modssl_bio_free_read の挙動に従い空文字列または NULL が返されます。

保守担当者向けメモ

  • 本関数は static であり、モジュール内でのみ利用されます。
  • raw 引数の制御により、UTF-8 変換の有無を切り替える設計となっています。
  • OpenSSL の ASN1_STRING_print_ex に依存しているため、OpenSSL のバージョンアップ時に出力形式の変更がないか注意してください。
