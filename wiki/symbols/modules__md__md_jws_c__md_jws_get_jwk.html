md_jws_get_jwk

定義

  • 種別: function
  • 定義箇所: modules/md/md_jws.c:28

目的

公開鍵（md_pkey_t）から、ACMEプロトコルで使用される標準的なJSON Web Key (JWK) 構造を生成します。

シグネチャまたは形状

apr_status_t md_jws_get_jwk(md_json_t **pjwk, apr_pool_t *p, struct md_pkey_t *pkey)

入力と出力

  • **入力**:
  • p: メモリ割り当てに使用する apr_pool_t。
  • pkey: JWK生成元の公開鍵構造体。
  • **出力**:
  • pjwk: 生成された md_json_t オブジェクトへのポインタが格納されます。
  • 戻り値: 成功時は APR_SUCCESS、失敗時はエラーコード（APR_EINVAL 等）を返します。

前提条件と不変条件

  • pkey が NULL の場合、関数は直ちに APR_EINVAL を返します。
  • 生成されるJWKは、RSA鍵として固定された kty: "RSA" フィールドを含みます。

アルゴリズムまたは振る舞い

1. md_json_create(p) を呼び出し、新しいJSONオブジェクトを初期化します。 2. md_pkey_get_rsa_e64 および md_pkey_get_rsa_n64 を使用して、公開鍵の指数（e）と係数（n）をBase64URLエンコードされた文字列として取得します。 3. JSONオブジェクトに以下のフィールドを設定します:

  • "e": 指数
  • "kty": "RSA"
  • "n": 係数

    • 4. 生成したJSONオブジェクトを *pjwk に格納し、APR_SUCCESS を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/md/md_acme_acct.c:520
  • modules/md/md_jws.c:58
  • **呼び出し先**:
  • md_json_create
  • md_json_sets
  • md_pkey_get_rsa_e64
  • md_pkey_get_rsa_n64

関連データ構造

  • md_json_t: JSONデータを保持する構造体。
  • md_pkey_t: 公開鍵情報を保持する構造体。

失敗モードと境界条件

  • pkey が NULL の場合は APR_EINVAL を返します。
  • メモリ不足等の理由で md_json_create が失敗した場合の挙動は、依存する md_json モジュールの実装に依存します。

保守担当者向けメモ

  • 本関数はRSA鍵専用です。将来的に他の鍵タイプ（ECDSA等）をサポートする場合は、kty の判定ロジックおよび鍵パラメータの抽出処理を拡張する必要があります。
  • 生成されるJSONのキー順序は md_json_sets の呼び出し順に依存します。ACMEサーバーが特定の順序を要求する場合、この順序が維持されているか確認してください。
