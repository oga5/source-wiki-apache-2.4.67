dav_core_find_liveprop

定義

  • 種別: function
  • 定義箇所: modules/dav/main/std_liveprop.c:204

目的

DAV: 名前空間に属する標準的なライブプロパティを、コアプロバイダのグループから検索します。

シグネチャまたは形状

int dav_core_find_liveprop(
    const dav_resource *resource,
    const char *ns_uri, 
    const char *name,
    const dav_hooks_liveprop **hooks)

入力と出力

  • **入力**:
  • resource: 対象となるDAVリソース（本関数内では未使用）。
  • ns_uri: 検索対象のプロパティ名前空間URI。
  • name: 検索対象のプロパティ名。
  • hooks: ライブプロパティのフック構造体を受け取るためのポインタのポインタ。
  • **出力**:
  • 戻り値: プロパティが見つかった場合は dav_do_find_liveprop の結果（通常はプロパティのID）、見つからない場合はエラーコードを返します。
  • hooks: 成功時、dav_core_liveprop_group へのポインタが格納されます。

前提条件と不変条件

  • dav_core_liveprop_group が初期化済みである必要があります。
  • ns_uri および name は有効な文字列である必要があります。

アルゴリズムまたは振る舞い

1. 内部関数 dav_do_find_liveprop を呼び出します。 2. その際、引数として dav_core_liveprop_group を渡し、コアプロバイダが管理するプロパティリストから指定された名前空間と名前のプロパティを検索します。 3. 検索結果およびフック情報を呼び出し元へ返却します。

呼び出し元と呼び出し先

  • **呼び出し元**: modules/dav/main/mod_dav.c にて dav_hook_find_liveprop を通じて登録され、DAVエンジンから呼び出されます。
  • **呼び出し先**: dav_do_find_liveprop（modules/dav/main/liveprop.c で定義）。

関連データ構造

  • dav_core_liveprop_group: 本関数が検索対象とする、コアライブプロパティの仕様を統合した構造体。
  • dav_hooks_liveprop: ライブプロパティ操作用の関数ポインタ群。

失敗モードと境界条件

  • 指定された ns_uri と name が dav_core_liveprop_group に存在しない場合、dav_do_find_liveprop は失敗を返します。
  • resource 引数は現在の実装では使用されていません。

保守担当者向けメモ

  • 本関数は DAV: 名前空間のプロパティのみを対象としています。他の名前空間のプロパティ検索については、別のプロバイダが担当します。
  • dav_do_find_liveprop の挙動に依存しているため、検索ロジックの変更時は liveprop.c の実装も併せて確認してください。
