authn_dbd_cr_conf

定義

  • 種別: function
  • 定義箇所: modules/aaa/mod_authn_dbd.c:45

目的

ディレクトリ単位の認証設定構造体 authn_dbd_conf を生成し、初期化する。

シグネチャまたは形状

static void *authn_dbd_cr_conf(apr_pool_t *pool, char *dummy)

入力と出力

  • **入力**:
  • pool: 設定構造体の割り当てに使用するメモリプール。
  • dummy: Apache 設定フレームワークの規約による未使用の引数。
  • **出力**:
  • void *: 初期化された authn_dbd_conf 構造体へのポインタ。

前提条件と不変条件

  • apr_pcalloc により、構造体内の全メンバはゼロ初期化される。

アルゴリズムまたは振る舞い

1. 指定されたメモリプール pool から authn_dbd_conf サイズ分のメモリを確保する。 2. 確保したメモリをゼロクリアして返す。

呼び出し元と呼び出し先

  • **呼び出し元**: modules/aaa/mod_authn_dbd.c:301（Apache の設定読み込みプロセスから呼び出される）。
  • **呼び出し先**: apr_pcalloc（APR ライブラリ）。

関連データ構造

  • authn_dbd_conf: ディレクトリ単位の SQL クエリ設定を保持する構造体。

失敗モードと境界条件

  • メモリ不足時には apr_pcalloc が例外を発生させる可能性がある（Apache の標準的なメモリ管理動作に依存）。

保守担当者向けメモ

  • 本関数は Apache の設定生成フックとして機能する。構造体メンバの追加やデフォルト値の変更が必要な場合は、apr_pcalloc 後の初期化処理を拡張すること。
