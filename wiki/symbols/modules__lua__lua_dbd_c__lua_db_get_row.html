lua_db_get_row

定義

  • 種別: function
  • 定義箇所: modules/lua/lua_dbd.c:230

目的

データベースの結果セットから行データを取得し、Lua テーブルとしてスタックにプッシュする。

シグネチャまたは形状

int lua_db_get_row(lua_State *L);

入力と出力

  • **入力**:
  • L: Lua 状態。
  • index 2 (オプション): 取得する行番号（整数）。指定がない場合は 0（全行取得）。
  • index 3 (オプション): カラム名を使用するかを示す真偽値（alpha フラグ）。
  • **出力**:
  • 成功時: 行データを含むテーブルをスタックにプッシュし、1 を返す。
  • 失敗時: 0 を返す。

前提条件と不変条件

  • lua_get_result_set(L) を通じて有効な lua_db_result_set が取得可能であること。
  • res->results が初期化済みであること。

アルゴリズムまたは振る舞い

1. lua_get_result_set で結果セット構造体を取得する。 2. 行番号が 0 の場合:

  • 全行をループで取得し、行番号をキー、行データを値とするネストされたテーブルを構築する。

    • 3. 行番号が指定されている場合:

  • 指定された行を apr_dbd_get_row で取得する。

    • 4. 行データ取得時:

  • alpha フラグが真であればカラム名をキーとし、偽であればカラムインデックス（1始まり）をキーとしてテーブルに格納する。
  • apr_dbd_get_entry で各カラムの値を取得し、テーブルに設定する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/lua/lua_dbd.c:345 (登録)
  • modules/lua/lua_dbd.c:438 (登録)
  • **呼び出し先**:
  • lua_get_result_set
  • apr_dbd_get_row
  • apr_dbd_get_entry
  • apr_dbd_get_name (alpha フラグが真の場合)

関連データ構造

  • lua_db_result_set: 結果セットのメタデータと APR DBD 関連リソースを保持。
  • apr_dbd_row_t: APR DBD レベルの行データ構造。

失敗モードと境界条件

  • apr_dbd_get_row が -1 を返した場合（行が存在しない、またはエラー）、関数は 0 を返す。
  • alpha フラグが真でカラム名が取得できない場合、キーとして "(oob)" が使用される。

保守担当者向けメモ

  • alpha フラグが真の場合、apr_dbd_get_name を使用してカラム名を取得する。
  • 戻り値のテーブル構造は、Lua 側で resultset(row_no) のように呼び出されることを想定している。
