unixd_set_group

定義

  • 種別: function
  • 定義箇所: modules/arch/unix/mod_unixd.c:234

目的

Apache HTTP Server の設定ファイルにおける Group ディレクティブを処理し、指定されたグループ名をシステム上のグループ ID に変換して、共有設定構造体 ap_unixd_config に格納します。

シグネチャまたは形状

static const char* unixd_set_group(cmd_parms *cmd, void *dummy, const char *arg)

入力と出力

  • **入力**:
  • cmd: 設定コマンドのパラメータ構造体。
  • dummy: 使用されません。
  • arg: 設定ファイルで指定されたグループ名（文字列）。
  • **出力**:
  • 成功時: NULL を返します。
  • 失敗時: エラーメッセージを示す文字列（const char*）を返します。

前提条件と不変条件

  • 本関数は、設定コンテキストが GLOBAL_ONLY（サーバー全体の設定）である必要があります。
  • ap_unixd_config 構造体は、モジュールのライフサイクルを通じて生存するグローバルな領域に存在します。

アルゴリズムまたは振る舞い

1. ap_check_cmd_context を呼び出し、現在の設定コンテキストが GLOBAL_ONLY であるか検証します。違反している場合はエラーメッセージを返して終了します。 2. 引数 arg（グループ名）を ap_unixd_config.group_name に代入します。 3. ap_gname2id(arg) を呼び出し、グループ名をシステム上のグループ ID に変換して ap_unixd_config.group_id に格納します。 4. NULL を返し、正常終了します。

呼び出し元と呼び出し先

  • **呼び出し元**: Apache の設定解析エンジンにより、Group ディレクティブが検出された際に呼び出されます（modules/arch/unix/mod_unixd.c:415 の AP_INIT_TAKE1 マクロ経由）。
  • **呼び出し先**:
  • ap_check_cmd_context: コンテキストの検証。
  • ap_gname2id: グループ名から ID への変換。

関連データ構造

  • ap_unixd_config: グループ名およびグループ ID を保持する共有設定構造体。
  • cmd_parms: 設定コマンドの実行時パラメータ。

失敗モードと境界条件

  • ap_check_cmd_context が GLOBAL_ONLY 以外のコンテキストを検出した場合、エラーメッセージが返されます。
  • ap_gname2id が指定されたグループ名を解決できない場合の挙動については、本ソース excerpt 内に明示的な記述はありません。

保守担当者向けメモ

  • 本関数は Group ディレクティブの解析専用です。
  • ap_unixd_config への書き込みは、サーバー起動時の設定フェーズで行われることを前提としています。
