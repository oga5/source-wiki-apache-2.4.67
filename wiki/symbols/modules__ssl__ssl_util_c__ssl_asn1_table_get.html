ssl_asn1_table_get

定義

  • 種別: function
  • 定義箇所: modules/ssl/ssl_util.c:228

目的

APR ハッシュテーブルから、指定されたキーに対応する ssl_asn1_t 構造体を取得します。

シグネチャまたは形状

ssl_asn1_t *ssl_asn1_table_get(apr_hash_t *table, const char *key);

入力と出力

  • **入力**:
  • table: 検索対象となる apr_hash_t 構造体へのポインタ。
  • key: 検索キーとなる文字列。
  • **出力**:
  • 成功時: 見つかった ssl_asn1_t 構造体へのポインタ。
  • 失敗時: NULL（キーが存在しない場合）。

前提条件と不変条件

  • table は apr_hash_make 等で初期化された有効なハッシュテーブルである必要があります。
  • key は APR_HASH_KEY_STRING として扱われるため、ヌル終端文字列である必要があります。

アルゴリズムまたは振る舞い

1. apr_hash_get を呼び出し、table 内を key で検索します。 2. 検索結果を ssl_asn1_t * 型にキャストして返却します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/ssl/ssl_engine_init.c:1589
  • modules/ssl/ssl_engine_pphrase.c:201
  • **呼び出し先**:
  • apr_hash_get (APR ライブラリ関数)

関連データ構造

  • ssl_asn1_t: ASN.1 データを保持する構造体。
  • apr_hash_t: APR が提供するハッシュテーブル。

失敗モードと境界条件

  • table が NULL の場合、APR の仕様に従いセグメンテーションフォールトが発生する可能性があります。
  • 指定した key がテーブル内に存在しない場合、NULL を返します。

保守担当者向けメモ

  • 本関数は単なる apr_hash_get のラッパーです。
  • ssl_asn1_table_set で格納されたデータは malloc で確保されているため、取得したポインタの生存期間管理には注意してください。
