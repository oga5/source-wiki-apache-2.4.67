stop_console_handler

定義

  • 種別: function
  • 定義箇所: server/mpm/winnt/service.c:200

目的

コンソールアプリケーションとして実行されている際に、以前登録されたコンソール制御ハンドラを解除するために使用されます。

シグネチャまたは形状

static void stop_console_handler(void)

入力と出力

  • 入力: なし
  • 出力: なし（戻り値なし）

前提条件と不変条件

  • 本関数は、事前に SetConsoleCtrlHandler を使用して console_control_handler が登録されている環境で呼び出されることを前提としています。

アルゴリズムまたは振る舞い

1. SetConsoleCtrlHandler API を呼び出します。 2. 第1引数に console_control_handler を指定し、第2引数に FALSE を渡すことで、当該ハンドラをシステムから削除（登録解除）します。

呼び出し元と呼び出し先

  • 呼び出し元: server/mpm/winnt/service.c:209 にて atexit(stop_console_handler) として登録されており、プロセス終了時に自動的に呼び出されます。
  • 呼び出し先: Windows API の SetConsoleCtrlHandler を呼び出します。

関連データ構造

  • console_control_handler: 本関数によって登録解除される対象のコールバック関数。

失敗モードと境界条件

  • SetConsoleCtrlHandler の実行結果については本関数内で評価・処理されていません。

保守担当者向けメモ

  • 本関数はプロセス終了時のクリーンアップ処理の一部です。atexit を介して呼び出されるため、プログラムの正常終了または exit() 呼び出し時に実行されます。
