etag_components_t

定義

  • 種別: typedef (unsigned long)
  • 定義箇所: include/http_core.h:491

目的

HTTP ETag 生成時に含めるべき要素（inode, size, mtime 等）をビットマスクとして管理するために使用されます。

シグネチャまたは形状

typedef unsigned long etag_components_t;

入力と出力

  • 入力: ビット演算によるフラグの追加・削除操作。
  • 出力: ETag 生成ロジックにおける条件判定用のビットマスク値。

前提条件と不変条件

  • unsigned long 型として定義されており、ビット演算によるフラグ操作を前提としています。

アルゴリズムまたは振る舞い

  • etag_bits、etag_add、etag_remove といった変数を通じて、ETag に含める属性の有効・無効を制御します。
  • server/core.c 等において、特定のビットフラグの有無を確認する判定に使用されます。

呼び出し元と呼び出し先

  • 呼び出し元: include/http_core.h 内の構造体メンバ、modules/http/http_etag.c、server/core.c。
  • 呼び出し先: 情報なし。

関連データ構造

  • core_dir_config (設定構造体の一部として利用される可能性あり)

失敗モードと境界条件

  • ビットマスクのオーバーフロー（unsigned long のビット幅を超えるフラグ定義は不可）。

保守担当者向けメモ

  • 本型は ETag の生成ルールをカスタマイズするための設定値として機能します。新しい ETag 要素を追加する場合は、既存のビット定義との重複を避ける必要があります。
