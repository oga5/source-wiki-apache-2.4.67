md_ocsp_cert_stat_value

定義

  • 種別: function
  • 定義箇所: modules/md/md_ocsp.c:124

目的

OCSP 証明書ステータスを表す文字列（JSON 等から取得）を、内部的な列挙型 md_ocsp_cert_stat_t に変換します。

シグネチャまたは形状

md_ocsp_cert_stat_t md_ocsp_cert_stat_value(const char *name)

入力と出力

  • **入力**: name (文字列)。証明書のステータスを示すラベル（例: "good", "revoked"）。
  • **出力**: md_ocsp_cert_stat_t 型の列挙値。

前提条件と不変条件

  • 入力 name が NULL の場合、または既知の文字列と一致しない場合は MD_OCSP_CERT_ST_UNKNOWN を返します。

アルゴリズムまたは振る舞い

1. name が "good" と一致する場合、MD_OCSP_CERT_ST_GOOD を返します。 2. name が "revoked" と一致する場合、MD_OCSP_CERT_ST_REVOKED を返します。 3. 上記以外（name が NULL を含む）の場合、MD_OCSP_CERT_ST_UNKNOWN を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: modules/md/md_ocsp.c:217 において、JSON データからステータスを読み取る際に呼び出されています。
  • **呼び出し先**: 標準ライブラリの strcmp を使用しています。

関連データ構造

  • md_ocsp_cert_stat_t: 本関数が返す列挙型。

失敗モードと境界条件

  • name が NULL の場合、安全に MD_OCSP_CERT_ST_UNKNOWN を返します。
  • 未定義の文字列が渡された場合、エラーを発生させず MD_OCSP_CERT_ST_UNKNOWN を返します。

保守担当者向けメモ

  • 本関数は md_ocsp_cert_stat_name と対になる変換関数です。新しいステータス値を追加する場合は、両方の関数を更新してください。
