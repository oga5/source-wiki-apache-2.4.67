on_frame_send

定義

  • 種別: function
  • 定義箇所: modules/http2/h2_stream.c:157

目的

HTTP/2 ストリームにおいて、送信されるフレームの種類に基づき、現在のストリーム状態に対する遷移処理を実行する。

シグネチャまたは形状

static int on_frame_send(h2_stream_state_t state, int frame_type)

入力と出力

  • **入力**:
  • state: 現在のストリーム状態 (h2_stream_state_t)。
  • frame_type: 送信対象の HTTP/2 フレーム種別 (int)。
  • **出力**:
  • 遷移後の新しいストリーム状態を int 型で返す。

前提条件と不変条件

  • trans_on_send テーブルが定義されており、H2_ALEN マクロによりそのサイズが正しく算出可能であること。

アルゴリズムまたは振る舞い

1. 内部関数 on_frame を呼び出す。 2. その際、送信用の遷移テーブル trans_on_send とそのサイズを引数として渡す。 3. on_frame の戻り値をそのまま呼び出し元へ返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • modules/http2/h2_session.c:732: NGH2_SET_CALLBACK を介して登録。
  • modules/http2/h2_stream.c:431: ストリーム状態更新時に直接呼び出し。
  • **呼び出し先**:
  • on_frame (同一ファイル内)

関連データ構造

  • h2_stream_state_t: ストリームの状態定義。
  • trans_on_send: 送信時の状態遷移を規定するテーブル。

失敗モードと境界条件

  • on_frame 内部での遷移定義外のフレーム種別が渡された場合の挙動は on_frame の実装に依存する。

保守担当者向けメモ

  • 本関数は on_frame への単なるラッパーである。状態遷移ロジックの変更が必要な場合は、trans_on_send テーブルの内容を確認すること。
