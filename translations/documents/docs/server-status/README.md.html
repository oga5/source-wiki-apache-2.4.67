server-status =============

動的チャートを使用した Apache httpd の mod_status の mod_lua バージョン

機能 ##

このスクリプトは、httpd の既知の mod_status 統計ページを拡張したものです。 シンプルな Quokka Chart API を使用して、プレーンテキスト情報では診断が困難な要素の多くを視覚化します。

動作については https://www.apache.org/server-status を参照してください。

要件 ##

  • Apache httpd 2.4.6 以上
  • mod_lua (Lua 5.1、5.2、または LuaJIT のいずれか)
  • mod_status がロードされていること (トラフィック統計を有効にするため)

インストール ##

まず、mod_lua をインストールします (configure 時に --enable-lua を指定して有効にできます)。

ハンドラーとしてインストールする場合:

ハンドラーとしてインストールするには、httpd.conf の適切な VirtualHost 内に以下を追加します。

LuaMapHandler ^/server-status$ /path/to/server-status.lua

Web アプリとしてインストールする場合:

通常の Web アプリとしてインストールするには、適切な VirtualHost 設定に以下を追加して、.lua スクリプトが mod_lua で処理されるようにします。

AddHandler lua-script .lua

その後、.lua スクリプトをドキュメントルートのどこかに配置し、そのページにアクセスしてください。

設定

Lua スクリプト内には、true または false に設定できるいくつかのオプションがあります。

  • show_warning: このページが意図的に公開されている旨の通知を表示するかどうか。
  • redact_ips: IP アドレスの最後の数ビットを 'x.x' に置き換えるかどうか。
  • show_modules: ロードされたモジュールのリストを表示するかどうか。
  • show_threads: スレッドの詳細を表示するかどうか。
