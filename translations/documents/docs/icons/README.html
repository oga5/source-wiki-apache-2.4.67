パブリックドメインアイコン

     これらのアイコンは、もともとX用Mosaicのために作成されたもので、過去にはNCSA httpdおよびApacheサーバーの配布物に含まれていました。これらはパブリックドメインであり、どのようなアプリケーションにも自由に取り入れることができます。オリジナルはKevin Hughes (kevinh@kevcom.com) によって作成されました。Andy Polyakovがアイコンの色を調整し、いくつかの新しい画像を追加しました。

     このセットへの追加に貢献したい場合は、httpdドキュメントプロジェクト <http://httpd.apache.org/docs-project/> までご連絡ください。

     これらのアイコンのほとんどは20x22ピクセルのサイズです。「small」ディレクトリには、Mike Brown (mike@hyperreal.org) が提供した16x16サイズの代替アイコンがあります。

推奨される用途

以下に、アイデアの出発点としていくつかの提案を挙げます。アイコンは必要に応じて自由に変更や名前の変更を行ってください。

     a.gif
          PostScriptやテキストレイアウト言語を表すために使用できます。

     alert.black.gif, alert.red.gif
          ディレクトリ内のREADMEファイルなど、重要な項目を強調するために使用できます。

     back.gif, forward.gif
          前後の領域へ移動するためのリンクとして使用できます。

     ball.gray.gif, ball.red.gif
          箇条書き（ビュレット）として使用できます。

     binary.gif
          バイナリファイルを表すために使用できます。

     binhex.gif
          BinHexエンコードされたデータを表すことができます。

     blank.gif
          プレースホルダーやスペーシング要素として使用できます。

     bomb.gif
          コアファイルを表すために使用できます。

     box1.gif, box2.gif
          汎用的な3Dアプリケーションや関連ファイルを表すために使用できます。

     broken.gif
          破損したデータを表すことができます。

     burst.gif
          新しく重要な項目に注意を引くために使用できます。

     c.gif
          C言語のソースコードを表すことができます。

     comp.blue.gif, comp.gray.gif
          これらの小さなコンピュータアイコンは、telnetやFTPセッションを表すことができます。

     compressed.gif
          圧縮データを表すことができます。

     continued.gif
          ディレクトリの続きのリストへのリンクとして使用できます。

     down.gif, up.gif, left.gif, right.gif
          リスト内での上下左右のスクロールや、アウトライン内の項目を示すために使用できます。

     dir.gif
          以下のfolder.gifと同じです。

     diskimg.gif
          フロッピーディスクストレージを表すことができます。

     dvi.gif
          DVIファイルを表すことができます。

     f.gif
          FORTRANやForthのソースコードを表すことができます。

     folder.gif, folder.open.gif, folder.sec.gif
          フォルダはディレクトリを表すことができます。セキュアなディレクトリや閲覧できないディレクトリを表すバージョンもあります。

     generic.gif, generic.sec.gif, generic.red.gif
          それぞれ、汎用ファイル、セキュアファイル、重要なファイルを表すことができます。

     hand.right.gif, hand.up.gif
          重要な項目を指し示すために使用できます（洒落です）。

     image1.gif, image2.gif, image3.gif
          さまざまな種類の画像フォーマットを表すことができます。

     index.gif
          WAISインデックスや検索機能を表すことができます。

     layout.gif
          HTMLやPDFファイルのように、グラフィックスとテキストレイアウトの両方を含むファイルやフォーマットを表すことができます。

     link.gif
          シンボリックリンクであるファイルを表すことができます。

     movie.gif
          さまざまな動画フォーマットを表すことができます。

     p.gif
          PerlやPythonのソースコードを表すことができます。

     pie0.gif ... pie8.gif
          検索結果からドキュメントのリストが返されるアプリケーションで使用できます。小さな円グラフの画像は、検索クエリに対するドキュメントの関連度を示すことができます。

     patch.gif
          パッチやdiffファイルを表すことができます。

     portal.gif
          オンラインサービスや3Dワールドへのリンクとして使用できます。

     pdf.gif, ps.gif, quill.gif
          PDFファイルやPostScriptファイルを表すことができます。

     screw1.gif, screw2.gif
          CADやエンジニアリングデータおよびフォーマットを表すことができます。

     script.gif
          Perl、Python、TCL、シェルスクリプトなどのさまざまなインタープリタ言語や、サーバー設定ファイルを表すことができます。

     sound1.gif, sound2.gif
          サウンドファイルを表すことができます。

     sphere1.gif, sphere2.gif
          3Dワールドやレンダリングアプリケーションおよびフォーマットを表すことができます。

     tar.gif
          TARアーカイブファイルを表すことができます。

     tex.gif
          TeXファイルを表すことができます。

     text.gif
          汎用的な（プレーン）テキストファイルを表すことができます。

     transfer.gif
          FTP転送やアップロード/ダウンロードを表すことができます。

     unknown.gif
          不明な種類のファイルを表すことができます。

     uu.gif, uuencoded.gif
          uuencodeされたデータを表すことができます。

     world1.gif, world2.gif
          3Dワールドやその他の3Dフォーマットを表すことができます。
