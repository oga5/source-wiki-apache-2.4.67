多言語カスタムエラードキュメント
-------------------------------------

'error' ディレクトリには、複数の言語による HTTP エラーメッセージが含まれています。
クライアントの優先言語が利用可能な場合、MultiViews 機能によって自動的に選択されます。この機能は、Options、Language、および ErrorDocument ディレクティブを介してデフォルトで有効になっています。

'error/include' ディレクトリ内の HTML ファイルを編集することで、ドキュメントのデザインやマークアップを設定できます。
