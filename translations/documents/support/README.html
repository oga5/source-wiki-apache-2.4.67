サポートファイル:

ab
	このベンチマークツールでサーバーに負荷をかけます。基本的なコマンドラインテストツールです。

apachectl
	Apache実行時制御スクリプト。管理者がApache httpdデーモンを制御したり、rc.dスクリプトから制御したりしやすくするためのものです。

apxs
	APache eXtenSionツール。DSO形式のモジュールの構築とインストールを容易にします。

dbmmanage
	mod_auth_dbで使用される高速なDBM形式のユーザー認証ファイルを作成および更新します。

htcacheclean
	mod_cache_diskの保存サイズを一定の制限内に保ちます。

htdigest
	DIGEST認証で使用されるユーザー認証ファイルを作成および更新します。mod_auth_digestを参照してください。

htpasswd 
	BASIC認証で使用されるユーザー認証ファイル（htpasswdファイルなど）を作成および更新します。mod_authを参照してください。

httpd.8
	Apacheの一般的なマニュアルページです。

log_server_status
	このスクリプトは、cronなどのツールによって頻繁な間隔で実行されるように設計されています。サーバーに接続してステータス情報をダウンロードします。情報を1行に再フォーマットし、ファイルに記録します。

logresolve
	ApacheログファイルのIPアドレスをホスト名に解決します。

phf_abuse_log.cgi
	このスクリプトは、Apache 1.0.3以前のバージョンに配布されていたCGIスクリプトに存在した、古くから修正済みのセキュリティホールを悪用しようとするユーザーを検出するために使用できます。

rotatelogs
	サーバーを停止することなくApacheログをローテーションします。

split-logfile
	このスクリプトは、結合されたバーチャルホストのアクセスログファイルを受け取り、その内容を個別のファイルに分割します。

suexec
	Switch User For Exec。Apache内部で使用されます。http://www.apache.org/docs/suexec.html にあるドキュメント「Apache suEXEC Support」を参照してください。

SHA1
	このディレクトリには、Netscape Webサーバーで使用されているSHA1形式のパスワードをApache 1.3.6で認識できるようにするためのユーティリティが含まれています。デフォルトではインストールされません。
