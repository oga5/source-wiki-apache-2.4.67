modules/README

このレベルのディレクトリ構造は以下の通りです。

aaa/
  このディレクトリには、認可および認証を扱うモジュールが含まれています。

arch/

cache/
  このディレクトリには、ファイルおよびデータのキャッシュ機能を実装するモジュールが格納されています。

database/
  apache DBDフレームワークは、SQLバックエンドへの接続を効率的に管理します。

cluster/
  複数のサーバーを連携させるためのモジュールです。

dav/
  このディレクトリには、WebDAV機能を実装するモジュールが格納されています。

echo/

examples/
  このディレクトリには、独自のApacheモジュールを開発する際に役立つサンプルコードが含まれています。

experimental/
  このディレクトリには、非常に興味深い機能を提供すると考えられるものの、まだ開発の初期段階にあり、将来的に大幅に変更される可能性があるモジュールを配置しています。このコードは公式にはサポートされていません。

filters/
  このディレクトリには、一般的なインラインデータフィルタリングを実行するモジュールが格納されています。

generators/
  このディレクトリには、データ生成機能を実行するモジュールが格納されています。

http/
  このディレクトリには、基本的なHTTPプロトコルを実装するモジュールが格納されています。

http2/
  このディレクトリには、HTTP/2プロトコルを実装するモジュールが格納されています。

loggers/
  このディレクトリには、ログ機能を処理するモジュールが格納されています。

mappers/
  このディレクトリには、URLマッピングおよび書き換えを処理するモジュールが格納されています。

metadata/
  このディレクトリには、ヘッダーメタデータを扱うモジュールが格納されています。

proxy/
  ここには、Apache用プロキシモジュールのコードが格納されています。

ssl/
  このディレクトリには、OpenSSL機能のためのコードが格納されています。

test/
  このディレクトリには、Apacheの様々なコンポーネントをテストするためのモジュールが格納されています。これらを本番環境のサーバーにコンパイルして含めるべきではありません。
