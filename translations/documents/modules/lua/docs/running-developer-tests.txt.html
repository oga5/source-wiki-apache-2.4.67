* mod_lua のビルド
  最初のステップは、building-from-subversion.txt の手順に従って mod_lua をビルドすることです。

* LuaSocket のビルドとインストール
    http://www.cs.princeton.edu/~diego/professional/luasocket/
    FreeBSD: /usr/ports/net/luasocket

* テストの実行
  1. Apache の httpd.conf を test/test_httpd.conf に置き換えます。
  2. 新しい httpd.conf を自身のディレクトリ構成に合わせてカスタマイズします。
  3. 最後に、テストを実行するために Apache を起動し、以下を実行します。
     $ cd test
     $ lua ./test.lua
     FreeBSD: lua-5.1 ./test.lua
