tcpdumpをハックするのが面倒な人（例えば、私のtcpdumpにはデータをASCIIで表示する-Xオプションがあります）は、壊れたクライアントをデバッグする際にHTTPヘッダーを取得するために、以下のようなものを使うことができます。

複雑なものではありませんが、便利です。

私は普段、このPerlスクリプトを使っていて、"tcpdumpscii"と呼んでいます。
実行時は "tcpdumpscii -s 1500 -x [その他のtcpdump引数]" とします。

  Bill

#!/import/misc/bin/perl
#
#
open(TCPDUMP,"tcpdump -l @ARGV|");
while (<TCPDUMP>) {
	if (/^\s+(\S\S)+/) {
		$sav = $_;
		$asc = "";
		while (s/\s*(\S\S)\s*//) {
			$i = hex($1);
			if ($i < 32 || $i > 126) {
				$asc .= ".";
			} else {
				$asc .= pack(C,hex($1));
			}
		}
		$foo = "." x length($asc);
		$_ = $sav;
		s/\t/        /g;
		s/^$foo/$asc/;
	}
	print;
}
